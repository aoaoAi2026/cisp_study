## 1. 平台概述

CISP 网络安全学习平台是一个**全栈 Web 学习系统**，旨在帮助学员系统化地备考 CISP（注册信息安全专业人员）认证考试，同时提供网络安全实操训练环境。

### 核心技术栈

- **前端**: React 18 + TypeScript + Vite 6
- **路由**: react-router-dom v6 (BrowserRouter)
- **样式**: Tailwind CSS
- **动画**: framer-motion
- **图标**: lucide-react
- **代码编辑器**: Monaco Editor
- **状态管理**: Zustand (userStore / learningStore / achievementStore)
- **Python 运行**: Pyodide（浏览器端）
- **数据持久化**: localStorage + SQLite（后端）
- **移动端打包**: Capacitor（Android APK）
- **错误处理**: ErrorBoundary 组件 + 前端容错

### 平台特色（v2.0 全面升级）

- **📚 系统化学习**: 90天完整学习路径 + 30天冲刺模式 + 4个专项计划（基础/渗透/防御/AI安全）
- **🎮 游戏化激励**: XP 经验值、等级系统、连续打卡、徽章成就、Confetti 庆祝特效
- **🛡️ 安全实操**: 11 个实验模块（XSS/SQL注入/CTF/密码学等）
- **💻 在线编程**: Monaco + Pyodide + 后端API多语言代码执行
- **📝 模拟考试**: 历年真题库 + 模拟考试 + 错题本
- **🔗 资源整合**: 安全资源库、工具网站导航、社区交流
- **🛠️ 在线工具箱 ⭐新增⭐: **200+ 实用工具**，涵盖编码解码、哈希计算、进制转换、密码学工具、随机数据生成器、文本处理、颜色/图像、时间日期、单位换算、网络工具、生活计算（房贷/个税/BMI），**全部纯前端实现，一键使用**
- **🧱 稳定性增强**: 全局 ErrorBoundary 错误边界，单个页面异常不会导致整体崩溃
- **📱 移动端支持**: 响应式设计，支持手机浏览器；提供 Capacitor Android APK 打包方案

## 2. 快速开始

### 环境要求

- **Node.js** ≥ 18
- **npm** ≥ 9
- 推荐使用 PowerShell 或 Git Bash（Windows）

### 启动开发服务器

```bash
cd cisp
npm install
npm run dev
```

服务默认运行在 **http://localhost:6776**。如果端口被占用，Vite 会自动选择下一个可用端口（如 6777、5173）。

### 生产构建

```bash
npm run build    # 编译 TypeScript + Vite 打包
npm run preview  # 预览构建结果（默认端口 4173）
```

### APK 打包（Android）

```bash
# 先构建前端
npm run build

# 同步到 Android 项目
npm run cap:sync

# 打开 Android Studio 构建 APK
npx cap open android
```

APK 输出位置: `android/app/build/outputs/apk/debug/app-debug.apk`

### 类型检查

```bash
npm run check    # 仅类型检查，不构建
```

### 注册/登录

打开浏览器访问 `http://localhost:6776/auth`：

| 操作 | 说明 |
|------|------|
| **注册** | 输入用户名（≥3字符）、密码（≥6字符）、邮箱（可选），点击"注册账号" |
| **登录** | 输入已注册的用户名和密码，点击"登录" |
| **🚀 游客体验（无需注册）** | 一键创建随机账号登录，适合快速体验平台功能 |

> 数据存储在后端 SQLite 数据库，游客数据也会持久化。

## 3. 页面功能详解

### 3.1 登录/注册 (AuthPage)

**路由**: `/auth`  
**类型**: 公开页面（无需登录即可访问）

#### 功能区域

**登录/注册切换**
- 顶部有两个切换按钮："登录"和"注册"
- 选中的模式高亮显示为青色
- 切换无需重新加载页面，即时切换表单

**表单字段**
- **用户名**: 必填，最少 3 个字符
- **邮箱**: 仅注册时显示，可选字段
- **密码**: 必填，最少 6 个字符

**操作按钮**
- **登录/注册账号**: 提交表单，成功后跳转到首页仪表盘
- **🚀 游客体验（无需注册）**: 一键创建随机账号登录，适合快速体验

**错误提示**
- 登录失败或注册失败时，在密码框下方显示红色错误信息框

---

### 3.2 首页仪表盘 (Dashboard)

**路由**: `/`  
**访问条件**: 登录后可见

#### 页面布局

首页是信息聚合中心，从上到下包含以下区域：

##### ① 粒子背景 + 欢迎区域
- 动态粒子动画背景
- 显示用户昵称、连续学习天数、距离 CISP 考试倒计时
- "开始今日学习"主按钮（跳转到学习路径

##### ② 统计卡片（4 个）
| 卡片 | 图标 | 内容 |
|------|------|------|
| 学习天数 | 📅 | 累计学习天数 |
| 连续学习 | 🔥 | 当前连续打卡天数 |
| 获得成就 | 🏆 | 已解锁徽章数量 |
| 积分 | ⭐ | 当前 XP 积分 |

##### ③ 每日挑战卡片
- 每天随机生成 8 种挑战之一（如"限时问答"、"编码挑战"等）
- 显示挑战名称、描述、状态
- 点击"接受挑战"开始交互

##### ④ 本周打卡日历
- 7 天日历视图（周一至周日）
- 已完成学习的日期显示绿色对勾
- 未完成显示的日期用半透明标记

##### ⑤ 学习小贴士
- 每次刷新随机展示一条学习建议

##### ⑥ 学习进度环 (ProgressRing)
- 圆形进度条展示总体完成百分比
- 动画过渡效果

##### ⑦ 今日学习任务列表
- 展示当天的待办学习任务
- 可勾选标记完成

##### ⑧ 12 周学习路径概览
- 快速跳转到学习路径页的各周
- 显示每周完成进度

##### ⑨ 学习数据统计
| 指标 | 说明 |
|------|------|
| 闪卡练习 | 已练习题目数 |
| 番茄专注 | 累计专注分钟数 |
| 学习笔记 | 已创建笔记数 |
| 徽章积分 | 已获得积分/目标积分 |

### 3.3 学习路径 (LearningPath)

**路由**: `/learning`

#### 功能说明

**学习模式选择**
- **完整 90 天模式**: 每天一节课程，系统化覆盖所有知识域
- **冲刺 30 天模式**: 浓缩版，每天任务量增大，适合考前突击

**12 周时间线**
- 纵向时间线布局，每周一个区块
- **点击周标题** 可展开/收起该周的学习内容
- 展开后显示每天的课程卡片

**每日课程卡片状态**

| 状态 | 颜色 | 说明 |
|------|------|------|
| ✅ 已完成 | 绿色边框 + 对勾 | 该日内容已学完 |
| 🔵 进行中 | 蓝色高亮 | 当前可学习 |
| 🔒 锁定 | 灰色 + 锁图标 | 需要先完成前面的天数 |

**进度统计**
- 顶部显示连续学习天数和总进度
- ProgressRing 进度环直观展示完成比例

**操作指南**
1. 选择需要的学习模式（完整90天或冲刺30天）
2. 点击一个周区块展开该周课程
3. 点击当天课程卡片 → 跳转到 DailyLearning 页面
4. 必须按顺序解锁，不能跳课

### 3.4 每日学习 (DailyLearning)

**路由**: `/learning/:dayId`

#### 页面结构

**顶部导航栏**
- **← 前一天** / **后一天 →** 按钮快速切换天数
- 显示当前天数标题和学习目标

**主内容区 — Markdown 课程渲染**
- 课程内容以 Markdown 格式渲染
- 支持标题、列表、表格、代码块、引用等

**代码示例区**
- 展示安全相关的代码示例
- **复制按钮**: 一键复制代码到剪贴板
- **运行按钮**: 在浏览器中运行 Python 代码（通过 Pyodide）

**练习题测验**
- 每节课末尾有配套选择题
- **答题流程**:
  1. 阅读题目，从 4 个选项中点选答案
  2. 点击"提交"查看得分
  3. 点击"查看解析"了解正确答案和详细解释

**大神笔记**
- 展示预设的"学霸"学习笔记
- 包含关键知识点总结和记忆口诀

**学习笔记**
- 文本框自由书写学习笔记
- **自动保存**: 输入停止 800ms 后自动保存
- 笔记与当前课程关联

**番茄钟组件**
- 25 分钟倒计时专注学习
- 开始/暂停/重置操作
- 完成后自动记录专注时间

**错题本追踪**
- 标记做错的题目
- 可在测验中心的错题本统一回顾

**推荐工具和靶场链接**
- 课程相关的外部学习资源
- 直接跳转或新标签页打开

**"完成今日学习"按钮**
- 点击后标记当天课程为已完成
- 解锁下一天课程
- 获得 XP 经验值奖励

### 3.6 Cyber 每日课程 (CyberDailyLearning)

**路由**: `/cyber-learning/:planId`

这是整个平台**功能最丰富**的页面，融合了学习、编程、测验、笔记、番茄钟、游戏化多个子系统。

#### ① 游戏化统计面板
- **XP 经验条**: 当前经验值 / 升级所需经验值 + 等级标识
- **连续打卡**: 显示当前连续学习天数
- **学习热力图**: 类似 GitHub 贡献图，用颜色深浅展示每天学习活跃度
- **徽章展示**: 最近获得的 3 个徽章
- **测验均分**: 所有测验的平均正确率

#### ② Day 导航栏
- 横向课程列表，快速跳转到任意课程
- **搜索框**: 输入关键词搜索课程
- **键盘快捷键**: 按 `←` / `→` 键切换前后课程

#### ③ 课程大纲（可折叠）
- 当前课程完整章节树
- 每个章节可展开查看子章节
- **锚点导航**: 点击子章节快速滚动到对应内容

#### ④ Markdown 课程内容
- 同 DailyLearning 的 Markdown 渲染

#### ⑤ Monaco Editor 代码编辑器
- 内嵌 VS Code 同款代码编辑器
- 支持 **Python / JavaScript / Java** 语法高亮
- **在线运行**: 点击 Run 按钮通过 Pyodide 执行 Python 代码
- 适合编写和测试安全脚本

#### ⑥ 编程练习
- 每个练习有难度标签（简单/中等/困难）
- **测试用例**: 显示输入和期望输出
- **提示**: 点击展开解题思路
- **答案**: 点击查看参考答案
- 代码通过测试后自动获得 XP 奖励

#### ⑦ 代码示例
- 展示实际安全场景的代码示例
- 支持复制和运行

#### ⑧ 实验靶场
- **Docker 实验**: 提供 Docker 命令一键启动实验环境
- **在线实验**: 链接到在线靶场
- **本地实验**: 详细环境准备步骤和操作流程

#### ⑨ 推荐工具 + 学习资源
- 课程相关的实用工具链接
- 扩展阅读资料

## 4. 错误边界与稳定性说明

### 4.1 ErrorBoundary 组件

应用在 `src/App.tsx` 顶层用 `ErrorBoundary` 组件包裹整个路由，确保：

- **单个页面异常不会导致整体白屏**
- 异常时显示友好的错误提示，包含错误信息
- 提供"🔄 刷新页面"按钮重试
- 避免因数据文件格式、网络波动或新功能代码 bug 影响学习体验

### 4.2 容错机制

| 机制 | 说明 |
|------|------|
| **前端路由降级** | 未匹配路由显示 404 页 |
| **数据加载降级** | 课程/题库加载失败时显示占位信息而非崩溃 |
| **localStorage 异常保护** | 数据读写异常时自动降级为临时内存存储 |
| **API 超时处理** | 后端 API 超时（>5s）自动降级为纯前端模式 |
| **在线工具容错** | 单个工具计算异常仅影响该工具，其他工具仍可使用 |

## 7. 常见问题 (FAQ)

### Q: 游客体验能保存数据吗？

**A**: 能。游客账号自动注册到后端 SQLite，数据与正式账号一致，不会丢失。

### Q: 学习进度为什么不能跳过？

**A**: 系统设计为按顺序解锁，确保知识学习的系统性。如急需跳转可直接在 URL 输入目标页面，或使用随机练习模式。

### Q: 代码运行支持哪些语言？

**A**:

- **Pyodide（浏览器运行）**: Python，无需后端
- **后端 API**: Python / JavaScript / Java

### Q: 模拟考试可以中途退出吗？

**A**: 模拟考试有暂停功能（隐藏题目），但不支持保存进度。交卷或超时后自动评分。

### Q: 数据如何备份？

**A**: 在个人中心 → 学习数据管理 → 导出数据，可将学习记录导出为文件。

### Q: 如何重置学习进度？

**A**: 个人中心 → 学习数据管理 → 重置数据（会清空所有进度，谨慎操作）。

### Q: 题库数据在哪里？可以自己修改吗？

**A**: 所有题库、课程内容存储在 `src/data/` 目录下的 TypeScript 文件中。可以直接编辑 `.ts` 文件增删改题目，修改后重新 `npm run build` 即可生效。

### Q: 平台支持移动端吗？

**A**: 前端基于 React + Tailwind CSS 响应式设计，支持手机浏览器访问。`android/` 目录下提供了 Capacitor 安卓打包方案，可构建为 APK。

### Q: 后端必须启动吗？

**A**: 非必须。大部分功能（课程浏览、Python 代码运行、测验、实验、在线工具箱等）纯前端即可使用。后端仅提供用户注册/登录、多语言后端代码执行、社区发帖等需要持久化的功能。

### Q: 在线工具箱的数据会上传到服务器吗？

**A**: 不会。所有在线工具的计算都在浏览器端完成，纯前端实现，输入的数据完全不经过后端服务器。

### Q: 在线工具箱有多少工具？

**A**: v2.0 提供 **200+ 工具**，分为 12 大类：编码解码、哈希计算、进制转换、安全工具、开发工具、生成器、文本处理、颜色/图像、时间/日期、网络工具、计算工具、其他实用工具。

### Q: 为什么要使用在线工具箱而不是外部网站？

**A**:

1. **离线可用**：纯前端实现，断网也能用
2. **隐私安全**：所有计算在本地完成，不向任何第三方发送数据
3. **快速访问**：直接集成在学习平台中，无需新开标签页
4. **工具全面**：200+ 工具一站式覆盖日常开发与学习需求
5. **一键复制**：统一界面，操作简单
6. **分类清晰**：按功能分类，便于查找

### Q: 如何构建 Android APK？

**A**: 按以下步骤：
1. `npm run build` 构建前端
2. `npx cap sync` 同步到 Android 项目
3. `npx cap open android` 打开 Android Studio
4. 在 Android Studio 中 Build → Build APK
5. 输出位置: `android/app/build/outputs/apk/debug/app-debug.apk`

### Q: 页面崩溃后如何恢复？

**A**: v2.0 内置 ErrorBoundary 组件，遇到错误时会显示友好提示，点击"刷新页面"按钮即可恢复。如果错误重复出现，可在个人中心重置数据或清除浏览器缓存。

### Q: 如何提交 bug 或建议新功能？

**A**: 欢迎在社区交流页面发帖反馈，或直接修改 `src/data/` 下的内容（需开发环境）。
