# 第三章 安全事件检测与研判

> 第3章 | 55页

3.1 安全事件监测手段

- 安全设备告警：防火墙、IDS/IPS、WAF、EDR
- 日志分析：系统日志、应用日志、安全日志
- 流量分析：异常流量、恶意通信、C2连接
- 威胁情报：IOC匹配、威胁狩猎

3.2 常见安全事件类型

- 恶意代码感染：病毒、木马、勒索软件
- 网络攻击：DDoS、SQL注入、XSS
- 数据泄露：数据窃取、数据泄露
- 未授权访问：账号被盗、权限提升
- 内网渗透：横向移动、域控攻陷

3.3 安全事件研判方法

- 事件真实性验证
- 影响范围评估
- 危害程度分析
- 事件定级（一般、较大、重大、特别重大）

3.4 误报排除技巧

- 告警上下文分析
- 多源数据交叉验证
- 白名单机制
- 基线对比分析
