### 16.1.2 不同平台的排名机制

**积分制排名**

大多数SRC平台采用积分制排名，通过漏洞数量和质量计算积分。

```markdown
## 积分计算规则示例

### 基础积分
- 严重漏洞：100积分
- 高危漏洞：50积分
- 中危漏洞：20积分
- 低危漏洞：5积分

### 加成系数
- 首个发现：1.5倍积分
- 重复漏洞：0积分
- 优质报告：1.2倍积分
- 快速响应：1.1倍积分

### 积分计算公式
总积分 = Σ(漏洞积分 × 加成系数)

### 示例计算
白帽子A本月提交：
- 2个严重漏洞（首个发现）：100 × 1.5 × 2 = 300积分
- 3个高危漏洞：50 × 3 = 150积分
- 5个中危漏洞：20 × 5 = 100积分
- 优质报告加成：(300+150+100) × 1.2 = 660积分

月度总积分：660积分
```

#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
漏洞报告生成工具
自动生成标准格式的漏洞报告
"""

import datetime
import json

class ReportGenerator:
    """漏洞报告生成器"""
    
    def __init__(self):
        self.templates = {
            'sqli': self._sqli_template(),
            'xss': self._xss_template(),
            'upload': self._upload_template(),
            'auth_bypass': self._auth_bypass_template(),
        }
    
    def _sqli_template(self):
        """SQL注入报告模板"""
        return """# 漏洞报告：{system_name} SQL注入漏洞

## 基本信息
| 项目 | 内容 |
|-----|------|
| 漏洞标题 | {system_name} {module} SQL注入漏洞 |
| 漏洞类型 | SQL注入 |
| 危害等级 | {severity} |
| 发现时间 | {found_time} |
| 报告时间 | {report_time} |

## 漏洞概述
{system_name} {module} 存在SQL注入漏洞，攻击者可通过 {param} 参数注入恶意SQL语句，
获取数据库敏感信息。

## 漏洞详情
### 漏洞位置
- 目标URL：{url}
- 参数名称：{param}
- 请求方法：{method}

### 技术细节
- 注入类型：{inject_type}
- 数据库类型：{db_type}
- 数据库版本：{db_version}

## 复现步骤
{reproduce_steps}

## PoC
```python
{poc_code}
```

## 危害分析
{harm_analysis}

## 修复建议
{fix_suggestion}
"""
    
    def generate_report(self, vuln_type, data):
        """生成报告"""
        template = self.templates.get(vuln_type)
        if not template:
            return None
        
        # 填充时间
        data['found_time'] = datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')
        data['report_time'] = datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')
        
        # 生成报告
        report = template.format(**data)
        return report
    
    def save_report(self, report, filename):
        """保存报告"""
        with open(filename, 'w', encoding='utf-8') as f:
            f.write(report)
        print(f"[+] 报告已保存: {filename}")

# 使用示例
if __name__ == "__main__":

        'poc_code': '# PoC代码',
        'harm_analysis': '可获取用户敏感数据',
        'fix_suggestion': '使用参数化查询'

            "'",
            "1'",
            "1 AND 1=1",
            "1 AND 1=2",
            "1' AND '1'='1",
            "1' AND '1'='2",
        ]
        
        for payload in payloads:
            test_url = f"{url}?{param}={payload}"
            try:
                resp = self.session.get(test_url, timeout=self.timeout)
                if "error" in resp.text.lower() or "sql" in resp.text.lower():
                    self.results.append({
                        'type': 'SQL注入',
                        'url': url,
                        'param': param,
                        'payload': payload,
                        'evidence': '错误信息'
                    })
                    return True
            except:
                pass
        return False
    
    def test_xss(self, url, param):
        """测试XSS"""
        payloads = [
            "<script>alert(1)</script>",
            "<img src=x onerror=alert(1)>",
            "<svg onload=alert(1)>",
        ]
        
        for payload in payloads:
            test_url = f"{url}?{param}={payload}"
            try:
                resp = self.session.get(test_url, timeout=self.timeout)
                if payload in resp.text:
                    self.results.append({
                        'type': 'XSS',
                        'url': url,
                        'param': param,
                        'payload': payload,
                        'evidence': 'Payload出现在响应中'
                    })
                    return True
            except:
                pass
        return False
    
    def test_path_traversal(self, url, param):
        """测试路径遍历"""
        payloads = [
            "../../../etc/passwd",
            "..\\..\\..\\windows\\win.ini",
            "....//....//....//etc/passwd",
        ]
        
        for payload in payloads:
            test_url = f"{url}?{param}={payload}"
            try:
                resp = self.session.get(test_url, timeout=self.timeout)
                if "root:" in resp.text or "[extensions]" in resp.text:
                    self.results.append({
                        'type': '路径遍历',
                        'url': url,
                        'param': param,
                        'payload': payload,
                        'evidence': '敏感文件内容'
                    })
                    return True
            except:
                pass
        return False
    
    def scan_url(self, url, params):
        """扫描单个URL"""
        for param in params:
            self.test_sqli(url, param)
            self.test_xss(url, param)
            self.test_path_traversal(url, param)
    
    def batch_scan(self, urls_params):
        """批量扫描"""
        with ThreadPoolExecutor(max_workers=10) as executor:
            for url, params in urls_params:
                executor.submit(self.scan_url, url, params)
    
    def get_results(self):
        """获取结果"""
        return self.results
    
    def save_results(self, filename):
        """保存结果"""
        import json
        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(self.results, f, ensure_ascii=False, indent=2)
        print(f"[+] 结果已保存: {filename}")

### 16.11.1 积分制排名详解

```markdown
## 积分制排名深度解析

### 基础积分规则

#### 漏洞等级积分
- 严重漏洞：100积分
- 高危漏洞：50积分
- 中危漏洞：20积分

- 数据敏感程度：敏感数据+15积分

### 加成系数

#### 发现加成
- 首个发现：1.5倍积分
- 重复发现：0积分
- 独立发现：1.2倍积分

#### 报告加成

- 正常响应：1.0倍积分
- 延迟响应：0.9倍积分

总积分 = Σ(基础积分 × 影响因素 × 发现加成 × 报告加成 × 响应加成)

- 快速响应：1.1倍

### 16.12.1 自动化脚本开发

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
自动化漏洞测试框架
支持多种漏洞类型的批量测试
"""

import requests
import re
import json
import threading
from concurrent.futures import ThreadPoolExecutor
from urllib.parse import urljoin, parse_qs

class VulnerabilityScanner:
    """漏洞扫描器"""
    
    def __init__(self, threads=10, timeout=10):
        self.threads = threads
        self.timeout = timeout
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) Chrome/120.0.0.0'
        })
        self.results = []
        self.lock = threading.Lock()
    
    def test_sqli(self, url, params):
        """测试SQL注入"""
        payloads = [
            "'",
            "1'",
            "1 AND 1=1",
            "1 AND 1=2",
            "1' AND '1'='1",
            "1' AND '1'='2",
            "' OR '1'='1",
            "' OR 1=1--",
        ]
        
        for param in params:
            for payload in payloads:
                test_url = f"{url}?{param}={payload}"
                try:
                    resp = self.session.get(test_url, timeout=self.timeout)
                    if any(keyword in resp.text.lower() for keyword in ["error", "sql", "mysql", "syntax"]):
                        with self.lock:
                            self.results.append({
                                'type': 'SQL注入',
                                'url': url,
                                'param': param,
                                'payload': payload,
                                'evidence': '错误信息'
                            })
                        return True
                except:
                    continue
        return False
    
    def test_xss(self, url, params):
        """测试XSS"""
        payloads = [
            "<script>alert(1)</script>",
            "<img src=x onerror=alert(1)>",
            "<svg onload=alert(1)>",
            "<body onload=alert(1)>",
            "%3Cscript%3Ealert(1)%3C/script%3E",
        ]
        
        for param in params:
            for payload in payloads:
                test_url = f"{url}?{param}={payload}"
                try:
                    resp = self.session.get(test_url, timeout=self.timeout)
                    if "<script>" in resp.text or "alert(" in resp.text:
                        with self.lock:
                            self.results.append({
                                'type': 'XSS',
                                'url': url,
                                'param': param,
                                'payload': payload,
                                'evidence': 'Payload出现在响应中'
                            })
                        return True
                except:
                    continue
        return False
    
    def test_ssrf(self, url, params):
        """测试SSRF"""
        payloads = [
            "http://127.0.0.1",
            "http://192.168.1.1",
            "http://169.254.169.254",
            "http://localhost",
        ]
        
        for param in params:
            for payload in payloads:
                test_url = f"{url}?{param}={payload}"
                try:
                    resp = self.session.get(test_url, timeout=self.timeout)
                    if resp.status_code != 404:
                        with self.lock:
                            self.results.append({
                                'type': 'SSRF',
                                'url': url,
                                'param': param,
                                'payload': payload,
                                'evidence': f'状态码: {resp.status_code}'
                            })
                        return True
                except:
                    continue
        return False
    
    def test_auth_bypass(self, url, params, session_cookie):
        """测试越权访问"""
        test_session = requests.Session()
        test_session.headers.update({
            'Cookie': session_cookie,
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) Chrome/120.0.0.0'
        })
        
        for param in params:
            # 尝试访问其他用户数据
            for user_id in [1, 100, 999]:
                test_url = f"{url}?{param}={user_id}"
                try:
                    resp = test_session.get(test_url, timeout=self.timeout)
                    if resp.status_code == 200 and 'user' in resp.text.lower():
                        with self.lock:
                            self.results.append({
                                'type': '越权访问',
                                'url': url,
                                'param': param,
                                'payload': str(user_id),
                                'evidence': '可访问其他用户数据'
                            })
                        return True
                except:
                    continue
        return False
    
    def scan_url(self, url, params, session_cookie=None):
        """扫描单个URL"""
        print(f"[*] 扫描: {url}")
        
        self.test_sqli(url, params)
        self.test_xss(url, params)
        self.test_ssrf(url, params)
        
        if session_cookie:
            self.test_auth_bypass(url, params, session_cookie)
    
    def batch_scan(self, urls, session_cookie=None):
        """批量扫描"""
        print(f"[*] 开始扫描，共 {len(urls)} 个目标")
        
        with ThreadPoolExecutor(max_workers=self.threads) as executor:
            for url, params in urls:
                executor.submit(self.scan_url, url, params, session_cookie)
    
    def get_results(self):
        """获取结果"""
        return self.results
    
    def save_results(self, filename):
        """保存结果"""
        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(self.results, f, ensure_ascii=False, indent=2)

**自动化报告生成脚本**

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
自动化报告生成工具
根据扫描结果生成标准格式的漏洞报告
"""

import datetime
import json
from jinja2 import Template

class ReportGenerator:
    """报告生成器"""
    
    def __init__(self):
        self.templates = {
            'sqli': self._get_sqli_template(),
            'xss': self._get_xss_template(),
            'ssrf': self._get_ssrf_template(),
            'auth_bypass': self._get_auth_bypass_template(),
        }
    
    def _get_sqli_template(self):
        """SQL注入报告模板"""
        return Template("""# 漏洞报告：{{ system_name }} SQL注入漏洞

## 基本信息

| 项目 | 内容 |
|-----|------|
| 漏洞标题 | {{ system_name }} {{ module }} SQL注入漏洞 |
| 漏洞类型 | SQL注入 |
| 危害等级 | {{ severity }} |
| 发现时间 | {{ found_time }} |
| 报告时间 | {{ report_time }} |

## 漏洞概述

{{ system_name }} {{ module }} 存在SQL注入漏洞，攻击者可通过 {{ param }} 参数注入恶意SQL语句，获取数据库敏感信息。

## 漏洞详情

### 漏洞位置
- 目标URL：{{ url }}
- 参数名称：{{ param }}
- 请求方法：{{ method }}

### 技术细节
- 注入类型：{{ inject_type }}
- 数据库类型：{{ db_type }}

## 复现步骤

1. 访问 {{ url }}
2. 在 {{ param }} 参数中输入测试Payload
3. 观察返回结果

## PoC

```
{{ poc_code }}
```

## 危害分析

- 可获取数据库敏感信息
- 可修改数据库数据
- 可能导致数据泄露

## 修复建议

1. 使用参数化查询
2. 对输入进行验证
3. 实施最小权限原则

---

报告人：{{ author }}
报告时间：{{ report_time }}
""")
    
    def generate_report(self, vuln_type, data):
        """生成报告"""
        template = self.templates.get(vuln_type)
        if not template:
            return None
        
        data['found_time'] = datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')
        data['report_time'] = datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')
        
        return template.render(**data)
    
    def batch_generate(self, results, output_dir):
        """批量生成报告"""
        for i, result in enumerate(results, 1):
            data = {
                'system_name': '目标系统',
                'module': '未知模块',
                'severity': '高危',
                'url': result['url'],
                'param': result['param'],
                'method': 'GET',
                'inject_type': '未知',
                'db_type': '未知',
                'poc_code': f"{result['url']}?{result['param']}={result['payload']}",
                'author': '白帽子'
            }
            
            report = self.generate_report(result['type'].replace('注入', '').replace('访问', ''), data)
            if report:
                filename = f"{output_dir}/{i:03d}_{result['type']}_{result['param']}.md"
                with open(filename, 'w', encoding='utf-8') as f:
                    f.write(report)
                print(f"[+] 报告已生成: {filename}")

# 使用示例
if __name__ == "__main__":
    generator = ReportGenerator()
    
    # 加载扫描结果
    with open('scan_results.json', 'r', encoding='utf-8') as f:
        results = json.load(f)
    
    # 批量生成报告
    generator.batch_generate(results, 'reports')
