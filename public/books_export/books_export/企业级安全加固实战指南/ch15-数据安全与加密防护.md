## 15.3 密码学基础

### 15.3.1 加密算法分类

```
加密算法分类：

对称加密算法（Symmetric）：
├── 特点：加密解密用同一个密钥
├── 优点：速度快、适合大数据量
├── 缺点：密钥分发困难
├── 常见算法：
│   ├── AES（推荐，128/256位）
│   ├── 3DES（已过时）
│   ├── DES（不安全）
│   ├── Blowfish
│   └── SM4（国密）
└── 应用：文件加密、数据库加密、通信加密

非对称加密算法（Asymmetric）：
├── 特点：公钥加密，私钥解密
├── 优点：无需共享密钥、可签名
├── 缺点：速度慢、不适合大数据
├── 常见算法：
│   ├── RSA（推荐，2048位以上）
│   ├── ECC（椭圆曲线，更高效）
│   ├── DSA（数字签名算法）
│   ├── SM2（国密）
│   └── 量子加密（研究中）
└── 应用：密钥交换、数字签名、证书

哈希算法（Hash）：
├── 特点：单向、不可逆、定长输出
├── 用途：完整性校验、密码存储、指纹
├── 常见算法：
│   ├── SHA-256/SHA-512（推荐）
│   ├── SHA-3
│   ├── MD5（不安全，碰撞）
│   ├── SHA-1（不安全）
│   └── SM3（国密）
└── 应用：文件完整性、密码哈希、数字签名

密钥派生算法（KDF）：
├── 特点：从密码派生加密密钥
├── 常见算法：
│   ├── PBKDF2
│   ├── bcrypt
│   ├── scrypt
│   └── Argon2（推荐，OWASP首选）
└── 应用：密码存储、密钥派生
```

### 15.3.2 算法选择建议

```
安全算法选择建议：

加密：
- 对称加密：AES-256-GCM
- 非对称加密：RSA-4096 或 ECC (secp256r1/secp384r1)
- 国密算法：SM2 + SM4 + SM3

哈希：
- 一般用途：SHA-256
- 高安全：SHA-512
- 国密：SM3

密码哈希：
- 首选：Argon2id
- 次选：bcrypt (cost 12+)
- 兼容：PBKDF2 (SHA-256, 600000+ iterations)

TLS/SSL：
- 协议：TLS 1.2 + TLS 1.3
- 套件：ECDHE-ECDSA-AES128-GCM-SHA256
- 证书：ECC 或 RSA 2048+

避免使用的算法：
- 对称：DES, 3DES, RC4, Blowfish(旧)
- 哈希：MD5, SHA-1
- 非对称：RSA < 2048位
- TLS：SSLv2, SSLv3, TLS 1.0, TLS 1.1
```

```python
# Python 文件加密示例（使用cryptography库）
from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
import base64
import os

def generate_key():
    """生成密钥"""
    return Fernet.generate_key()

def encrypt_file(input_file, output_file, key):
    """加密文件"""
    fernet = Fernet(key)
    with open(input_file, 'rb') as f:
        data = f.read()
    encrypted = fernet.encrypt(data)
    with open(output_file, 'wb') as f:
        f.write(encrypted)

def decrypt_file(input_file, output_file, key):
    """解密文件"""
    fernet = Fernet(key)
    with open(input_file, 'rb') as f:
        encrypted = f.read()
    data = fernet.decrypt(encrypted)
    with open(output_file, 'wb') as f:
        f.write(data)

def key_from_password(password: str, salt: bytes) -> bytes:
    """从密码派生密钥"""
    kdf = PBKDF2HMAC(
        algorithm=hashes.SHA256(),
        length=32,
        salt=salt,
        iterations=600000,
    )
    key = base64.urlsafe_b64encode(kdf.derive(password.encode()))
    return key

# 使用示例
key = generate_key()
encrypt_file('plain.txt', 'plain.enc', key)
decrypt_file('plain.enc', 'decrypted.txt', key)
```

# Python + MySQL 字段加密示例
from cryptography.fernet import Fernet
import pymysql
import base64

class FieldEncryptor:
    def __init__(self, key):
        self.fernet = Fernet(key)
    
    def encrypt(self, plaintext: str) -> str:
        """加密字符串"""
        if plaintext is None:
            return None
        encrypted = self.fernet.encrypt(plaintext.encode())
        return base64.b64encode(encrypted).decode()
    
    def decrypt(self, ciphertext: str) -> str:
        """解密字符串"""
        if ciphertext is None:
            return None
        encrypted = base64.b64decode(ciphertext.encode())
        return self.fernet.decrypt(encrypted).decode()

## 15.6 数据脱敏

### 15.6.1 脱敏方法

```
数据脱敏方法：

1. 替换（Substitution）
   - 用虚构但合理的数据替换
   - 如：姓名→"张三"，手机号→"138****8000"
   - 保持格式一致性

2. 掩码（Masking）
   - 部分显示，部分隐藏
   - 如：银行卡号→"6222 **** **** 1234"
   - 身份证号→"1101**********1234"

3. 混淆（Shuffling）
   - 打乱数据顺序
   - 如：姓名列表随机打乱
   - 保持分布特性

4. 数值变换
   - 数值型数据加减随机值
   - 保持统计特性
   - 如：工资 ±10%随机浮动

5. 截断（Truncation）
   - 只保留部分数据
   - 如：邮箱只显示首字母和域名
   - "z***@example.com"

6. 加密（Encryption）
   - 可逆加密，有权限可解密
   - 适用于需要恢复原文的场景
```

### 15.6.2 常见数据脱敏规则

```python
# Python 数据脱敏函数示例
import re
import hashlib

def mask_phone(phone: str) -> str:
    """手机号脱敏：138****8000"""
    if not phone or len(phone) < 11:
        return phone
    return phone[:3] + '****' + phone[7:]

def mask_id_card(id_card: str) -> str:
    """身份证号脱敏：1101**********1234"""
    if not id_card or len(id_card) < 18:
        return id_card
    return id_card[:4] + '********' + id_card[14:]

def mask_bank_card(card_no: str) -> str:
    """银行卡号脱敏：6222 **** **** 1234"""
    if not card_no or len(card_no) < 16:
        return card_no
    return card_no[:4] + ' **** **** ' + card_no[-4:]

def mask_email(email: str) -> str:
    """邮箱脱敏：z***@example.com"""
    if not email or '@' not in email:
        return email
    name, domain = email.split('@', 1)
    if len(name) <= 1:
        return '*@' + domain
    return name[0] + '***@' + domain

def mask_name(name: str) -> str:
    """姓名脱敏：张* 或 李**"""
    if not name:
        return name
    if len(name) <= 1:
        return '*'
    if len(name) == 2:
        return name[0] + '*'
    return name[0] + '*' * (len(name) - 1)

def mask_address(address: str) -> str:
    """地址脱敏：保留到区/县级"""
    if not address:
        return address
    # 简单实现：保留前10个字符
    if len(address) > 10:
        return address[:10] + '***'
    return address + '***'

def hash_data(data: str, salt: str = '') -> str:
    """哈希脱敏（不可逆，但可用于匹配）"""
    if not data:
        return data
    return hashlib.sha256((data + salt).encode()).hexdigest()

# 使用示例
print(mask_phone('13800138000'))      # 138****8000
print(mask_id_card('110101199001011234'))  # 1101**********1234
print(mask_email('zhangsan@example.com'))  # z***@example.com
print(mask_name('张三'))                   # 张*
```

## 15.8 备份与恢复

### 15.8.1 备份策略

```
备份3-2-1原则：

3份数据副本
2种不同存储介质
1份异地存储

备份类型：
├── 完全备份（Full Backup）
│   ├── 备份所有数据
│   ├── 恢复快
│   ├── 占用空间大
│   └── 适合小数据量
├── 增量备份（Incremental）
│   ├── 只备份上次备份后变化的
│   ├── 节省空间
│   ├── 恢复慢（需要所有增量）
│   └── 适合大数据量
├── 差异备份（Differential）
│   ├── 备份上次全量后变化的
│   ├── 恢复只需全量+最后差异
│   ├── 空间介于全量和增量之间
│   └── 折中选择
└── 连续数据保护（CDP）
    ├── 实时记录变化
    ├── 可恢复到任意时间点
    ├── 占用空间大
    └── 核心系统使用

备份保留策略：
├── 每日备份：保留30天
├── 每周备份：保留12周
├── 每月备份：保留12个月
├── 每年备份：永久保留
└── 遵循合规要求
```

MYSQL_USER="backup"
MYSQL_PASS="backup_password"
FILES_TO_BACKUP=(
    "/etc"
    "/home"
    "/var/www"
    "/opt/data"
)

# 确保备份目录存在
mkdir -p "$BACKUP_DIR"

echo "===== 备份开始: $DATE ====="

# 1. MySQL数据库备份
echo "[1/3] 备份MySQL数据库..."
mysqldump -u$MYSQL_USER -p$MYSQL_PASS --all-databases --single-transaction --routines --triggers | \
    gzip > "$BACKUP_DIR/mysql_$DATE.sql.gz"

# 2. 文件备份
echo "[2/3] 备份文件..."
for dir in "${FILES_TO_BACKUP[@]}"; do
    if [ -d "$dir" ]; then
        filename=$(echo "$dir" | tr '/' '_' | sed 's/^_//')
        tar czf "$BACKUP_DIR/${filename}_$DATE.tar.gz" "$dir"
    fi
done

# 3. 配置文件加密（可选）
echo "[3/3] 加密备份文件..."
# openssl enc -aes-256-cbc -salt -in "$BACKUP_DIR/mysql_$DATE.sql.gz" \
#     -out "$BACKUP_DIR/mysql_$DATE.sql.gz.enc" -k "encryption_password"
# rm -f "$BACKUP_DIR/mysql_$DATE.sql.gz"

# 清理旧备份
echo "清理 $KEEP_DAYS 天前的备份..."
find "$BACKUP_DIR" -type f -mtime +$KEEP_DAYS -delete

# 备份结果统计
echo ""
echo "===== 备份完成 ====="
echo "备份目录: $BACKUP_DIR"
echo "备份文件列表:"
ls -lh "$BACKUP_DIR"/*_$DATE* 2>/dev/null
echo ""
echo "当前备份总大小:"
du -sh "$BACKUP_DIR"

# 发送通知（可选）
