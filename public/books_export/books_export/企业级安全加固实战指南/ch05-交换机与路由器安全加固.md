# 第五章 交换机与路由器安全加固

## 5.1 网络设备加固概述

网络设备是整个网络的基础设施，一旦被攻破，攻击者可以监控、篡改所有经过的流量，甚至直接接管整个网络。网络设备的安全加固是纵深防御的重要环节。

### 5.1.1 网络设备安全风险

**常见攻击面：**
```
┌─────────────────────────────────────────┐
│           网络设备攻击面                │
├─────────────────────────────────────────┤
│ 管理面攻击                               │
│ ├── 弱口令暴力破解                       │
│ ├── 管理端口暴露公网                     │
│ ├── 默认凭证未修改                       │
│ └── 管理协议明文传输（Telnet）            │
├─────────────────────────────────────────┤
│ 控制面攻击                               │
│ ├── 路由协议欺骗（BGP、OSPF）             │
│ ├── ARP欺骗与中间人攻击                   │
│ ├── DHCP欺骗攻击                         │
│ ├── STP生成树攻击                        │
│ └── VLAN跳跃攻击                         │
├─────────────────────────────────────────┤
│ 数据面攻击                               │
│ ├── MAC地址洪水攻击                      │
│ ├── DDoS流量攻击                         │
│ ├── 流量嗅探                             │
│ └── 流量劫持                             │
├─────────────────────────────────────────┤
│ 设备本身漏洞                             │
│ ├── 固件漏洞利用                         │
│ ├── 缓冲区溢出                           │
│ ├── 未授权访问漏洞                       │
│ └── 后门账户                             │
└─────────────────────────────────────────┘
```

### 5.1.2 加固整体框架

```
网络设备安全加固三层架构：

管理平面（Management Plane）
├── 安全管理访问（SSH替代Telnet）
├── 强身份认证（AAA、TACACS+、RADIUS）
├── 权限分级（Role-Based Access Control）
├── 操作审计（Syslog、命令审计）
└── 管理平面保护（限制管理IP）

控制平面（Control Plane）
├── 路由协议安全认证
├── STP安全增强
├── ARP安全防护
├── DHCP Snooping
├── IP Source Guard
└── 控制平面限速

数据平面（Data Plane）
├── 端口安全（Port Security）
├── VLAN安全隔离
├── ACL访问控制列表
├── 流量过滤与限速
└── MAC地址安全
```

## 5.2 管理安全加固

### 5.2.1 禁用不安全的管理协议

**Cisco IOS 配置：**
```ios
! 禁用Telnet，仅启用SSH
line vty 0 15
 transport input ssh
 transport output ssh
 exec-timeout 15 0
 privilege level 15
 logging synchronous
 login local
exit

! 禁用HTTP管理，启用HTTPS
no ip http server
ip http secure-server
ip http secure-port 8443
ip http access-class 1 in

! 禁用不必要的服务
no service pad
no service finger
no service tcp-small-servers
no service udp-small-servers
no ip bootp server
no ip domain-lookup
! 注意：如果需要DNS解析，保留ip domain-lookup

! 禁用CDP（根据实际需求，邻居发现协议）
no cdp run
! 或在不需要的接口单独禁用
interface GigabitEthernet0/1
 no cdp enable
exit

! 禁用LLDP
no lldp run
```

**华为/H3C 配置：**
```
# 禁用Telnet服务器
undo telnet server enable

# 启用SSH服务器
ssh server enable
ssh user admin
ssh user admin authentication-type password
ssh user admin service-type stelnet

# VTY线路配置
user-interface vty 0 4
 authentication-mode aaa
 protocol inbound ssh
 idle-timeout 15 0
 screen-length 30
 history-command max-size 256
quit

# 禁用HTTP管理
undo http server enable

# 启用HTTPS管理
http secure-server enable
http secure-port 8443
```

### 5.2.2 SSH安全配置

**Cisco IOS SSH加固：**
```ios
! 生成RSA密钥（至少2048位）
crypto key generate rsa general-keys label SSH-KEY modulus 4096

! 配置SSH版本为2
ip ssh version 2

! SSH认证重试次数
ip ssh authentication-retries 3

! SSH超时时间
ip ssh time-out 60

! 启用SSH密钥认证（可选）
ip ssh pubkey-chain
  username admin
    key-string
      <粘贴公钥内容>
    exit
  exit
exit

! 限制SSH访问的源IP
access-list 10 permit 192.168.1.0 0.0.0.255
access-list 10 permit 10.0.0.0 0.0.0.255
access-list 10 deny any

line vty 0 15
 access-class 10 in
 transport input ssh
exit

! 启用登录失败延迟（防暴力破解）
login delay 2
login block-for 120 attempts 5 within 60
```

**华为SSH加固：**
```
# 生成本地密钥对
rsa local-key-pair create
# 密钥长度选择2048或更高

# SSH版本2
ssh server version 2

# SSH认证重试次数
ssh server authentication-retries 3

# SSH超时
ssh server timeout 60

# 配置SSH用户
ssh user admin
ssh user admin authentication-type password
ssh user admin service-type all
ssh user admin sftp-directory flash:

# ACL限制SSH访问
acl number 2010
 rule permit source 192.168.1.0 0.0.0.255
 rule deny source any
quit

# VTY应用ACL
user-interface vty 0 4
 acl 2010 inbound
 protocol inbound ssh
quit
```

### 5.2.3 本地账户与AAA认证

**本地账户安全（Cisco）：**
```ios
! 使用强密码加密（类型5或更高）
enable secret 5 <加密后的密码>
username admin privilege 15 secret 5 <加密后的密码>
username readonly privilege 7 password <密码>

! 密码加密存储
service password-encryption

! 配置密码策略（新版本IOS支持）
security passwords min-length 12
security passwords history 5
security authentication failure rate 3 log
```

**AAA认证配置（Cisco + TACACS+）：**
```ios
! 启用AAA
aaa new-model

! TACACS+服务器配置
tacacs-server host 192.168.1.100 key <共享密钥>
tacacs-server host 192.168.1.101 key <共享密钥>

! 认证方法列表
aaa authentication login default group tacacs+ local

! 授权方法列表
aaa authorization exec default group tacacs+ local
aaa authorization commands 15 default group tacacs+ local

! 计费方法列表
aaa accounting exec default start-stop group tacacs+
aaa accounting commands 15 default start-stop group tacacs+

! 登录失败处理
aaa authentication attempts login 3
aaa authentication attempts username 3
```

**华为AAA配置：**
```
# 启用AAA
aaa

# 配置RADIUS服务器模板
radius-server template radius_group
 radius-server authentication 192.168.1.100 1812
 radius-server shared-key cipher <共享密钥>
 radius-server retransmit 3
 radius-server timeout 5
quit

# 配置认证方案
authentication-scheme auth_scheme
 authentication-mode radius local
quit

# 配置授权方案
authorization-scheme author_scheme
 authorization-mode radius local
quit

# 配置计费方案
accounting-scheme acc_scheme
 accounting-mode radius
 accounting start-fail online
quit

# 域下绑定方案
domain default
 authentication-scheme auth_scheme
 authorization-scheme author_scheme
 accounting-scheme acc_scheme
 radius-server radius_group
quit
```

### 5.2.5 SNMP安全配置


## 5.3 端口安全加固

### 5.3.1 端口基础安全配置

**Cisco 接入端口安全配置：**
```ios
! 接入端口标准配置模板
interface range GigabitEthernet0/1 - 48
 ! 设置为接入模式
 switchport mode access
 ! 关闭DTP协商
 switchport nonegotiate
 ! 配置端口安全
 switchport port-security
 switchport port-security maximum 2
 switchport port-security mac-address sticky
 switchport port-security violation restrict
 ! 启用BPDU防护（防STP攻击）
 spanning-tree bpduguard enable
 ! 启用根防护（接入端口）
 spanning-tree guard root
 ! DHCP Snooping信任端口（仅上联口设为信任）
 ip dhcp snooping limit rate 100
 ! IP源防护
 ip verify source
 ! 动态ARP检测
 ip arp inspection limit rate 100
 ! 风暴控制
 storm-control broadcast level 5.0
 storm-control multicast level 5.0
 storm-control action shutdown
 ! 关闭未使用端口并放入VLAN 999
 shutdown
 switchport access vlan 999
exit

! 上联端口配置
interface GigabitEthernet0/49
 switchport mode trunk
 switchport trunk allowed vlan 10,20,30
 switchport trunk native vlan 999
 switchport nonegotiate
 ! DHCP Snooping信任端口
 ip dhcp snooping trust
 ! DAI信任端口
 ip arp inspection trust
 ! BPDU过滤
 spanning-tree bpdufilter enable
exit
```

### 5.3.2 DHCP Snooping

防止DHCP欺骗攻击：

```ios
! 全局启用DHCP Snooping
ip dhcp snooping
ip dhcp snooping vlan 10,20,30
! 验证DHCP报文的CHADDR字段
ip dhcp snooping verify mac-address
! DHCP报文速率限制（防止DHCP耗尽攻击）
ip dhcp snooping limit rate 100

! 上联口设为信任端口
interface GigabitEthernet0/49
 ip dhcp snooping trust
exit

! 查看DHCP Snooping绑定表
show ip dhcp snooping binding
```

### 5.3.3 动态ARP检测（DAI）

防止ARP欺骗攻击：

```ios
! 全局启用DAI（基于DHCP Snooping绑定表）
ip arp inspection vlan 10,20,30

! 验证源MAC和目标IP
ip arp inspection validate src-mac dst-mac ip

! 上联口设为信任端口
interface GigabitEthernet0/49
 ip arp inspection trust
exit

! 配置ARP速率限制
ip arp inspection limit rate 100 burst interval 1

! 查看DAI统计
show ip arp inspection statistics
```

### 5.3.4 IP Source Guard

防止IP欺骗攻击：

```ios
! 基于DHCP Snooping绑定表过滤IP源地址
interface range GigabitEthernet0/1 - 48
 ip verify source
exit

! 静态IP主机需要手动配置绑定
ip source binding 0011.2233.4455 vlan 10 192.168.10.100 interface GigabitEthernet0/10

! 查看IP源绑定
show ip source binding
```

### 5.3.5 Port Security端口安全

```ios
! 限制端口MAC地址数量
interface GigabitEthernet0/10
 switchport mode access
 switchport port-security
 ! 最大MAC地址数
 switchport port-security maximum 2
 ! 自动学习并保存MAC
 switchport port-security mac-address sticky
 ! 违规模式：shutdown / restrict / protect
 switchport port-security violation shutdown
exit

! 违规自动恢复（可选）
errdisable recovery cause psecure-violation
errdisable recovery interval 300

! 查看端口安全状态
show port-security interface GigabitEthernet0/10
show port-security address
```

## 5.4 VLAN安全加固

### 5.4.1 VLAN基础安全

```ios
! 禁用未使用的端口，放入黑洞VLAN
vlan 999
 name BlackHole
exit

interface range GigabitEthernet0/20 - 48
 switchport mode access
 switchport access vlan 999
 shutdown
exit

! Native VLAN不要使用VLAN 1
interface GigabitEthernet0/49
 switchport trunk native vlan 999
 switchport trunk allowed vlan 10,20,30
exit

! 管理VLAN独立
vlan 100
 name Management
exit
interface Vlan100
 ip address 10.0.100.1 255.255.255.0
exit

! 禁用VTP（或使用VTP透明模式）
vtp mode transparent
vtp domain SECURE-DOMAIN
vtp password <VTP密码>

! 限制Trunk端口允许的VLAN
interface range GigabitEthernet0/49 - 52
 switchport trunk allowed vlan 10,20,30,100
exit
```

### 5.4.2 防止VLAN跳跃攻击

```ios
! 接入端口设置为接入模式，禁止Trunk协商
interface range GigabitEthernet0/1 - 48
 switchport mode access
 switchport nonegotiate
exit

! Trunk端口明确指定模式，关闭DTP
interface GigabitEthernet0/49
 switchport mode trunk
 switchport nonegotiate
exit
```

### 5.4.3 PVLAN私有VLAN隔离

适用于服务器区，服务器之间不能互访，但都能访问网关：

```ios
exit

```

## 5.5 STP生成树安全

### 5.5.1 STP基础加固

```ios
! 启用Rapid PVST+（快速生成树）
spanning-tree mode rapid-pvst

! 配置根桥
spanning-tree vlan 10,20,30 root primary
! 或配置优先级
spanning-tree vlan 10 priority 4096

! 启用BPDU防护（接入端口）
spanning-tree portfast bpduguard default

! 启用PortFast（接入端口加速）
spanning-tree portfast default

! 启用根防护（防止非根交换机成为根桥）
interface range GigabitEthernet0/1 - 48
 spanning-tree guard root
exit

! 启用环路防护（防单向链路故障）
spanning-tree loopguard default

! 启用UDLD（单向链路检测）
udld enable
interface range GigabitEthernet0/49 - 52
 udld port aggressive
exit
```

## 5.6 路由协议安全

### 5.6.1 OSPF安全认证

```ios
! OSPF区域启用MD5认证
router ospf 1
 router-id 1.1.1.1
 area 0 authentication message-digest
exit

! 接口配置OSPF认证密钥
interface GigabitEthernet0/0
 ip ospf authentication message-digest
 ip ospf message-digest-key 1 md5 <OSPF认证密钥>
exit
```

### 5.6.2 BGP安全加固

```ios
router bgp 65001
 bgp router-id 1.1.1.1
 
 ! BGP邻居认证
 neighbor 10.0.0.2 remote-as 65002
 neighbor 10.0.0.2 password <BGP密码>
 
 ! 限制前缀数量（防前缀溢出）
 neighbor 10.0.0.2 maximum-prefix 1000 80
 
 ! 过滤前缀列表
 neighbor 10.0.0.2 prefix-list BGP-IN in
 neighbor 10.0.0.2 prefix-list BGP-OUT out
 
 ! TTL安全检测（直连邻居）
 neighbor 10.0.0.2 ttl-security hops 1
exit
```

### 5.6.3 路由过滤与防环

```ios
! 前缀列表过滤
ip prefix-list DEFAULT permit 0.0.0.0/0
ip prefix-list RFC1918 deny 10.0.0.0/8 le 32
ip prefix-list RFC1918 deny 172.16.0.0/12 le 32
ip prefix-list RFC1918 deny 192.168.0.0/16 le 32
ip prefix-list RFC1918 permit 0.0.0.0/0 le 32

! AS路径过滤
ip as-path access-list 1 permit ^65002$
ip as-path access-list 1 deny .*

! 路由策略
route-map BGP-IN permit 10
 match ip address prefix-list RFC1918
 match as-path 1
exit
```

## 5.7 控制平面保护

### 5.7.1 Control Plane Policing (CoPP)

```ios
! CoPP策略配置
! 允许的管理流量
class-map match-all MGMT-TRAFFIC
 match access-group 100
exit

! 路由协议流量
class-map match-all ROUTING-TRAFFIC
 match access-group 101
exit

! ICMP流量
class-map match-all ICMP-TRAFFIC
 match access-group 102
exit

! 所有其他流量
class-map match-all OTHER-TRAFFIC
 match access-group 103
exit

! 策略映射
policy-map CONTROL-PLANE-POLICY
 class MGMT-TRAFFIC
  police 10000000 conform-action transmit exceed-action drop
 class ROUTING-TRAFFIC
  police 5000000 conform-action transmit exceed-action drop
 class ICMP-TRAFFIC
  police 1000000 conform-action transmit exceed-action drop
 class OTHER-TRAFFIC
  police 100000 conform-action transmit exceed-action drop
 class class-default
  drop
exit

! 应用到控制平面
control-plane
 service-policy input CONTROL-PLANE-POLICY
exit

! ACL定义
access-list 100 permit tcp 192.168.1.0 0.0.0.255 any eq 22
access-list 100 permit udp 192.168.1.0 0.0.0.255 any eq 161
access-list 101 permit ospf any any
access-list 101 permit tcp any any eq bgp
access-list 102 permit icmp any any echo-reply
access-list 102 permit icmp any any echo-request
access-list 103 permit ip any any
```

```
  □ 未使用端口已关闭
  □ 禁用DTP自动协商

□ VLAN安全
  □ VLAN 1不用于业务
  □ Native VLAN使用专门VLAN
  □ Trunk端口限制允许VLAN
  □ 管理VLAN独立
  □ VTP透明模式或禁用

□ 路由安全

  □ 路由前缀过滤
  □ TTL安全检测
  □ 最大前缀限制

  □ 日志级别合理
  □ 配置变更审计
```

## 5.11 本章小结

本章全面介绍了交换机与路由器的安全加固：

1. **管理安全**：SSH替代Telnet、强认证、RBAC权限分级、SNMPv3
2. **端口安全**：Port Security、DHCP Snooping、DAI、IP Source Guard
3. **VLAN安全**：VLAN隔离、防跳跃攻击、PVLAN、VTP安全

6. **控制平面**：CoPP策略限制控制平面流量
7. **日志监控**：Syslog、NTP、配置变更审计
8. **设备自身**：固件管理、配置备份、物理安全
