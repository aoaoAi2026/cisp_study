## 2.1 信息收集概述

### 2.1.1 信息收集的重要性

信息收集是渗透测试的第一步，也是最关键的一步。俗话说"知己知彼，百战不殆"，充分的信息收集能够大大提高渗透测试的成功率。

**1. 信息收集的目标**

```
信息收集目标：

1. 扩大攻击面
   - 发现更多可攻击的目标
   - 找到系统的薄弱点
   - 识别潜在的攻击路径

2. 了解目标环境
   - 网络架构
   - 技术栈
   - 安全配置

3. 降低被发现风险
   - 选择正确的攻击时间
   - 使用合适的攻击方法
   - 减少不必要的扫描
```

**2. 信息收集分类**

```
被动信息收集：
├── 不直接接触目标
├── 利用公开渠道获取信息
├── 难以被目标发现
└── OSINT工具

主动信息收集：
├── 直接与目标交互
├── 可能留下痕迹
├── 信息更准确
└── 扫描探测工具
```

### 2.1.2 信息收集流程

```
信息收集流程：

1. 明确目标
   ├── 确定测试范围
   ├── 识别关键资产
   └── 定义优先级

2. 公开情报收集（OSINT）
   ├── 搜索引擎
   ├── Whois查询
   ├── DNS枚举
   └── 社交媒体

3. 主动探测
   ├── 端口扫描
   ├── 服务识别
   ├── 指纹识别
   └── 漏洞扫描

4. 社会工程收集
   ├── 人员信息
   ├── 组织架构
   └── 业务流程

5. 信息整理分析
   ├── 去重归类
   ├── 关联分析
   └── 攻击面绘制
```

### 2.2.3 DNS信息收集

**1. DNS记录类型**

| 记录类型 | 说明 | 用途 |
|----------|------|------|
| A | IPv4地址 | 直接获取IP |
| AAAA | IPv6地址 | IPv6资产发现 |
| CNAME | 别名记录 | 发现关联域名 |
| MX | 邮件服务器 | 邮件系统信息 |
| NS | 域名服务器 | 发现DNS服务器 |
| TXT | 文本记录 | SPF/验证信息 |
| SOA | 授权起始 | DNS配置信息 |
| PTR | 反向解析 | IP反查域名 |

**2. DNS查询命令**

```bash
# 查询A记录
nslookup target.com
dig target.com A

# 查询所有记录
dig target.com ANY

# 查询MX记录
dig target.com MX

# 查询TXT记录（SPF）
dig target.com TXT

# 追踪DNS解析路径
dig +trace target.com

# 反向查询
dig -x 192.168.1.1
```

**3. 子域名发现**

```bash
# 使用dnsenum
dnsenum target.com

# 使用fierce
fierce -dnsserver 8.8.8.8 -dns target.com -wordlist subdomains.txt

# 使用dnsrecon
dnsrecon -d target.com -t brt -D subdomains.txt

# 使用 amass（推荐）
amass enum -passive -d target.com
amass enum -active -d target.com

# 使用sublist3r
sublist3r -d target.com

# 使用Shodan（API查询）
# 需要Shodan API Key
shodan host target.com
```

## 2.3 主动信息收集

### 2.3.1 端口扫描

**1. Nmap基础使用**

```bash
# 基本扫描
nmap target.com

# 指定端口扫描
nmap -p 22,80,443 target.com
nmap -p 1-1000 target.com
nmap -p- target.com  # 全端口扫描

# 服务版本检测
nmap -sV target.com

# 操作系统检测
nmap -O target.com

# 全面扫描
nmap -A target.com

# 快速扫描
nmap -T4 -F target.com
```

**2. Nmap扫描类型**

```bash
# TCP SYN扫描（需要root）
nmap -sS target.com

# TCP连接扫描
nmap -sT target.com

# UDP扫描
nmap -sU target.com

# Ping扫描（主机发现）
nmap -sn target.com/24

# ACK扫描（防火墙检测）
nmap -sA target.com

# 窗口扫描
nmap -sW target.com
```

**3. Nmap高级选项**

```bash
# 躲避IDS/IPS
nmap -f target.com  # 分片
nmap --mtu 24 target.com  # 指定MTU
nmap --scan-delay 1s target.com  # 延迟扫描
nmap --badsum target.com  # 伪造校验和

# 输出选项
nmap -oA output target.com  # 输出所有格式
nmap -oN output.nmap target.com  # 输出正常格式
nmap -oX output.xml target.com  # 输出XML格式
nmap -oG output.gnmap target.com  # 输出Grep格式

# 脚本扫描
nmap --script=vuln target.com  # 漏洞脚本
nmap --script=default target.com  # 默认脚本
nmap --script=http-enum target.com  # 指定脚本
```

**4. Masscan大规模扫描**

```bash
# Masscan快速扫描
masscan -p1-65535 192.168.1.0/24 --rate=10000

# 结合Nmap使用
# 先Masscan快速发现开放端口
masscan -p1-1000 192.168.1.0/24 -oJ masscan.json

# 后Nmap详细扫描
nmap -sV -p$(cat masscan.json | jq -r '.[] | .ports[0] | .port') 192.168.1.0/24
```

### 2.3.2 服务识别与指纹识别

**1. 服务版本识别**

```bash
# Nmap服务版本
nmap -sV target.com

# 使用Nmap脚本
nmap --script=banner target.com

nmap --script=http-title target.com

# 手动Banner获取
nc -nv target.com 80

```

**2. Web指纹识别**

```bash
# Whatweb指纹识别
whatweb target.com
whatweb -a 3 target.com  # 深度扫描

# Wappalyzer指纹识别
# 浏览器插件
# 或命令行
wappalyzer https://target.com

# 手动指纹识别
curl -I target.com  # Headers信息
curl target.com | grep -i keyword  # 页面特征
```

**3. 常见指纹特征**

```
Web服务器指纹：
├── Apache → Server: Apache
├── Nginx → Server: nginx
├── IIS → Server: Microsoft-IIS
└── Tomcat → Server: Apache-Coyote

编程语言指纹：
├── PHP → X-Powered-By: PHP
├── ASP.NET → X-Powered-By: ASP.NET
├── Java → Server: Apache Tomcat/JBoss
└── Python → Set-Cookie: expires=

框架指纹：
├── jQuery → <script>jQuery
├── Bootstrap → <link>bootstrap
├── React → <div id="root">
└── Vue.js → __VUE_DEVTOOLS_GLOBAL_HOOK__
```

### 2.4.2 综合信息收集脚本

```bash
#!/bin/bash
# 信息收集自动化脚本

TARGET=$1
OUTPUT_DIR="recon_$TARGET"
mkdir -p $OUTPUT_DIR

echo "[*] Starting reconnaissance on $TARGET"

# Whois查询
echo "[*] Whois query..."
whois $TARGET > $OUTPUT_DIR/whois.txt

# DNS信息
echo "[*] DNS enumeration..."
dig any $TARGET > $OUTPUT_DIR/dns.txt
dnsenum $TARGET -f /usr/share/wordlists/dnsmap.txt -o $OUTPUT_DIR/dnsenum.xml 2>/dev/null

# 子域名发现
echo "[*] Subdomain enumeration..."
amass enum -passive -d $TARGET -o $OUTPUT_DIR/amass.txt
sublist3r -d $TARGET -o $OUTPUT_DIR/sublist3r.txt 2>/dev/null

# 邮箱收集
echo "[*] Email harvesting..."
theHarvester -d $TARGET -b all -f $OUTPUT_DIR/emails.xml 2>/dev/null

# 端口扫描
echo "[*] Port scanning..."
nmap -sV -T4 -p- -oA $OUTPUT_DIR/nmap $TARGET

# Web指纹
echo "[*] Web fingerprinting..."
whatweb $TARGET --log-xml=$OUTPUT_DIR/whatweb.xml 2>/dev/null

# 目录扫描
echo "[*] Directory enumeration..."
dirb http://$TARGET /usr/share/wordlists/dirb/common.txt -o $OUTPUT_DIR/dirb.txt

echo "[*] Reconnaissance completed!"
echo "[*] Results saved to $OUTPUT_DIR/"
```
