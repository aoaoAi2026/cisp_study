## 8.1 社会工程学是什么？

### 8.1.1 用大白话解释

```
社会工程学 = 利用人的心理弱点来攻击

不是技术漏洞，是人的漏洞。

比如：
  你接到一个电话，对方说：
  "您好，我是公安局的，您涉嫌洗钱，
   请把您的钱转到安全账户接受调查。"

  你一慌，就转钱了...
  这就是社会工程学攻击——电话诈骗。
```

### 8.1.2 为什么社工攻击这么有效

```
因为人有这些弱点：

1. 信任权威
   → 冒充领导、警察、IT部门，大家容易信

2. 乐于助人
   → 别人找你帮忙，一般不会拒绝

3. 恐惧心理
   → "你账户被盗了！""你违法了！"
   → 一慌就容易犯错

4. 贪婪心理
   → "中奖了！""红包！""免费领！"
   → 一贪心就上当了

5. 好奇心
   → "这是什么？""看看再说"
   → 一点开就中招了

6. 从众心理
   → "大家都在弄这个"
   → 跟着就做了

7. 习惯/懒惰
   → 懒得想，别人说啥就是啥
   → 密码都一样，图省事
```

## 8.2 钓鱼邮件——最常见的社工攻击

### 8.2.1 什么是钓鱼邮件

```
钓鱼邮件 = 伪装成正常邮件的诈骗邮件

比如：
  发件人写着 "公司IT部门 <it@company.com>"
  标题写着 "【紧急】您的邮箱密码即将过期，请立即修改"
  内容里有个链接，点进去是个做得很像公司邮箱登录页的假网站
  你一输入账号密码，密码就到骗子手里了
```

### 8.2.2 常见的钓鱼邮件套路

```
套路1：冒充IT/HR部门
  "您的邮箱已满，请登录清理"
  "工资条已发，请查收"
  "年假系统更新，请尽快登录"

套路2：冒充领导
  "这是今年的项目清单，你看看"
  "这个合同急着签，你处理一下"
  "我在开会，你先转笔钱到这个账户"
  （俗称"冒充老板"诈骗）

套路3：冒充快递/外卖
  "您的快递丢了，给您赔偿"
  "外卖放门口了，有问题点链接联系"

套路4：冒充银行/支付平台
  "您的账户有异常交易"
  "您的信用卡逾期了"
  "实名认证需要补充资料"

套路5：中奖/优惠
  "恭喜您中奖了！"
  "限时优惠，只剩最后一天"
  "免费领会员，点击领取"

套路6：恐吓威胁
  "你的秘密我都知道了"
  "不给钱就把你照片发出去"
  "我在你电脑里装了病毒"
  （俗称"敲诈信"）
```

### 8.2.3 一封"标准"的钓鱼邮件长啥样

我们来拆解一封典型的钓鱼邮件：

```
┌──────────────────────────────────────────────────┐
│ 发件人：公司IT部 <it-support@companny.com>       │
│                                             ↑    │
│                                       注意域名！   │
│                                  多了个n，假的    │
│                                                  │
│ 收件人：你 <you@company.com>                      │
│ 标题：【紧急】邮箱密码将于24小时后过期            │
│                                                  │
│ ─────────────────────────────────────────────── │
│                                                  │
│ 尊敬的员工：                                      │
│                                                  │
│   您好！由于公司邮件系统升级，                    │
│ 您的邮箱密码将于24小时后过期。                    │
│                                                  │
│   为避免影响您的正常工作，                        │
│ 请尽快点击下方链接修改密码：                      │
│                                                  │
│  👉 立即修改密码 👈                               │
│     （看起来像个按钮，其实是个链接）              │
│                                                  │
│   如有疑问，请联系IT部门。                        │
│                                                  │
│                         公司IT部门               │
│                         2024年X月X日             │
│                                                  │
└──────────────────────────────────────────────────┘
```

钓鱼邮件的几个特征：

```
1. 发件人地址不对
   公司域名是 company.com，
   它的是 companny.com（多了个n）
   或者用免费邮箱（163、QQ、Gmail...）

2. 制造紧迫感
   "紧急""立即""24小时内""最后一天"
   让你没时间思考，慌慌张张就点了

3. 链接地址不对
   鼠标放链接上看看真实地址
   显示的文字是"公司邮箱"，
   实际链接可能是 xxx.fishing.com

4. 附件可疑
   带个附件，名字很诱人
   "工资条.exe""合同.docm""名单.zip"
   一打开就中毒

5. 语法/格式不对
   错字、格式乱、语气不对
   （当然现在高级的钓鱼邮件写得也很像了）

6. 要你提供敏感信息
   要密码、要银行卡、要身份证号
   正规公司不会通过邮件要你密码的！
```

### 8.2.4 鱼叉式钓鱼（Spear Phishing）

普通钓鱼是广撒网，捞到一个算一个。
鱼叉式钓鱼是精准打击，专门针对某个人。

```
普通钓鱼：
  随便发一万封，总有几个傻的会上当
  成功率低，但量大

鱼叉式钓鱼：
  先收集目标的信息（姓名、职位、公司、爱好...）
  然后专门写一封针对他的邮件
  因为内容很"个性化"，更容易上当
  成功率高多了

比如针对财务人员的：
  冒充老板发邮件，让转钱
  因为知道财务是谁、老板叫什么、公司叫什么
  写得特别真实，很容易上当

针对技术人员的：
  冒充GitHub发邮件，说你项目有bug
  点进去是假的GitHub登录页
  一输密码就没了
```

> 📝 **APT攻击里鱼叉钓鱼是最常用的入口**，
> 很多高级攻击都是从一封精心伪造的钓鱼邮件开始的。

## 8.3 电话社工——靠嘴皮子拿密码

### 8.3.1 什么是电话社工

```
电话社工 = 打电话骗人

冒充各种身份，
从对方嘴里套出密码、信息，
或者让对方做某些操作。
```

### 8.3.2 常见的电话社工套路

```
套路1：冒充IT支持
  "你好，我是IT部门的小王，
   我们在做系统升级，
   需要验证一下你的账号，
   你把密码告诉我一下，我帮你配置。"

  或者：
  "你的电脑中毒了！
   你按我说的操作，我帮你杀毒。
   先按Win+R，输入cmd...
   （其实是在给你装木马）"

套路2：冒充领导
  "小张啊，我是李总，
   我现在在外面开会，
   有个急事要处理，
   你先转5万块钱到这个账户，
   回去我给你签字报销。"
  （就是"猜猜我是谁"的升级版）

套路3：冒充快递/外卖
  "你好你的快递到了，
   地址看不清楚，你说一下地址和姓名？"
  （其实是套你信息）

套路4：冒充公检法
  "你好，我是XX公安局的，
   你涉嫌一起洗钱案，
   请配合调查。
   你把你所有的钱都转到安全账户，
   调查清楚了就还给你。"
  （这个大家都熟，经典诈骗）

套路5：冒充客服
  "你好我是淘宝客服，
   你买的那个东西质量有问题，
   我们给你双倍赔偿，
   你加一下我QQ，我发链接给你操作。"
```

### 8.3.3 电话社工的步骤

```
一个专业的电话社工大概会这么干：

1. 信息收集
   先搞清楚目标公司的信息：
   → 公司叫啥
   → 有哪些部门
   → 员工名字、职位、电话
   → IT部门叫什么
   → 最近有什么事（系统升级、搬家...）
   信息越多，越容易让人相信

2. 准备剧本
   想好要冒充谁，要说什么话
   对方可能会问什么问题，怎么回答
   提前准备好，别露馅

3. 打电话
   找个目标打过去
   一般先打前台或者普通员工
   一步一步套信息
   （不会一上来就找关键人物）

4. 建立信任
   用收集到的信息让对方相信你
   "我是XX部门的小王，
    你们部门李经理让我找你的..."
   （说出对方领导的名字，信任感就上来了）

5. 达成目的
   套密码/套信息/让对方操作
   见好就收，别聊太久聊露馅了
```

> ⚠️ **还是那句话**：
> 这些都是犯罪分子常用的手段，
> 我们了解是为了防范，
> 绝对不能去学了骗人！

## 8.4 物理社工——进到大楼里

### 8.4.1 尾随进入（Tailgating / Piggybacking）

```
尾随进入 = 跟着别人混进去

比如：
  公司大楼门口有门禁，要刷卡才能进。
  你假装是员工，
  跟在一个刷卡的人后面，
  他一开门你就跟着进去了，
  对方一般会帮你挡一下门，
  不会多想。

很多人都有这种"帮人挡门"的礼貌，
  这就被利用了。
```

### 8.4.2 冒充身份

```
穿个西装，带个工牌（假的），
  冒充：
  • IT工程师
  • 修空调的
  • 送快递的
  • 修打印机的
  • 面试的
  • ......

  大公司人多，谁也不认识谁，
  穿着像那么回事，一般没人拦你。
```

### 8.4.3 BadUSB——掉个U盘

```
在公司停车场、厕所、电梯里
  扔几个U盘。

U盘上写着：
  "工资表"
  "机密文件"
  "2024年终奖"
  "私人照片"
  ......

  肯定有人好奇捡起来插电脑上看，
  一插就中毒了！

  这种U盘叫"鸭子"(USB Rubber Ducky)，
  插上去会假装是键盘，
  自动敲命令，几秒就把后门装好了。
```

> 💡 **真实案例**：
> 有安全公司做过实验，
> 在公司园区里扔了一堆U盘，
> 捡起来插电脑上的人超过60%...
> 可见这个攻击的成功率有多高。

## 8.5 怎么防范社工攻击

### 8.5.1 个人防范

```
防社工，记住这几条：

1. 不轻信
   任何人找你要密码，一律不给！
   （真的IT部门不会要你密码的）
   
   任何人让你转钱，一定要核实！
   打电话、当面问，别光靠文字
   
   陌生链接不要点，陌生附件不要开

2. 核实身份
   电话里自称某某部门的，
   你说"我打官方电话回过去"
   他给你的电话别信，自己查官方电话

3. 保持冷静
   凡是说"紧急""立即""马上"的，
   都要多留个心眼，
   骗子就是想让你慌，
   一慌就容易犯错。
   越急的事越要慢下来核实。

4. 保护个人信息
   不要在网上随便晒：
   • 身份证照片
   • 家庭住址
   • 手机号
   • 工作单位
   • 孩子的学校、名字
   信息泄露越多，越容易被精准诈骗。

5. 密码安全
   • 不同网站用不同密码
   • 用密码管理器
   • 开启两步验证（2FA）

6. 多了解常见骗局
   知道骗子有哪些套路，
   就不容易上当了。
   平时多看看反诈宣传，
   国家反诈中心APP了解一下~
```

### 8.5.2 企业防范

```
企业防社工：

1. 安全意识培训
   定期给员工做安全培训，
   告诉他们常见的钓鱼邮件、诈骗手段，
   每年至少来一次钓鱼演练，
   看看有多少人会上当。

2. 制度保障
   • 转账必须多人复核
   • 密码绝不允许通过电话/邮件索要
   • 敏感操作必须二次验证

3. 技术防护
   • 邮件网关（过滤钓鱼邮件）
   • 沙箱（附件自动检测）
   • 域名仿冒监测
   • 终端防护（EDR）

4. 物理安全
   • 严格的门禁制度
   • 访客登记+陪同
   • 摄像头监控
   • 桌面清理政策（人走锁屏、文件收好）

5. 应急响应
   万一有人中招了怎么办？
   有流程，有预案，
   知道找谁，知道怎么处理
   把损失降到最低
```
