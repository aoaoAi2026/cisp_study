## 14.2 无线网络基础

先补点基础，
不然讲攻击你可能听不懂。

### 14.2.1 WiFi 的加密方式

WiFi有哪些加密方式？
主要有这么几代：

```
  1. WEP（Wired Equivalent Privacy）
     • 最早的加密方式
     • 非常不安全，几分钟就能破解
     • 现在基本没人用了
     • 但老设备可能还在用

  2. WPA（Wi-Fi Protected Access）
     • 替代WEP的，比WEP安全
     • 但也有漏洞，现在也不推荐用了

  3. WPA2 / WPA3
     • WPA2是现在最主流的
       大部分WiFi用的都是WPA2-PSK（AES）
     • WPA3是最新的，更安全
       新出的设备开始支持了
     • WPA2也不是绝对安全，
       有KRACK漏洞之类的
     • 但日常用基本没问题
       只要密码够强

  4. 开放网络（无加密）
     • 就是没有密码的WiFi
     • 谁都能连
     • 所有数据明文传输，谁都能抓
     • 最不安全
```

现在大家家用、公司用的，
基本都是 **WPA2-PSK**（预共享密钥，也就是密码）。
WPA3正在普及，
但还没到WPA2那么普遍。

### 14.2.2 无线网卡的模式

做无线渗透测试，
需要你的无线网卡支持一些特殊的功能。
普通的网卡只支持**管理模式（Managed Mode）**，
也就是正常连WiFi用的。

做渗透测试还需要：
```
  • 监听模式（Monitor Mode）
    可以抓周围所有的无线数据包
    不连某个WiFi，也能收到信号
    抓包、破解都需要这个模式

  • 数据包注入（Packet Injection）
    可以发送伪造的数据包
    做Deauth攻击、FakeAP这些都需要
```

不是所有无线网卡都支持监听模式和注入的。
买网卡的时候要注意，
选支持的芯片。
常见的支持比较好的有：
- RT3070
- RTL8187L / RTL8188CUS
- Atheros AR9271
- 等等

买之前查清楚，
别买回来不支持就尴尬了。

Kali本身支持很多无线网卡，
插上去基本就能用。

## 14.3 WEP 破解（了解一下就行）

先简单说说WEP破解。
WEP现在基本淘汰了，
但作为历史，还是了解一下。

WEP为什么不安全？
因为它的加密算法有严重的缺陷，
只要抓到足够多的数据包（IV），
就能算出密码。
快的话几分钟，慢的话几十分钟，
基本都能破。

现在基本没人用WEP了，
但一些老设备、老路由器可能还开着。
如果你看到WEP的WiFi，
那基本等于没加密。

怎么破？
用Aircrack-ng套件，
抓足够的IV，然后aircrack-ng一跑就出来了。
这里就不细讲了，
因为用得太少了。

## 14.4 WPA/WPA2 破解

重点来了——
WPA/WPA2的破解。

现在大多数WiFi都是WPA/WPA2的，
这个大家最关心。

### 14.4.1 WPA 破解原理

WPA/WPA2怎么破解？
它的加密算法本身是安全的，
直接破解加密算法基本不可能。
那怎么破？

**跑字典。**

简单说就是：
1. 先抓到WPA的四次握手包（4-way Handshake）
2. 然后用字典一个一个试密码
3. 哪个密码算出来的结果和握手包里的一致，
   哪个就是正确的密码

用人话说就是：
**拿到了握手包，
然后用字典穷举密码。**

所以WPA破解的关键是两个：
- **能不能抓到握手包**
- **字典够不够好**（密码在不在字典里）

如果WiFi密码是强密码（很长、很复杂），
字典里没有，
那基本就破不出来。
如果是弱密码（比如12345678、手机号、生日），
那就很容易跑出来。

### 14.4.2 抓握手包的过程

怎么抓握手包？
原理是这样的：

当一个设备（比如手机、电脑）
连接WiFi的时候，
会和路由器进行一次"四次握手"，
用来验证密码、协商密钥。
这个握手过程的数据包，
是不加密的，或者说可以被抓到的。
我们把这四个包抓下来，
就可以用来跑字典了。

但是，
如果没有人连接WiFi，
就没有握手包，怎么办？
这时候就要用**Deauth攻击**（取消认证攻击）。
简单说就是：
**给已经连上WiFi的设备发一个伪造的断开包，
把它踢下线。
设备被踢下线之后，
一般会自动重新连接，
这时候就能抓到握手包了。**

有点损，
但很有效。

### 14.4.3 用 Aircrack-ng 破解 WPA 的步骤

具体怎么操作？
用Aircrack-ng套件，
大致步骤是这样的：
（了解一下就行，别拿去干坏事）

```
步骤大致如下：

  1. 把网卡设置为监听模式
     airmon-ng start wlan0

  2. 扫描周围的WiFi
     airodump-ng wlan0mon
     找到目标WiFi的BSSID、信道、ESSID

  3. 抓握手包
     airodump-ng -c 信道 -w 文件名 --bssid 目标BSSID wlan0mon
     开着抓包，等有人连接就能抓到

  4. 如果没人连，就用Deauth踢人
     另开一个终端：
     aireplay-ng -0 10 -a 目标BSSID wlan0mon
     （-0 是Deauth模式，10是发10个包）

  5. 踢完人，设备会自动重连，
     这时候就能抓到握手包了
     airodump-ng上面会显示 "WPA handshake: ..."

  6. 拿到握手包之后，开始跑字典
     aircrack-ng -w 字典文件 抓到的cap文件

  7. 如果密码在字典里，
     跑一会儿就出来了
     显示 "KEY FOUND!"
```

大概就是这么个流程。
还是那句话，
密码在字典里才能破出来，
不在就破不出来。
强密码的WPA2，
还是很安全的。

### 14.4.4 WPA3 呢？

WPA3是最新的WiFi加密标准，
比WPA2安全不少。
针对WPA2的KRACK漏洞什么的，
WPA3都修复了。
而且WPA3用的是SAE握手，
抓握手包跑字典这种方法，
在WPA3上不好使了。

所以现在新设备、新路由器，
尽量开WPA3，
更安全。

当然，
WPA3也不是绝对安全，
也有一些针对WPA3的攻击被研究出来，
但总体来说比WPA2安全多了。

## 14.5 其他无线攻击手段

除了破解WiFi密码，
无线网络还有很多其他攻击手段。
给你介绍几种常见的。

### 14.5.1 邪恶热点 / 钓鱼WiFi（Evil Twin）

第一个，
**邪恶热点（Evil Twin）**，
也叫钓鱼WiFi。
前面故事里讲的就是这个。

什么是邪恶热点？
简单说就是：
**攻击者搭一个跟正常WiFi名字一样的假WiFi，
让受害者连上去。
连上去之后，
受害者所有的上网流量都经过攻击者的设备，
攻击者可以监听、篡改、钓鱼...**

比如酒店里本来有个叫 "Hotel_WiFi" 的WiFi，
攻击者也搭一个同名的，
信号还更强一点。
受害者手机自动连上信号强的那个，
就连到攻击者的假WiFi上了。
然后攻击者想干嘛干嘛。

这就是邪恶热点攻击。
在公共场所（咖啡馆、机场、酒店、商场）
特别常见，
因为很多人到了一个地方，
就搜免费WiFi，
哪个信号好连哪个，
根本不管是不是真的。

连上了邪恶热点会怎么样？
```
  • 你的所有上网数据都可能被监听
  • HTTP明文的内容全能看到（账号密码、聊天记录...）
  • 可以篡改你访问的网页（比如在网页里插木马）
  • 可以做DNS劫持，把你引导到钓鱼网站
  • 可以偷你Cookie，直接登你的账号
  • 等等
```

非常危险。
所以公共场所的免费WiFi，
别乱连。
尤其是不需要密码的开放WiFi，
更要小心。

### 14.5.2 Karma 攻击 / 自动连接攻击

第二个，
**Karma攻击**。

什么意思呢？
很多手机、电脑，
会主动向外广播自己曾经连过的WiFi的名字（SSID），
找这些WiFi在不在附近。
比如你之前连过 "Starbucks_WiFi"，
你的手机就会时不时地问一句：
"Starbucks_WiFi在吗？"

Karma攻击就是利用这个：
**攻击者的假AP，
收到什么SSID的请求，
就回应说"我就是！"
不管你搜的是什么WiFi名字，
它都冒充。
你的手机一看，
"哎，我之前连过的WiFi在这！"
自动就连上去了。**

用户自己可能都没察觉，
手机就自动连上攻击者的WiFi了。
然后就跟邪恶热点一样，
任人宰割。

这个挺阴险的，
因为很多设备都有自动连接的功能，
不知不觉就中招了。

### 14.5.3 Deauth / 干扰攻击

第三个，
**Deauth攻击**，
前面提过了。
就是给目标设备发取消认证的包，
把它从WiFi上踢下去。

这个攻击可以用来：
- 抓握手包（前面说过的）
- 干扰别人上网（一直踢，让人连不上）
- 配合邪恶热点用（把人从正常WiFi踢下去，
  让他自动连到假WiFi上）

Deauth攻击非常容易实施，
效果也很明显。
而且防起来还挺麻烦的。

### 14.5.4 WPS 破解

第四个，
**WPS（Wi-Fi Protected Setup）破解。**

WPS是什么？
就是路由器上那个WPS按钮，
按一下就能连WiFi，
不用输密码。
或者用PIN码连接。

WPS有个漏洞，
PIN码可以被暴力破解，
而且很快。
破出PIN码之后，
就能算出WiFi的密码。
不管你密码设得多复杂，
只要WPS开着，
就可能被破。

所以现在路由器一般默认都关WPS了。
但有些老路由器还开着。
如果你的路由器还开着WPS，
建议关掉。

### 14.5.5 中间人攻击（MITM）

第五个，
**中间人攻击（Man-in-the-Middle, MITM）。**

这个其实不是无线专属的，
但在无线环境下特别容易实现。
尤其是你连了邪恶热点之后，
攻击者天然就是中间人。

中间人攻击能干嘛？
```
  • 嗅探：听你所有的流量
  • 篡改：改你访问的网页、改你下载的文件
  • 注入：在网页里插恶意代码、插广告
  • DNS劫持：把你引导到假网站
  • SSL剥离：把HTTPS降级成HTTP，然后监听
  • 等等
```

非常危险。
尤其是在公共WiFi下，
尽量不要操作敏感的东西，
比如网银、邮箱、重要账号。
要用的话，
用自己的流量，别连公共WiFi。

## 14.6 无线安全防范

讲了这么多攻击，
该讲讲怎么防了。
分两部分说：
普通用户怎么防？
企业怎么防？

### 14.6.1 普通用户的无线安全

```
  1. 密码设强一点
     • WiFi密码别太简单
     • 别用12345678、手机号、生日这种
     • 尽量长一点、复杂一点
     • 大小写+数字+符号，12位以上最好
     • 强密码的WPA2，基本破不了

  2. 用 WPA2 / WPA3 加密
     • 别用WEP，等于没加密
     • 尽量用WPA2或者WPA3
     • WPA3更好
     • 加密方式选AES，别选TKIP

  3. 关掉 WPS
     • WPS不安全，容易被破解
     • 路由器里把WPS功能关了

  4. 隐藏SSID？作用不大
     • 很多人喜欢隐藏WiFi名字（隐藏SSID）
     • 其实作用不大，想找还是能找到
     • 只能防一下小白
     • 不是什么有效的安全手段

  5. MAC地址过滤？也一般
     • MAC地址过滤也能防防小白
     • 但MAC地址可以伪造，
       想绕还是能绕过去
     • 可以开着增加点门槛，但别指望它

  6. 公共WiFi别乱连
     • 外面的免费WiFi，能不连就不连
     • 尤其是不需要密码的开放WiFi
     • 不确定的WiFi别乱连
     • 重要操作（网银、支付、登重要账号）
       用自己的流量，别用公共WiFi

  7. 公共WiFi下用VPN
     • 如果一定要用公共WiFi
       尽量挂个VPN
     • 流量加密，就算被监听也不怕
     • 能有效防中间人攻击

  8. 关闭自动连接未知WiFi
     • 手机别设置成自动连接开放网络
     • 连WiFi前确认一下是不是真的
     • 别什么WiFi都自动连
     • 防止Karma攻击、邪恶热点
```

### 14.6.2 企业的无线安全

企业的无线安全就更复杂了。
大公司一般不会只用WPA2-PSK（一个密码大家用），
而是用 **WPA2-Enterprise / WPA3-Enterprise**，
也就是802.1x认证，
每个用户有自己的账号密码，
或者用证书认证。
这样安全得多。

```
企业无线安全建议：

  1. 用 WPA2/WPA3 企业版（802.1x）
     • 别用预共享密钥（PSK）
     • 每个人自己的账号，或者证书认证
     • 安全性高很多

  2. 访客网络和内部网络隔离
     • 访客WiFi一个VLAN，
       内部办公WiFi一个VLAN
     • 互不相通
     • 防止访客网络出问题影响内网

  3. 无线入侵检测/防御（WIDS / WIPS）
     • 专门的无线安全设备
     • 能发现邪恶热点、Deauth攻击、
       非法AP、非法客户端...
     • 大企业建议上

  4. 物理安全
     • 无线信号别溢出去太多
     • 合理布置AP，调整功率
     • 别让外面也能收到很强的信号
     • 减少攻击面

  5. 定期安全评估
     • 定期做无线渗透测试
     • 看看有没有安全问题
     • 及时修复
```

## 14.8 本章总结

这一章我们讲了无线网络渗透测试，
总结一下：

**WiFi加密方式：**
- WEP —— 极不安全，基本淘汰
- WPA —— 比WEP好，但也有漏洞
- WPA2 —— 现在主流，密码够强的话比较安全
- WPA3 —— 最新，更安全
- 开放网络 —— 等于没加密，最危险

**WPA/WPA2破解：**
- 原理：抓四次握手包，然后跑字典
- 关键：能不能抓到握手包、字典好不好
- 密码在字典里才能破，强密码基本破不了
- Deauth攻击可以把人踢下线，然后抓重连的握手包

**常见的无线攻击：**
- 邪恶热点（钓鱼WiFi）—— 同名假WiFi，骗你连
- Karma攻击 —— 利用自动连接，你搜什么我就是什么
- Deauth攻击 —— 踢人下线、干扰
- WPS破解 —— 破WPS的PIN码拿密码
- 中间人攻击 —— 监听、篡改、劫持你的流量

**无线安全防范（个人）：**
- 强密码
- WPA2/WPA3 + AES
- 关WPS
- 公共WiFi别乱连
- 公共WiFi下用VPN
- 别自动连未知WiFi

**企业无线安全：**
- WPA2/WPA3 企业版（802.1x）
- 访客网络隔离
- WIDS/WIPS
- 物理信号控制
- 定期评估

最后送你一句话：

> **WiFi就像空气，
> 无处不在，
> 但你永远不知道，
> 空气里有没有毒。**
