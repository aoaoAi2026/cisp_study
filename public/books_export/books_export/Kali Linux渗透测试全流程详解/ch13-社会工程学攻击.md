## 13.1 先讲一个故事

### 13.1.1 "一封邮件搞定整个公司"

我有个朋友，
做红队的。
有一次他做一个项目，
目标是一家挺大的公司。
他在外网摸了快一周，
漏洞没找到几个，
外围防守做得挺好的，
打不进去。

怎么办？
他决定换个思路——
搞社工。

他先做信息收集，
通过领英、公司官网、
各种公开渠道，
收集到了公司几十号员工的姓名、
职位、邮箱地址。
还有公司的组织架构、
部门设置、
常用的术语和话术。

然后他做了一个钓鱼邮件。
伪装成公司IT部门发的，
说"公司邮箱系统升级，
请点击以下链接验证您的账号，
否则邮箱将在24小时内停用。"
邮件做得跟真的一样，
logo、签名、格式，
跟公司正规邮件一模一样。
链接指向他搭的一个钓鱼网站，
也做得跟公司邮箱登录页一模一样，
根本看不出区别。

他把这封邮件群发给了收集到的几十个员工。

结果怎么样？
不到一个小时，
就有十几个人点了链接，
输入了自己的邮箱账号密码。
其中还有一个部门经理。

他拿到账号密码之后，
直接登了公司的邮箱系统。
从邮箱里又找到了VPN的地址、
内部系统的链接、
各种内部文档。
再然后，
用VPN登进内网，
一路打穿，
用了不到两天，
整个内网就拿下了。

花了几百万做的安全防护，
一封几十块钱成本的钓鱼邮件，
就给破了。

## 13.2 社会工程学概述

先简单介绍一下社会工程学。

### 13.2.1 什么是社会工程学？

社会工程学，
英文叫Social Engineering，简称社工。
定义有很多，
我给你一个通俗的：

> **社会工程学就是利用人的心理弱点、
> 信任、贪婪、恐惧、好奇等情绪，
> 通过欺骗、诱导、伪装等手段，
> 获取信息、权限或者其他利益的行为。**

简单说就是：
**不攻机器，攻人。**

为什么社工有效？
因为人是最薄弱的环节。
```
  • 人会贪小便宜
  • 人会害怕（怕账号被封、怕违法）
  • 人会好奇（这是什么？点开看看）
  • 人会信任（领导、同事、官方说的都信）
  • 人会偷懒（嫌麻烦，直接点链接）
  • 人会犯错
  • ......
```

只要抓住了人性的弱点，
社工就很容易成功。

### 13.2.2 社工的几个原则

社工攻击有几个常见的原则/心理学原理：

```
  1. 权威原则
     人们倾向于服从权威
     冒充领导、冒充官方、冒充警察...
     对方就容易听你的

  2. 从众原则
     大家都这么做，我也这么做
     比如"已有800人参加..."

  3. 稀缺原则
     东西越少越想要
     "限时优惠""仅剩3个名额"

  4. 喜好原则
     人们容易答应自己喜欢的人
     美女头像、声音好听、夸你几句...
     你就放下戒心了

  5. 承诺与一致原则
     一旦做出了一个小承诺，
     后面更容易答应更大的请求
     一步一步把你套进去

  6. 互惠原则
     人家给了你点好处，
     你就觉得欠人家的，想还
     先给你个小甜头，再提要求
```

社工的很多手法，
都是基于这些心理学原理。
抓住人性的弱点，
成功率就高。

## 13.3 常见的社工手法

社工的手法太多了，
我给你讲几种最常见的。

### 13.3.1 钓鱼攻击（Phishing）

第一个，
也是最常见的——
**钓鱼攻击（Phishing）。**

什么是钓鱼？
简单说就是：
**伪装成可信的一方，
诱导受害者点击链接、
下载附件、
或者输入敏感信息（账号密码、银行卡等）。**

最常见的就是钓鱼邮件。
伪装成银行、
伪装成官方、
伪装成公司IT部门、
伪装成HR...
发个邮件给你，
让你点链接、输密码，
或者下载附件运行。

除了邮件钓鱼，
还有：
- **短信钓鱼（Smishing）** —— 发短信骗你
- **语音钓鱼（Vishing）** —— 打电话骗你
- **网站钓鱼** —— 做个假网站骗你输密码

钓鱼是社工里最常用、
成本最低、
成功率也不低的一种手法。
很多攻击的入口都是钓鱼。

钓鱼的套路也很多：
```
  • 中奖了，点链接领奖
  • 您的账号异常，点链接验证
  • 您有一个快递/包裹，点链接查看
  • 工资条/考勤通知，下载附件查看
  • 领导/同事发的文件，打开看看
  • 发票/合同/订单，下载查看
  • 系统升级/安全警告，点链接操作
  • ......
```

套路层出不穷，
总有一款适合你。

### 13.3.2  pretexting（伪装/ pretext）

第二个，
**伪装欺骗（Pretexting）。**

什么意思呢？
就是**编造一个虚假的身份和场景，
以此来获取对方的信任，
从而套取信息或者让对方做某事。**

比如：
- 冒充IT部门的人，给你打电话，
  说"系统出问题了，需要验证一下你的密码"
  你一告诉他，他就拿到了。
- 冒充快递员，说你有个包裹，
  确认一下你的地址和手机号。
- 冒充银行客服，说你的账户异常，
  需要核实信息，问你身份证号、验证码...
- 冒充领导，让财务转钱...（这个就是大名鼎鼎的"杀猪盘"、冒充领导诈骗）

都是通过伪装身份，
获取信任，
然后套信息或者骗钱。

这种手法成功率也很高，
尤其是冒充领导、冒充官方，
很多人一听就怕，
对方说什么都信。

### 13.3.3 诱饵攻击（Baiting）

第三个，
**诱饵攻击（Baiting）。**

什么意思呢？
就是**放一个"诱饵"在那里，
吸引受害者自己上钩。**

最经典的例子就是**U盘攻击**。
找几个U盘，
里面放上病毒木马，
起个诱惑性的名字，
比如"工资表.xlsx.exe"、
"2024年终奖方案.doc.exe"、
"公司机密.doc"，
然后把这些U盘扔在公司停车场、
楼道里、电梯里。
肯定会有人好奇捡起来，
插到电脑上看，
一点开，就中毒了。

你别笑，
这个真的管用。
有人做过实验，
扔几十个U盘在公司附近，
有超过一半都会被人捡起来插电脑上。
好奇心害死猫。

除了U盘，
还有各种"诱饵"：
- 免费下载的"破解软件"，里面藏木马
- 免费的"电影资源"，带病毒
- 共享盘里放个诱惑性的文件
- 等等

都是利用人的贪心和好奇，
让你自己主动去点。

### 13.3.4 尾随和物理入侵

第四个，
**尾随（Tailgating / Piggybacking）
和物理入侵。**

什么是尾随？
就是**跟着有门禁权限的人，
混进公司大楼。**
比如人家刷卡进门，
你跟在后面，
说一句"忘带卡了"，
人家好心帮你挡一下门，
你就进去了。
进去之后，
你就可以在公司里到处晃，
找无人看管的电脑、
找敏感文件、
插个设备...

物理安全很多公司都不重视，
觉得有门禁就安全了。
实际上尾随的成功率非常高，
尤其是上下班高峰期，
人多的时候，
混进去太容易了。

### 13.3.5 水坑攻击（Watering Hole）

第五个，
**水坑攻击（Watering Hole）。**

这个名字挺有意思。
什么是水坑攻击？
就是：
**攻击者不直接攻击目标，
而是攻击目标经常访问的网站，
在网站里植入木马。
目标访问这个网站的时候，
就中毒了。**

就像动物去水坑喝水，
猎人在水坑里下毒一样。
所以叫水坑攻击。

比如：
- 你想攻击某家公司，
  知道他们公司的人经常上某个行业论坛、
  或者某个技术网站。
- 你先把这个网站拿下，
  在网站里挂上木马。
- 等公司的人访问这个网站，
  就中毒了，你就打进公司了。

水坑攻击的好处是：
- 目标主动来访问，隐蔽性强
- 不需要知道目标的邮箱、电话
- 很难追溯

当然，
成本也更高，
因为你得先拿下另一个网站。

### 13.3.6 其他社工手法

社工的手法太多了，
再简单列几个：

```
  • 肩窥（Shoulder Surfing）
    站在人家背后偷看密码、屏幕
    机场、咖啡馆、会议室...很常见

  • 垃圾桶潜水（Dumpster Diving）
    翻人家的垃圾桶
    找扔掉的文件、纸条、快递单...
    经常能找到敏感信息
    别笑，真的有人这么干，而且经常有收获

  • 虚假客服/技术支持
    冒充客服，说你电脑出问题了
    让你装远程协助软件，帮你"修"
    实际是控制你的电脑

  • 情感诈骗
    谈恋爱、交朋友
    取得信任之后骗钱骗信息
    也是社工的一种

  ......还有很多很多
```

社工的手法千变万化，
核心都是一样的——
**攻人，不攻系统。**

## 13.4 信息收集 —— 社工的基础

社工能不能成功，
很大程度上取决于**信息收集**做得好不好。
你对目标了解得越多，
社工的成功率就越高。

社工的信息收集，
收集什么呢？

```
  • 目标的姓名、性别、年龄
  • 职位、公司、部门
  • 邮箱、电话、社交账号
  • 兴趣爱好、生活习惯
  • 家人、朋友、关系网
  • 公司的组织架构、内部术语
  • 常用的系统、软件
  • 最近的活动、项目
  • ......
```

这些信息从哪来？
太多地方了。

```
信息来源：
  • 公司官网（团队介绍、新闻动态...）
  • 招聘网站（招人信息能透露出很多技术栈、组织架构）
  • 社交平台（领英、脉脉、微博、朋友圈...）
  • 码云/GitHub/GitLab等代码托管平台（不小心提交的敏感信息）
  • 网盘、文库、文档分享平台
  • 知乎、贴吧、论坛
  • 电商平台、外卖平台（地址、电话）
  • 各种泄露的数据库（社工库）
  • ......
```

只要你用心找，
能找到的信息比你想象的多得多。
很多人在网上几乎是"裸奔"的，
姓名、电话、住址、工作、
家人、兴趣爱好...
全能找到。

信息收集够了，
再做社工，
事半功倍。
你能叫出对方的名字，
说出他的部门、最近的项目，
他一下就信任你了，
后面的事就好办了。

## 13.6 怎么防范社工攻击？

讲了这么多攻击手法，
最后肯定要讲怎么防。
毕竟，
我们学习社工，
主要是为了防御。

社工怎么防？
说难也难，说简单也简单。
核心就是——**提高安全意识。**

### 13.6.1 个人层面的防范

对个人来说：

```
  1. 提高警惕，不要轻信
     • 陌生人的话别全信
     • 凡是涉及钱、密码、验证码的，
       一定要多方核实
     • 官方不会随便要你的密码和验证码

  2. 不点来路不明的链接，不下载来路不明的文件
     • 邮件里的链接、附件，要谨慎
     • 先确认发件人是不是真的
     • 链接可以先鼠标悬停看看真实地址
     • 下载的文件先查毒再打开
     • 尤其是exe、bat、js、docm、xlsb这些可执行的
       别随便点开

  3. 保护好个人信息
     • 不要在网上随便泄露自己的个人信息
     • 身份证号、手机号、住址、工作单位...
       能不填就不填
     • 不同的网站用不同的密码
       别一个密码走天下
     • 重要账号开两步验证

  4. 遇到可疑情况，多核实
     • 领导让你转钱？打个电话确认一下
     • IT部门要你密码？去IT部门问问
     • 银行说你账户异常？自己打官方电话查
     • 别对方说什么你都信

  5. 保持理性，别被情绪牵着走
     • 恐吓你的（说你违法、说你账号要封）
       别慌，先核实
     • 诱惑你的（中奖、赚钱、免费领）
       别贪，天上不会掉馅饼
     • 好奇的（这是什么？）
       控制点，别什么都点
```

总结一下就是：
**多一分警惕，少一分危险。
不贪、不怕、不轻信。**

### 13.6.2 企业层面的防范

对企业来说，
社工防护更重要。

```
  1. 安全意识培训
     • 定期给员工做安全意识培训
     • 讲常见的社工手法、钓鱼案例
     • 提高员工的安全意识
     • 这是最根本的，也是最有效的

  2. 钓鱼演练
     • 定期自己组织钓鱼演练
     • 看看有多少员工会上当
     • 对上当的员工进行再培训
     • 以练代防，很有效

  3. 技术防护手段
     • 邮件网关：过滤钓鱼邮件、恶意附件
     • 网页过滤：屏蔽已知的钓鱼网站
     • 杀软/EDR：拦截恶意程序
     • 堡垒机、权限最小化：就算进来了也干不了太多
     • 等等

  4. 制度和流程
     • 重要操作（比如转账）要有多重审批
     • 不是谁说一句就能转的
     • 身份核实机制
     • 门禁、访客管理（防尾随、防物理入侵）
     • 等等

  5. 应急响应
     • 万一真的被社工成功了
     • 有应急响应流程
     • 能及时发现、及时处置
     • 把损失降到最低
```

企业防社工，
要**人+技术+管理**三者结合。
光靠技术不行，
光靠人也不行，
得结合起来。

---

## 13.7 社工的道德与法律

最后必须讲一讲道德和法律问题。

### 13.7.1 社工是把双刃剑

社工是一把非常锋利的双刃剑。
用在正道上，
比如渗透测试、安全研究、
帮助企业发现人的弱点、
提高安全意识，
是好事。

但如果用在邪道上，
诈骗、窃密、敲诈勒索...
那就是犯罪。
而且社工的隐蔽性强、
成功率高，
危害特别大。

### 13.7.2 合法使用

记住：
**没有授权，
绝对不要对任何人使用社工攻击。**
不管是钓鱼、还是伪装欺骗，
都不行。
都是违法的，
轻则拘留罚款，
重则判刑。

- 学习研究，可以，用自己做实验
- 授权测试，可以，跟客户签好协议
- 未经允许，对别人使用，不行

我们学习社工，
是为了了解攻击手段，
更好地防御，
保护自己、保护企业。
不是为了去害人。

技术没有对错，
错的是使用技术的人。
希望大家都能做正确的事。

## 13.8 本章总结

这一章我们讲了社会工程学，
总结一下：

**什么是社工：**
- 利用人的心理弱点，通过欺骗诱导获取信息/权限
- 不攻机器，攻人
- 人是最薄弱的环节

**常见的社工手法：**
- 钓鱼攻击（邮件钓鱼、短信钓鱼、网站钓鱼...）—— 最常用
- 伪装欺骗（Pretexting）—— 冒充身份套信息
- 诱饵攻击（Baiting）—— 放诱饵等你上钩（U盘攻击等）
- 尾随与物理入侵 —— 混进物理场所
- 水坑攻击 —— 攻目标常去的网站
- 肩窥、垃圾桶潜水... 还有很多

**信息收集是社工的基础：**
- 信息越多，成功率越高
- 公开渠道能找到的信息比你想象的多

**Kali里的社工工具：**
- SET（Social-Engineer Toolkit）—— 社工工具包，功能强大
- BeEF —— 浏览器攻击框架
- Maltego —— 信息收集可视化

**怎么防范：**
- 个人：提高警惕、不轻信、不随便点链接下文件、保护个人信息、多方核实
- 企业：安全意识培训 + 钓鱼演练 + 技术防护 + 制度流程 + 应急响应

**道德与法律：**
- 社工是双刃剑
- 未经授权不要使用
- 学习是为了防御

最后送你一句话：

> **最强的系统，
> 也扛不住一个傻乎乎的用户。
> 但最好的防御，
> 是每个人的安全意识。**
