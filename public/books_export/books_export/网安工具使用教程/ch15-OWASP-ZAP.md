# 第十五章：OWASP ZAP - Web应用安全扫描器

## 15.1 OWASP ZAP 简介

### 什么是 OWASP ZAP？

OWASP ZAP（Zed Attack Proxy）是一个免费、开源的Web应用安全扫描器。

**OWASP ZAP**的特点：
- 免费开源
- 功能全面
- 易于使用
- 支持自动化
- 支持扩展

简单来说，OWASP ZAP是**最好的免费Web安全扫描器**。

### ZAP vs Burp Suite

| 特点 | ZAP | Burp Suite |
|------|-----|------------|
| 价格 | 免费 | 免费版/付费版 |
| 功能 | 全面 | 更全面 |
| 易用性 | 中等 | 较高 |
| 自动化 | 强 | 中等 |
| 扩展 | 有 | 更多 |

## 15.3 Linux 系统安装教程

### Kali Linux（预装）

Kali预装ZAP：
```bash
zaproxy
```

### Ubuntu/Debian 安装

```bash
sudo apt install zaproxy
```

### 下载安装

```bash
wget https://github.com/zaproxy/zaproxy/releases/download/v2.12.0/ZAP_2.12.0_Linux.tar.gz
tar -xzf ZAP_2.12.0_Linux.tar.gz
./zap.sh
```

## 15.5 代理设置详解

### 为什么需要代理？

ZAP作为中间代理，拦截和记录所有HTTP流量。

### ZAP代理配置

默认配置：
- 地址：127.0.0.1
- 端口：8080

修改代理：
1. 点击"Tools" → "Options" → "Local Proxies"
2. 修改地址和端口

### 浏览器代理配置

**Firefox配置：**
1. 设置 → 网络设置
2. 手动代理：127.0.0.1:8080

**Chrome配置：**
```bash
chrome --proxy-server="127.0.0.1:8080"
```

### 证书安装

访问HTTPS网站需要安装ZAP证书：
1. 启动ZAP代理
2. 访问任意HTTPS网站
3. ZAP提示安装证书
4. 或在浏览器导入证书

## 15.6 自动扫描详解

### 快速扫描

1. 在工具栏输入目标URL
2. 点击"Automated Scan"
3. 选择扫描类型
4. 点击"Attack"

### 扫描类型

| 类型 | 说明 |
|------|------|
| Spider | 爬取网站链接 |
| Active Scan | 主动攻击测试 |
| AJAX Spider | 爬取AJAX内容 |

### 扫描结果

扫描完成后，结果显示在"Alerts"标签页。

---

## 15.7 主动扫描详解

### 什么是主动扫描？

主动扫描会向目标发送攻击请求，测试各种漏洞。

### 执行主动扫描

1. 右键点击站点
2. 选择"Attack" → "Active Scan"
3. 配置扫描选项
4. 点击"Start Scan"

### 扫描策略

| 策略 | 说明 |
|------|------|
| Default | 默认策略 |
| Light | 轻量扫描 |
| Medium | 中等扫描 |
| Heavy | 全面扫描 |

---

## 15.8 被动扫描详解

### 什么是被动扫描？

被动扫描分析流量，不发送额外请求。

### 执行被动扫描

被动扫描默认开启，只需手动浏览目标站点即可。

### 扫描结果

在"Passive Scan"标签页查看结果。

---

## 15.9 Fuzzer功能详解

### 什么是Fuzzer？

Fuzzer对参数进行模糊测试，发现可能的漏洞。

### 执行Fuzz

1. 选择要测试的请求
2. 右键选择"Fuzz"
3. 选择要测试的参数
4. 选择Payload
5. 点击"Start Fuzzer"

### Payload类型

| 类型 | 说明 |
|------|------|
| 字符串 | 自定义字符串 |
| 文件 | 字典文件 |
| 数字 | 数字范围 |
| 正则 | 正则生成 |

## 15.10 实战案例：登录页面测试

### 场景说明

测试登录页面的安全性。

### 步骤

1. **配置代理**：设置浏览器代理为ZAP
2. **手动测试**：输入测试用户名密码
3. **分析请求**：在ZAP中查看登录请求
4. **Fuzz测试**：对用户名和密码进行模糊测试
5. **扫描**：执行主动扫描

### 关注点

- SQL注入
- 暴力破解
- 密码明文传输
- 会话管理问题
