## 9.1 先讲一个故事

### 9.1.1 "一个客服电话骗走了几十万"

我给你讲个真实的故事。

某公司财务小王，
某天接到一个电话，
对方自称是"税务局的李科长"，
说他们公司税务有问题，
要查账，
让小王加一下QQ，
发一些资料过去。

小王一听是税务局的，
不敢怠慢，
赶紧加了QQ。
对方把小王拉到了一个QQ群里，
群里还有"公司张总"、"财务刘经理"，
头像和名字都对得上。
张总在群里说，
他在外面开会，
有个紧急的合同要付款，
让小王先转80万到一个账户，
回头补流程。

小王一看是老板发话了，
而且群里还有领导，
也没多想，
就按照张总给的账户转了80万过去。
转完之后，
等了半天没动静，
小王有点慌，
给真的张总打了个电话，
张总说："我哪有让你转钱？
你被骗了！"

小王这才反应过来，
赶紧报警，
但钱已经被转走了，
追回来很难了。

---

你看，
这就是典型的社会工程学攻击——
**冒充领导、伪造场景、
利用心理，
让你在情急之下做出错误的决定。**
不需要什么技术漏洞，
只需要骗过你这个人，
钱就到手了。

这种"冒充领导诈骗"、
"冒充公检法诈骗"，
相信很多人都听说过。
每年都有很多人被骗，
损失惨重。
这就是社会工程学的威力——
**攻破人心，比攻破系统容易多了。**

这一章，
我们就来好好认识一下
社会工程学和钓鱼攻击，
看看它们都是怎么骗人的，
我们又该怎么防。

## 9.2 社会工程学概述

### 9.2.1 什么是社会工程学？

什么是社会工程学（Social Engineering）？
简单说就是：
**利用人的心理弱点，
通过欺骗、诱导、
操纵等手段，
获取信息或者达到某种目的的学问。**

用人话说就是：
**骗你，让你心甘情愿地
把信息告诉我，
或者帮我做事。**

社会工程学攻击的目标不是系统，
不是代码，
而是**人**。
利用人的心理弱点，
比如信任、恐惧、好奇、
贪婪、服从权威、
乐于助人...
这些人性的弱点，
来达到攻击目的。

为什么社会工程学这么有效？
因为**人是安全体系中最薄弱的环节**。
技术可以不断升级，
防火墙可以越来越厚，
但人这个弱点，
很难彻底解决。
而且社会工程学攻击成本低、
见效快、成功率高，
所以被广泛使用。

### 9.2.2 社会工程学的心理学基础

社会工程学为什么能成功？
因为它利用了人性的弱点。
人的大脑有很多"思维捷径"，
也就是认知偏误，
这些偏误平时能帮我们快速做决策，
但也容易被人利用。

常见的被利用的心理弱点：

```
  1. 权威服从
    人们倾向于服从权威
    比如领导、警察、老师、专家...
    他们说的话更容易相信
    冒充领导、冒充公检法
    就是利用这个心理

  2. 信任与熟悉
    对熟悉的人、信任的人
    会放松警惕
    冒充同事、冒充朋友、
    冒充合作伙伴
    就是利用这个

  3. 紧迫感 / 压力
    人在紧急、有压力的情况下，
    理性思考能力会下降
    容易做出错误的决定
    "马上就要！""来不及了！"
    就是给你施压，让你没时间思考

  4. 好奇心
    人都有好奇心
    "这是什么？""快来看！"
    勾引你去点、去看
    钓鱼邮件、钓鱼链接常用

  5. 贪婪
    贪小便宜
    "中奖了！""红包！""免费领！"
    利用人的贪心

  6. 恐惧
    人在害怕的时候会失去判断力
    "你违法了！""账户要冻结了！"
    "再不处理就完蛋了！"
    吓唬你，让你慌神

  7. 乐于助人
    很多人都有乐于助人的天性
    "我电脑坏了，能帮我看看吗？"
    "我账号登不上了，能借你账号用用吗？"
    利用人的善良

  8. 从众心理
    大家都在做的事情，应该没问题吧？
    "大家都领了，你还不领？"
    "已经有一万人参加了"
```

社会工程学攻击，
本质上就是利用这些人性的弱点，
设计套路，
让你上当。
防不胜防。

### 9.2.3 社会工程学的攻击步骤

社会工程学攻击一般分几步？
```
  1. 信息收集
    先收集目标的信息
    姓名、职位、联系方式、
    公司、同事、爱好、
    最近在干嘛...
    信息越多，越容易设计骗局
    （又回到信息收集了，
      信息收集真的很重要）

  2. 建立信任 / 伪装身份
    伪装成某个身份
    比如同事、领导、客户、
    客服、警察...
    或者通过一些操作让你信任他

  3. 利用心理弱点，诱导操作
    设计一个场景，
    利用前面说的各种心理弱点，
    让你在情绪波动下
    做出他想要的操作
    比如转钱、给密码、
    点链接、装软件...

  4. 达成目的，全身而退
    目的达到了，
    赶紧跑路，消失
    等你反应过来已经晚了
```

整个过程，
技术含量不高，
但心理学含量很高。

## 9.3 常见社会工程学攻击手法

社会工程学攻击的手法很多，
我给你讲几种常见的。

### 9.3.1 钓鱼攻击（Phishing）

第一个，
**钓鱼攻击**。
这是最常见、最经典的社工攻击。

什么是钓鱼攻击？
简单说就是：
**伪装成可信的人或机构，
通过邮件、短信、网站等方式，
诱导你点击恶意链接、
下载恶意文件、
或者泄露敏感信息（账号、密码、银行卡...）。**

就像钓鱼一样，
放个饵，
等你上钩。
所以叫钓鱼。

钓鱼攻击的种类很多：
```
  1. 邮件钓鱼
    最常见的
    发钓鱼邮件，
    冒充银行、快递、
    公司HR、IT部门...
    邮件里有恶意链接或者附件
    你一点就中招

  2. 短信钓鱼（Smishing）
    通过短信钓鱼
    "你的快递有问题，点链接查看"
    "你的账户异常，点链接处理"
    链接是钓鱼网站
    你一填账号密码就被骗了

  3. 语音钓鱼（Vishing）
    通过电话钓鱼
    冒充公检法、银行、客服...
    打电话给你，
    忽悠你转账或者泄露信息
    电信诈骗很多都是这种

  4. 网页钓鱼
    做一个假网站，
    跟真的网站一模一样
    比如假的淘宝、假的银行、
    假的公司登录页
    你输入账号密码，
    就到攻击者手里了

  5. 鱼叉式钓鱼（Spear Phishing）
    定向钓鱼
    不是广撒网，
    而是针对特定的人或公司
    提前收集了你的信息，
    伪装得更像，成功率更高
    比如冒充你的领导、同事、客户
    专门给你发邮件
    这种比普通钓鱼难防多了

  6. 鲸鱼攻击（Whaling）
    专门针对高管、大人物的鱼叉式钓鱼
    钓"大鱼"
    因为高管权限大，
      一旦上钩收益高
    所以叫鲸鱼攻击
```

钓鱼攻击是最常见的社工攻击，
也是企业面临的主要威胁之一。
后面我们会专门展开讲。

### 9.3.2 冒充身份

第二个，
**冒充身份**。
就是冒充成某个人，
获取你的信任，
然后骗你。

冒充什么身份？
```
  • 冒充领导
    就是我们开头讲的那种
    冒充老板、上级
    让你转钱、发资料
    财务人员最容易中招

  • 冒充同事
    冒充公司同事
    问你要资料、借账号、
    让你帮忙点个链接...
    "我是新来的XX，能发我一份资料吗？"

  • 冒充客户 / 合作伙伴
    "我是XX公司的，
      能给我发一下你们的产品资料吗？"
    慢慢套取信息

  • 冒充公检法 / 政府部门
    "我是公安局的，
      你涉嫌洗钱，
      把你钱转到安全账户..."
    经典诈骗套路

  • 冒充客服
    冒充银行客服、电商客服、
    快递客服...
    "你的订单有问题，给你退款"
    然后骗你银行卡信息和验证码

  • 冒充IT / 运维
    "我是IT部门的，
      系统升级，
      把你密码告诉我我帮你重置一下"
    骗账号密码

  ......
    什么身份好用就冒充什么
```

冒充身份是社工攻击的基础，
很多社工攻击第一步都是冒充身份。

### 9.3.3 尾随攻击（Tailgating）

第三个，
**尾随攻击**，
也叫"搭便车"。

什么意思？
就是跟着别人混进需要门禁的地方。
比如公司大楼需要刷工牌才能进，
攻击者就在门口等着，
等有员工刷卡进去的时候，
他跟在后面，
假装是一起的，
就混进去了。

很多人好心，
会帮后面的人挡一下门，
结果就把陌生人放进去了。
攻击者进去之后，
就可以在公司里到处逛，
偷资料、装木马、
接内网...
危害很大。

这属于物理层面的社工攻击，
看起来好像很简单，
但成功率其实很高。
因为大家都不好意思拒绝别人，
也不会随便怀疑别人。

### 9.3.4 垃圾桶潜水（Dumpster Diving）

第四个，
**垃圾桶潜水**。
名字很形象，
就是翻垃圾桶。

翻垃圾桶干嘛？
找有用的信息啊。
很多公司把没用的文件、
快递单、旧硬盘、
废弃的办公设备...
直接扔垃圾桶里。
攻击者就去翻，
说不定能翻到：
- 写了密码的便签纸
- 没粉碎的机密文件
- 没销毁的旧硬盘
- 员工信息、联系方式
- 公司内部通讯录
- 等等...

别觉得不可能，
真的有很多公司这么粗心。
老派的黑客特别喜欢干这个，
因为成本低，收获大。

### 9.3.5 诱饵攻击（Baiting）

第五个，
**诱饵攻击**。
放个诱饵，
等你上钩。

最经典的就是**U盘攻击**：
攻击者把带病毒的U盘，
故意丢在公司停车场、
电梯口、卫生间...
U盘上可能还写着"工资表"、
"机密"、"2024年终奖"
之类的字。
你捡到了，
好奇心驱使你插到电脑上看看，
结果U盘里的病毒就运行了，
你的电脑就中招了。

这就是诱饵攻击。
利用人的好奇心和贪心。

除了U盘，
还有什么CD、移动硬盘、
甚至假冒的充电宝...
都可以做诱饵。
你以为你捡到宝了，
其实是个陷阱。

### 9.3.6 借口攻击（Pretexting）

第六个，
**借口攻击**。
就是编造一个虚假的场景和理由，
来骗取你的信息或信任。

比如：
- 冒充调查人员，做问卷调查，
  顺便套取你的个人信息
- 你中奖了，要领奖先交保证金
- 冒充快递员，说你有个快递地址不清楚，
  问你要地址和电话
- 等等...

本质就是编个故事，
让你相信，
然后达到目的。
很多电信诈骗都是这个路数。

## 9.4 钓鱼攻击详解

钓鱼攻击是社工攻击里
最常见、最重要的一种，
我们单独拿一节详细讲讲。

### 9.4.1 钓鱼邮件的常见套路

钓鱼邮件长什么样？
都有哪些套路？

常见的钓鱼邮件套路：
```
  1. 冒充公司内部
    • "您的邮箱即将过期，请立即登录验证"
    • "IT部门通知：请修改您的密码"
    • "您的工资条已生成，请查收"
    • "您有一份新的考勤异常通知"
    伪装成公司HR、IT、行政部门
    用公司内部的名义
    员工一般不会怀疑

  2. 冒充合作方 / 客户
    • "XX公司项目合同，请查收"
    • "最新报价单，附件查看"
    • "订单确认，请查看"
    冒充合作伙伴或者客户
    业务相关的，
    你很可能会点开

  3. 冒充服务提供商
    • "您的账户存在异常，请立即验证"
    • "您的快递已送达，点击查看"
    • "您的发票已开具，附件下载"
    冒充银行、快递、云服务商...
    跟你日常相关的

  4. 利用热点事件
    什么火就冒充什么
    比如疫情期间的"疫苗通知"
    世界杯期间的"中奖通知"
    赶热点，提高点击率

  5. 威胁恐吓
    "您的账户涉嫌违规，即将被冻结"
    "我们检测到您的账号被盗，
      请立即登录修改密码"
    利用恐惧心理，
    让你慌慌张张就点了
```

钓鱼邮件的特点：
```
  • 发件人地址可疑，
    看起来像但仔细看不对
    比如 service@apple-support-scam.com
    而不是 service@apple.com

  • 语气紧急，制造紧迫感
    "立即！""马上！""24小时内！"
    让你没时间思考

  • 链接地址可疑
    鼠标悬停上去看，
    真实地址和显示的不一样
    或者域名是仿冒的

  • 附件可疑
    附件是exe、js、vbs、docm等
    可执行文件或带宏的文件
    正常邮件很少发这些

  • 拼写错误、语法不通
    很多钓鱼邮件是机翻的，
    读起来怪怪的

  • 问候语很通用
    "尊敬的用户"、"亲爱的客户"
    连你名字都不知道
    正规公司一般会称呼你的名字
```

### 9.4.2 钓鱼网站的特点

除了钓鱼邮件，
还有钓鱼网站。
钓鱼网站长什么样？

```
  特点：
    1. 域名是仿冒的
       看起来很像，但仔细看不一样
       比如 taobao.com → ta0bao.com
       把o换成0，不仔细看看不出来
       或者用子域名伪装
       比如 taobao.scam.com

    2. 页面做得跟真的一模一样
       基本上像素级复制
       不看地址根本分不出来

    3. 登录表单是假的
       你输入账号密码，
       就发到攻击者后台了
       然后可能跳转到真网站，
       或者提示密码错误让你再输
       你根本发现不了

    4. 没有HTTPS或者证书有问题
       当然现在很多钓鱼网站也有HTTPS了
       所以这个只能作为参考

    5. 其他功能点不了
       除了登录框，
       其他链接可能都是假的，
       点不动
```

钓鱼网站的迷惑性很强，
尤其是鱼叉式钓鱼，
专门做一个跟你们公司登录页
一模一样的网站，
你根本分不出来。
一定要看地址栏！

### 9.4.3 鱼叉式钓鱼为什么更危险？

前面提到了鱼叉式钓鱼，
为什么它更危险？

因为普通钓鱼是广撒网，
群发几万封邮件，
能骗一个是一个。
伪装比较粗糙，
仔细看能看出来。

但**鱼叉式钓鱼**不一样，
它是**定向的、定制化的**：
- 提前收集了目标的详细信息
- 知道目标叫什么、在哪个公司、
  什么职位、最近在做什么项目
- 邮件里会提到具体的人和事，
  非常有迷惑性
- 伪装成目标熟悉的人（领导、同事、客户）
- 成功率比普通钓鱼高得多

比如，
攻击者知道你正在跟XX公司谈合作，
就冒充XX公司的对接人，
给你发邮件，
说"这是最新的合同，请看一下"，
附件是个木马。
你一看是对接人发来的，
又确实有合作这回事，
很可能就点开了。

这就是鱼叉式钓鱼的可怕之处——
**它是专门为你量身定做的骗局。**

更高级的还有**鲸钓（Whaling）**，
专门钓公司的高管、CEO、CFO...
这些人权限大，
一旦上钩损失惨重。

## 9.5 社会工程学的防御

讲了这么多社工攻击的套路，
那我们怎么防？

社会工程学攻击虽然防不胜防，
但也不是完全没办法。
防御社工攻击，
要从**人、流程、技术**三个层面入手。

### 9.5.1 人的层面：安全意识培训

最重要的，
当然是**提高人的安全意识**。
人是薄弱环节，
那就要把人这个环节加固。

怎么做？
**安全意识培训。**

```
  培训什么？
    • 什么是社会工程学，有哪些套路
    • 怎么识别钓鱼邮件、钓鱼网站
    • 密码安全（强密码、不 reused）
    • 不要随便泄露敏感信息
    • 不要随便点陌生链接和附件
    • 不要随便相信陌生人
    • 遇到可疑情况怎么办
    • 真实案例分析
    ......

  培训方式：
    • 定期培训，新员工入职必训
    • 钓鱼演练（模拟钓鱼，测试员工）
    • 安全知识宣传、海报、文章
    • 案例分享
    • 考试、考核
```

光培训一次没用，
要**持续培训、反复强调**，
还要配合**钓鱼演练**——
公司自己给自己员工发钓鱼邮件，
看看谁会点，
点了的人再针对性培训。
这样效果才好。

安全意识的提升不是一朝一夕的事，
是个长期的过程。
但一旦建立起来，
效果是非常明显的。

### 9.5.2 流程层面：规范与制度

光靠人自觉还不够，
还要有**制度和流程**来约束。

```
  比如：
    • 重要操作必须多方确认
      比如转账，
      不能一个人说了算，
      必须多人审批，
      电话或当面确认
      防止冒充领导诈骗

    • 信息分级授权
      什么人能看什么信息，
      都有规定
      不能随便问就给

    • 身份验证流程
      有人打电话来要敏感信息，
      不能他说什么就是什么
      要走正规的身份验证流程
      比如回拨官方电话确认

    • 门禁与访客管理
      严格的门禁制度
      访客要登记、要有人陪
      不能随便让人进

    • 文档销毁制度
      涉密文件要粉碎，
      旧硬盘要消磁销毁
      不能随便扔

    • 报告与应急流程
      员工发现可疑情况，
      知道该找谁、怎么报
      有畅通的上报渠道
```

用流程和制度来弥补人的不足，
就算有人想犯错误，
流程上也走不通。
这才是根本的解决办法。

### 9.5.3 技术层面：技术防护

技术虽然不是万能的，
但技术能帮我们挡住很多攻击。

```
  邮件安全：
    • 反垃圾邮件网关
    • 钓鱼邮件检测
    • 恶意附件扫描
    • 外部邮件标识
    • 链接重写与检测
    • 发件人认证（SPF/DKIM/DMARC）

  网页安全：
    • 浏览器安全提示
    • 密码管理器（防钓鱼网站）
    • 多因素认证（MFA）
      就算密码被钓鱼偷走了，
      还有第二重验证，进不去
      这个非常有效！

  终端安全：
    • 杀毒软件 / EDR
    • 应用白名单
    • USB管控
    • 屏幕锁定
    • 最小权限

  网络安全：
    • 网络分段
    • 内网访问控制
    • 可疑流量检测
```

技术防护是基础，
能自动挡住很多已知的攻击，
减轻人的压力。

### 9.5.4 个人怎么防社工攻击？

最后给每个人几条实用的建议：

```
  1. 保持警惕，不要轻易相信
     涉及到钱、密码、敏感信息的，
     一定要多留个心眼
     不要别人说什么你就信什么

  2. 注意核实身份
     陌生人来电、来消息，
     涉及敏感操作的，
     一定要核实身份
     比如领导让你转钱，
     打个电话确认一下
     不要怕麻烦

  3. 不要随便点陌生链接和附件
     邮件、短信里的链接和附件，
     不确定的别点、别开
     鼠标悬停看看真实地址
     官方的事情去官方渠道确认

  4. 保护好个人信息
     不要随便在网上泄露个人信息
     社交平台上少发隐私内容
     快递单撕掉再扔
     信息泄露越多，越容易被社工

  5. 用强密码，开启双因素认证
     不同网站不同密码
     重要账号一定要开MFA
     就算密码被钓鱼走了，
     还有第二道防线

  6. 遇到紧急情况冷静
     越是说"紧急"、"马上"的，
     越要冷静
     骗子就是想让你慌，
     慌了就容易出错
     停一停，想一想，再决定

  7. 多了解社工套路
     知道骗子有哪些花招，
     才不容易被骗
     多看看诈骗案例，
     提高警惕性
```

记住一句话：
**天上不会掉馅饼，
也不会掉馅饼。
无缘无故的好处、
突如其来的危险、
素不相识的求助...
都多问几个为什么。**

## 9.6 本章总结

这一章我们讲了社会工程学与钓鱼攻击。
总结一下：

**社会工程学概述：**
- 利用人的心理弱点，通过欺骗达到目的
- 攻击的是"人"，不是系统
- 人是安全体系中最薄弱的环节
- 心理学基础：权威服从、信任、紧迫感、
  好奇心、贪婪、恐惧、乐于助人...

**常见社工攻击手法：**
- **钓鱼攻击**：邮件钓鱼、短信钓鱼、语音钓鱼、
  鱼叉式钓鱼、鲸鱼攻击
- **冒充身份**：领导、同事、客户、公检法、客服、IT...
- **尾随攻击**：跟着混进门禁
- **垃圾桶潜水**：翻垃圾桶找信息
- **诱饵攻击**：U盘等诱饵
- **借口攻击**：编造场景骗取信息

**钓鱼攻击详解：**
- 钓鱼邮件常见套路：冒充内部、冒充客户、
  冒充服务商、热点事件、威胁恐吓
- 钓鱼网站特点：仿冒域名、页面以假乱真、
  假登录表单
- 鱼叉式钓鱼更危险：定向、定制化、成功率高

**防御社工攻击：**
- **人**：安全意识培训 + 钓鱼演练，长期持续
- **流程**：制度规范，多方确认，审批流程
- **技术**：邮件安全、MFA、终端安全...
- 个人：保持警惕、核实身份、不点陌生链接、
  保护隐私、用MFA、遇事冷静

最后送你一句话：

> **最坚固的城堡，
> 往往是从内部被攻破的。
> 而人心，
> 就是那道最容易被攻破的门。**
