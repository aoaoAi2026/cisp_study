## 📖 写在前面

上一章我们讲了外网打点，讲了各种技术层面的突破方法。这一章我们来讲另一种打点方式——钓鱼。

可能你会说："钓鱼不就是发个假邮件吗？有什么好讲的？"

那你可太小看钓鱼了。

钓鱼是红队最常用的打点方式之一，也是成功率最高的方式之一。为什么？因为**人永远是最薄弱的环节。**

技术防护做得再好，只要有一个人点了钓鱼邮件，那一切防护都白搭。

我给你讲个真实的故事：

有一次护网，有个红队去打一个金融公司，技术层面扫了半天，没找到什么漏洞。那家公司安全做得挺不错的，技术上真的没什么突破口。

后来红队决定试试钓鱼。他们先收集了目标公司员工的信息，发现有个IT运维管理员姓李，就给他发了一封邮件。

邮件内容是这样的：

```

我是XX项目的负责人，我们项目组准备上线一个新系统，需要您帮忙开通一下VPN权限。

附件是项目需求文档和权限申请表，麻烦您帮忙处理一下。

谢谢！

项目负责人：张三
2024年1月15日
```

附件是一个Word文档，里面有宏代码，打开就会自动下载木马执行。

李工看到邮件，没怎么怀疑，就打开了附件，然后......

然后红队就拿到了一台机器的权限，进而打穿了整个内网。

**你说，钓鱼厉害不厉害？**

这一章我们就来好好聊聊钓鱼，钓鱼都有哪些玩法，怎么提高上钩率，有哪些注意事项。

## 6.1 钓鱼是个啥？

### 6.1.1 用大白话解释钓鱼

钓鱼，简单说就是：**伪装成别人信任的东西，骗你点链接、下文件、输密码，然后趁机搞事情。**

就像现实中的钓鱼一样——你放个鱼饵，鱼过来吃，一咬钩就被钓上来了。

网络钓鱼也是一个道理：攻击者放个"鱼饵"（假邮件、假网站、假文件...），受害者"上钩"（点了、下载了、输密码了），然后攻击者就得手了。

### 6.1.2 为什么钓鱼这么好用？

为什么钓鱼这么受红队欢迎？原因很简单：

**第一，成功率高。**

你技术打点可能打半天打不进去，钓鱼只要有一个人点了，直接就进去了。而且现在人每天收那么多邮件，总有看走眼的时候。

**第二，成本低。**

不用挖漏洞，不用找0day，只要会写邮件、会做网站就行。成本低，见效快。

**第三，防不胜防。**

技术漏洞可以打补丁，人的漏洞怎么补？你总不能不让员工收邮件吧？你总不能不让员工上网吧？

有句话说得好：**"再厚的防火墙，也防不住一个手贱的员工。"**

**第四，技术不够，社工来凑。**

有时候技术真的打不进去，但钓鱼可以绕过所有技术防护。你防火墙再好、补丁打得再及时，只要有人点击了你的钓鱼邮件，你就进来了。

### 6.1.3 钓鱼的分类

钓鱼的玩法可多了，不是只有发邮件一种。我给你列几种常见的：

```
按方式分：
  • 邮件钓鱼 —— 发假邮件，最经典
  • 短信钓鱼 —— 发假短信（钓鱼链接）
  • 电话钓鱼（vishing）—— 打电话骗
  • 即时通讯钓鱼 —— 微信、QQ、钉钉社工
  • 社交媒体钓鱼 —— 微博、LinkedIn私信

按目标分：
  • 大规模撒网（Spearphishing）—— 广撒网，钓到谁算谁
  • 精准钓鱼 —— 针对特定的人，专门设计

按内容分：
  • 钓鱼链接 —— 骗你点链接，跳到钓鱼网站
  • 钓鱼附件 —— 骗你下载文件，执行恶意代码
  • 钓鱼网站 —— 骗你输入密码，偷账号
  • 语音钓鱼（vishing）—— 骗你电话里说密码
```

## 6.2 邮件钓鱼详解

邮件钓鱼是最经典、最常用的钓鱼方式。

### 6.2.1 邮件钓鱼的基本流程

邮件钓鱼分这么几步：

```
1. 收集目标邮箱
   → 从各种渠道收集目标员工的邮箱

2. 准备钓鱼内容
   → 写钓鱼邮件
   → 做钓鱼网站（可选）
   → 准备恶意附件（可选）

3. 伪装发件人
   → 伪装成公司内部/合作伙伴/权威机构
   → 让邮件看起来可信

4. 发送邮件
   → 通过邮件服务器/第三方服务发送
   → 注意发送频率，别被当垃圾邮件

5. 等鱼上钩
   → 有人点链接了？
   → 有人下载附件了？
   → 有人输入密码了？

6. 收网
   → 马儿上线了！
   → 开始下一步操作
```

### 6.2.2 目标邮箱怎么收集？

收集目标邮箱，是钓鱼的第一步。

**从公开渠道收集：**

1. **公司官网**——很多公司官网有"联系我们"、"团队介绍"页面，上面有邮箱
2. **LinkedIn/脉脉**——搜公司名，能找到员工的邮箱格式
3. **招聘网站**——招聘岗位要求里可能有邮箱
4. **GitHub**——很多员工用公司邮箱注册GitHub
5. **Whois**——有些公司域名的Whois信息里有管理员邮箱

**猜测邮箱格式：**

如果你知道几个人的邮箱，就能猜出格式：
- 如果"张三"的邮箱是 `zhangsan@example.com`
- 那"李四"很可能是 `lisi@example.com`
- "王明明"可能是 `wangmingming@example.com`

常见格式：
```
zhangsan@example.com
zhang.san@example.com
zsan@example.com
zhangsan123@example.com
```

### 6.2.3 钓鱼邮件怎么写？

钓鱼邮件写得好不好，直接决定成功率。

**钓鱼邮件三要素：**

```
1. 标题吸引人
   → 让人想点进来

2. 内容可信
   → 让收件人相信这是真的

3. 行动诱饵
   → 让收件人有理由行动（点链接、下载附件）
```

**好的钓鱼邮件长什么样？**

给你几个模板：

**模板一：工资/奖金主题（高点击率）**

```
标题：2024年度奖金发放通知

正文：

各位同事：

根据公司2024年度绩效考核结果，您的年度奖金已核算完毕，现予以发放。

具体金额请登录OA系统查看：https://xxx-oa.com

登录方式：工号+密码

如有疑问请联系人力资源部。

人力资源部
2024年1月20日
```

看到"奖金"两个字，99%的人都会想点进去看看自己发了多少。

**模板二：系统升级/安全通知（高可信度）**

```
标题：【安全通知】您的邮箱账户存在异常，请立即处理

正文：

尊敬的用户：

我们检测到您的邮箱账户存在异常登录行为：
- 登录时间：2024-01-20 03:45:23
- 登录地点：广东省深圳市
- 登录IP：1.2.3.4

如果这不是您本人的操作，您的账户可能已被盗用。为了保护您的账户安全，请立即点击以下链接验证您的身份：

https://fake-login-site.com/verify

如果不及时处理，您的账户将被锁定。

IT安全中心
2024年1月20日
```

看到"账户异常"、"可能被盗用"，很多人会紧张，就点进去了。

**模板三：项目合作/工作相关（职场人必点）**

```
标题：关于XX项目需求变更的紧急通知

正文：

张经理您好，

抱歉打扰，我是XX项目的甲方代表。收到通知，由于公司战略调整，XX项目需求有重大变更，请尽快查收附件中的需求文档，并在本周五前反馈意见。

如有疑问请随时联系我。

谢谢配合！

XX公司项目负责人：李华
电话：138-xxxx-xxxx
```

职场人都很怕"项目需求变更"，看到这种邮件肯定会点。

**模板四：快递/外卖/物流（生活场景）**

```
标题：您的快递已到达，请注意查收【顺丰速运】

正文：

您好，您的包裹已到达，请注意查收。

快递单号：SF1234567890
收件地址：XX市XX区XX路XX号
收件人：张三

如需查询物流详情，请点击：https://fake-tracking.com

顺丰速运
```

快递、外卖这类和生活相关的，也很诱人。

**钓鱼邮件的注意事项：**

- 用真实的公司名称（如果能仿得像）
- 内容要像正常邮件，不要有明显的错别字、奇怪的语法
- 最好能叫出对方名字，显得更真实
- 发布时间选工作日的早上8-9点或者下午2-3点
- 不要群发太多，一天几十封慢慢发
- 链接和附件要起个好名字，让人想点

### 6.2.4 发件人怎么伪装？

发件人伪装是邮件钓鱼的关键。

如果发件人显示是"xx钓鱼站"，那肯定没人点。但如果是"IT安全中心"，那可信度就高多了。

**常见的伪装方式：**

**1. 伪装公司内部**

如果能收集到公司内部的邮箱，可以伪装成IT部门、HR部门、财务部门等。

```
真实邮箱：admin@example.com
伪装发件人：IT安全中心 <admin@example.com>
```

**2. 伪造显示名称**

很多邮件客户端只显示"显示名称"，不显示完整邮箱。可以：

```
显示名称：IT安全中心
实际邮箱：hacker@evil.com
```

但这种容易被识破，因为完整邮箱会显示出来。

**3. 利用相似域名**

注册一个和目标公司很像的域名：
- 目标是 example.com
- 你注册 exampl3.com（3代替e）
- 或者 example-support.com
- 或者 1example.com（1代替l）

不太注意看真的看不出来。

**4. SPF/DKIM欺骗（高级）**

这个比较复杂，需要邮件服务器配置得当。不展开讲了。

### 6.2.5 钓鱼链接怎么做？

钓鱼邮件里，一般都会有一个链接，让用户点击跳转到钓鱼网站。

**钓鱼链接怎么伪装？**

1. **短链接**——用短链接服务（如 t.cn、bit.ly）把长链接缩短
2. **URL重定向**——利用URL参数跳转，比如 `https://www.example.com/redirect?url=https://钓鱼网站.com`
3. **子域名伪装**—— `https://fake-login-example.com/` 看起来像 `example.com` 的子站
4. **HTML邮件**——在HTML邮件里显示文本是 `www.example.com`，实际链接是 `www.fake.com`

### 6.2.6 钓鱼网站怎么搭？

钓鱼网站是骗用户输入密码的页面。

**钓鱼网站要仿得像：**

1. **克隆真实网站**——用工具（如Social Engineering Toolkit）克隆一个登录页面
2. **细节要像**——logo、标题、输入框、按钮，都要像真的
3. **URL要像**——用相似域名：`https://fake-login-example.com/`
4. **要有功能**——用户输入账号密码之后，要能提交、保存

**钓鱼网站要注意：**

- 要能接收用户输入的数据（一般存到日志文件或者数据库）
- 登录失败要显示错误信息，不能直接报错（要像正常的登录失败）
- 登录成功可以跳转到真实网站（让用户不怀疑）
- 最好加上SSL证书（HTTPS），看起来更正规

**钓鱼网站推荐工具：**

- **Social Engineering Toolkit (SET)**——一键克隆网站
- **PhishLair**——自动化钓鱼网站管理
- **钓鱼云平台**——有些平台提供现成的钓鱼页面模板

## 6.3 恶意附件怎么做？

有些钓鱼邮件不骗密码，而是骗你下载附件执行恶意代码。

### 6.3.1 宏病毒（Office文档）

最经典的就是Word/Excel宏病毒。

**Word宏病毒示例：**

```vb
Sub Auto_Open()
    ' 自动执行的宏，打开文档时运行
    Dim shell As Object
    Set shell = CreateObject("WScript.Shell")
    
    ' 执行PowerShell命令，下载木马并执行
    shell.Run "powershell -windowstyle hidden -enc BASE64编码的命令", 0, False
End Sub
```

用户打开Word文档，宏就自动执行，下载木马到本地运行。

**怎么让用户启用宏？**

1. 文档打开时显示"安全警告：宏已被禁用"，提示用户"点击启用内容"
2. 文档里写"请启用宏以查看完整内容"
3. 伪装成正常的工作文档（报表、合同、需求文档）

### 6.3.2 Lnk文件（快捷方式）

Lnk文件是Windows快捷方式，可以配置为运行任意命令。

创建一个.lnk文件，指向：
```
powershell -windowstyle hidden -enc BASE64编码的命令
```

用户双击.lnk文件，就会执行恶意命令。

### 6.3.3 自解压文件

用WinRAR或者7-Zip创建一个自解压文件：
- 添加正常文件（如一个PDF）
- 添加恶意文件（如木马.exe）
- 配置自解压选项：先打开正常文件，再执行恶意文件

用户解压之后，会看到正常文件（木马在后台偷偷执行）。

### 6.3.4 ISO/IMG镜像

创建一个ISO或IMG镜像文件，里面放：
- 一个正常的PDF文件
- 一个恶意的可执行文件
- 一个自动运行的配置（让恶意程序自动执行）

用户挂载镜像，会看到正常文件，双击会运行恶意程序。

### 6.3.5 如何提高附件的诱惑力？

附件名字很重要：
- 工资调整方案.docm
- 项目需求变更通知.docm
- 机密合同.pdf.exe（伪装成PDF）
- 年终奖发放表.xlsx
- 客户资料.zip

要让人一看就想打开。

## 6.4 短信钓鱼

短信钓鱼（Smishing）和邮件钓鱼类似，但是通过短信发送。

### 6.4.1 短信钓鱼的内容

短信钓鱼一般是这样的：

```
【银行】尊敬的用户，您的账户存在异常交易，请立即点击链接验证：https://fake-bank.com

【快递】您的快递已到达，请点击链接取件：https://fake-express.com

【社保局】您的社保卡已被停用，请点击链接重新激活：https://fake-ssb.com

【公司IT】您的VPN账户需要重新验证，请点击：https://fake-vpn.com
```

短信钓鱼的链接一般很短（因为短信有字数限制），可以用短链接。

### 6.4.2 短信钓鱼怎么发？

短信钓鱼一般通过：
- 短信猫池（大量手机卡）
- 第三方短信平台（要审核，但有漏洞）
- 伪基站（技术要求高，违法风险大）

## 6.5 电话钓鱼（Vishing）

电话钓鱼就是直接打电话骗。

### 6.5.1 电话钓鱼的话术

```
场景一：冒充IT支持
"您好，我是公司IT部门的张工。我们检测到您的电脑有异常，需要远程协助处理。请您打开电脑，运行我们发给您的一个诊断工具。"

场景二：冒充领导
"喂，小王吗？我是王总。现在有个急事需要处理，你把你的VPN账号密码发给我，我这边急用。"

场景三：冒充客户
"您好，我是XX公司的，我们有个项目要合作，需要你们提供一个VPN账号给我们远程对接。"
```

电话钓鱼成功率很高，因为电话很难验证身份。但需要有一定的社工能力。

### 6.5.2 电话钓鱼的注意事项

- 声音要自信，不能心虚
- 背景音要像（办公室里嘈杂的环境）
- 话术要提前准备，流畅自然
- 最好有对方的信息，显得真实

## 6.6 社工qq/微信

有时候不用发邮件，直接社工QQ、微信也行。

### 6.6.1 社工qq

- 加目标公司员工为好友
- 伪装成同事、客户、合作方
- 聊天熟悉之后，发文件/链接

### 6.6.2 社工微信

- 通过群聊加好友
- 伪装成熟人介绍
- 分享钓鱼链接

## 6.7 钓鱼的注意事项

最后讲几个钓鱼的注意事项，都是经验之谈。

### 6.7.1 不要太贪心

钓鱼邮件不要贪多，一天发几十封、上百封，很容易被邮件网关拦截，或者触发反垃圾邮件机制。

**建议：**
- 每天发20-50封
- 分批次发（早上20、中午10、下午20）
- 目标要精准，不要乱发

### 6.7.2 内容要像真的

钓鱼邮件要做得像真的，不要有明显的破绽：
- 不要有错别字
- 不要有奇怪的语法
- 格式要像正常的邮件
- 发件人要像真实的身份

### 6.7.3 链接和附件要起好名字

链接和附件的名字要诱人：
- "工资调整方案.docm"
- "项目需求变更.docm"
- "年终奖发放表.xlsx"

不要起：
- "木马.docm"
- "钓鱼.exe"

### 6.7.4 钓鱼网站要像真的

钓鱼网站的细节要到位：
- Logo要清晰
- 布局要像
- URL要像
- 最好有SSL证书

### 6.7.5 注意OPSEC

钓鱼也要注意OPSEC：
- 用代理发送邮件
- 用假信息注册域名
- 钓鱼网站不要放真实IP（用CDN）
- 收到的账号密码要加密存储

### 6.7.6 及时收网

鱼上钩了要赶紧收：
- 马上线了赶紧连上去
- 确认C2上线了再操作
- 后续横向要快，别等马被杀了

## 6.8 钓鱼成功率提升技巧

最后分享几个提升钓鱼成功率的小技巧。

### 6.8.1 时间选择

钓鱼邮件的发送时间很重要：
- 工作日早上8-9点最佳（刚上班，容易看邮件）
- 下午2-3点也可以（下午犯困，容易走神）
- 避免周五下午和周末（没人看邮件）

### 6.8.2 目标选择

钓鱼也要挑目标：
- **IT运维人员**——权限高，一台机器能管很多机器
- **HR和行政**——警惕性可能低，邮箱经常收各种附件
- **销售**——经常和客户打交道，容易相信陌生人
- **财务**——权限高，经常处理各种文件
- **新员工**——不熟悉公司情况，容易被骗

### 6.8.3 内容定制

钓鱼内容要定制化：
- 根据目标的职位、岗位定制内容
- 根据公司的情况定制内容
- 根据时事热点定制内容（比如疫情期间，钓鱼邮件里提"疫情防控"）

### 6.8.4 信任背书

如果能用真实的人名、公司名、事件名当背书，成功率会高很多。
