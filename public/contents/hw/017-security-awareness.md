# 护网全员安全意识培训手册

---

## 一、培训分层体系

```
安全意识培训分层：

高管层 (C-Level):
  重点：安全风险认知、安全投资决策、合规责任
  形式：30分钟宣讲 + 管理层安全责任承诺书

IT/运维人员:
  重点：安全操作规范、补丁管理、配置基线、应急响应
  形式：2小时培训 + 实操考核

普通员工:
  重点：钓鱼邮件识别、密码管理、数据保护、社交工程防范
  形式：1小时培训 + 在线考试

外包/供应商人员:
  重点：数据保密协议、最小权限、访问控制
  形式：30分钟 + 保密承诺书签署
```

## 二、核心培训模块

### 2.1 密码安全

```
密码三原则：
✓ 复杂：≥12位，大小写+数字+符号
✓ 唯一：每个系统密码不同（用密码管理器）
✓ 隐私：不告知他人/不在邮件/聊天中发送密码

密码管理器推荐：
  KeePass, Bitwarden, 1Password

护网期间：
  ⚠️ 任何人以任何理由索要密码 → 拒绝并上报
  ⚠️ 任何"系统升级需要密码"的邮件 → 是钓鱼
```

### 2.2 钓鱼邮件识别

```
钓鱼邮件六大特征：
☐ 发件人地址可疑（admin-company@gmail.com ≠ admin@company.com）
☐ 紧急语气（"您的账户将在24小时内被关闭"）
☐ 语法错误（机器翻译痕迹）
☐ 陌生链接（悬停链接 ≠ 显示的文字）
☐ 异常附件（.exe/.js/.zip密码保护）
☐ 索要敏感信息（密码/验证码/身份证号）

遇到可疑邮件：
→ 不点击、不回复、不下载附件
→ 点击"举报钓鱼邮件"按钮
→ 通过其他方式（电话/企业IM）向发件人确认
```

### 2.3 社会工程防范

```
社会工程常见手段：
  • 冒充IT: "我是IT部门，需要你的密码排查问题"
  • 冒充领导: "我是王总，帮我查一下XX信息"(声音克隆)
  • 尾随进门: 跟着员工进入门禁区
  • USB丢包: 在地面放"遗失"的U盘→好奇插入→感染
  • 虚假WiFi: "Company-Free-WiFi" → 窃取流量

应对原则：
  ✓ 任何索要密码/信息 → 通过官方渠道核实
  ✓ 不认识的USB → 不插入
  ✓ 未经审批的人员 → 不带领参观
  ✓ 异常情况 → 立即报告安全部门
```

## 三、考核机制

```
培训考核：

在线考试:
  满分100分，合格80分
  不合格 → 补考 → 仍不合格 → 通报部门负责人
  题库覆盖：密码/钓鱼/社交工程/数据保护

钓鱼测试:
  每月/每季度向全员发送仿真钓鱼邮件
  → 点击链接者 → 自动跳转到培训页面（不做惩罚）
  → 输入密码者 → 强制重新培训
  → 主动举报者 → 表扬/小奖励

行为积分：
  举报钓鱼：+10分
  参加安全培训：+5分
  发现安全漏洞并报告：+50分
  点击钓鱼链接：+0分（仅提醒）
  重复点击：-10分
```

```
  □ 安全意识屏保/桌面壁纸

  □ 钓鱼测试邮件模板（3-5种不同类型）

  □ 培训效果评估问卷
```
