# 分保与等保的区别与联系

---

## 一、定义对比

| 维度 | 等保 | 分保 |
|------|------|------|
| 全称 | 网络安全等级保护 | 涉密信息系统分级保护 |
| 法律 | 《网络安全法》 | 《保守国家秘密法》 |
| 对象 | 非涉密信息系统 | **涉密**信息系统 |
| 主管部门 | 公安部/网信办 | 国家保密局 |
| 核心标准 | GB/T 22239 | BMB17/BMB20 系列 |
| 等级 | 5级(1-5) | 3级(秘密/机密/绝密) |
| 测评 | 等保测评机构 | 保密测评机构 |

---

## 二、分级对应

```
等保 ←→ 分保 大致对应:

等保三级 ≈ 秘密级(保护要求相当)
等保四级 ≈ 机密级
等保五级 ≈ 绝密级

但两者不能直接等同!
分保要求远高于同等级等保
```

---

## 三、分保核心要求

```
物理隔离:
  ✦ 涉密网络与互联网物理隔离
  ✦ 独立的机房/独立的网线/独立的设备
  ✦ 严禁涉密设备接入互联网

电磁防护:
  ✦ 屏蔽机房/屏蔽机柜(TEMPEST防护)
  ✦ 红黑隔离(涉密信号与通信信号分离)

人员管理:
  ✦ 涉密人员需持证上岗
  ✦ 定期政审
  ✦ 离岗脱密期管理

产品要求:
  ✦ 使用通过国家保密局认证的保密产品
  ✦ 进口设备需安全保密检查
```

---

## 四、常见场景判断

```
某系统存储员工通讯录 → 等保
某系统存储国家秘密文件 → 分保
某系统同时存储两类数据 → 按最高定级(分保)
某涉密单位非涉密办公网 → 等保
某涉密单位涉密内网 → 分保
```
