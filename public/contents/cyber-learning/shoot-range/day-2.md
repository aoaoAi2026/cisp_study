## 2.1 PHPStudy是个啥？

### 2.1.1 用大白话解释

PHPStudy（也叫小皮面板）是个啥东西？

简单说，它就是一个**"网站环境一键安装包"**。

你想在自己电脑上跑PHP网站，得装好多东西：
- Apache（Web服务器，负责响应网页请求）
- MySQL（数据库，存数据的）
- PHP（解释PHP代码的）
- 还有各种扩展、配置...

新手光装这些东西就能装一天，还容易出各种错。

PHPStudy就是把这些东西都打包好了，一键安装，装完就能用。不用你一个个装，也不用你搞复杂的配置。

就像买方便面——
你自己煮面要买面条、调料、锅、煤气...很麻烦。
方便面呢？开水一泡就能吃。

PHPStudy就是安全圈的"方便面"——一键搞定Web环境。

### 2.1.2 为什么用PHPStudy？

为啥选PHPStudy？有这几个好处：

1. **简单**：下一步下一步就装好了，新手也能会
2. **方便**：图形化界面，点鼠标就能操作，不用敲命令
3. **功能全**：Apache、Nginx、MySQL、PHP，啥都有
4. **版本多**：PHP5.x、PHP7.x、PHP8.x，想用哪个用哪个
5. **免费**：个人用免费，不用花钱

对于新手来说，PHPStudy简直是神器。

## 2.2 下载PHPStudy

### 2.2.1 去哪里下？

PHPStudy的官网是：https://www.xp.cn/

或者直接百度搜"PHPStudy"，第一个就是。

> ⚠️ **注意：**
>
> 一定要从官网下载！

### 2.2.2 下载哪个版本？

打开官网，你会看到好几个版本：

- **PHPStudy（小皮面板）Windows版** ← 我们选这个
- PHPStudy Linux版（服务器用的）
- PHPStudy 8.1版（新版，界面更好看）
- PHPStudy 2018版（老版本，比较经典）

**新手建议下载最新的PHPStudy 8.x版本。**

界面好看，功能全，bug也少。

### 2.2.3 下载步骤

1. 打开官网 https://www.xp.cn/
2. 点击"下载"按钮
3. 选择Windows版本
4. 点击下载
5. 等它下完

下载的文件大概几十MB，很快就能下完。

## 2.3 安装PHPStudy

### 2.3.1 安装步骤

下载完之后，你会得到一个安装包，名字大概是 `phpStudy_x64_8.1.xxx.exe` 这样的。

双击运行它，开始安装。

**第一步：同意协议**
- 勾选"我已阅读并同意"
- 点"下一步"

**第二步：选择安装位置**
- ⚠️ **非常重要：安装路径不要有中文，也不要有空格！**
- 比如别装在 `D:\我的软件\PHPStudy` 这种路径
- 建议装在 `D:\phpstudy_pro\` 或者 `C:\phpstudy_pro\`
- 选好路径之后点"下一步"

> 💡 **为什么不能有中文？**
>
> 因为很多PHP程序、漏洞靶场都是外国人写的，对中文路径支持不好。
>
> 装在中文路径下，后面可能会出现各种莫名其妙的问题。
>
> 别给自己找麻烦，老老实实装在英文路径下。

**第三步：选择组件**
- 一般默认就行（Apache + MySQL + PHP）
- 点"下一步"

**第四步：开始安装**
- 点"安装"
- 等进度条走完

**第五步：安装完成**
- 点"完成"
- 勾选"立即运行PHPStudy"

### 2.3.2 启动服务

装完之后，PHPStudy会自动打开。

你会看到一个主界面，左边是菜单，右边是状态。

**找到"首页"或者"环境"页面，你会看到：**
- Apache（Web服务器）
- MySQL（数据库）
- Nginx（另一个Web服务器）
- FTP（文件传输，一般用不上）

**点击Apache和MySQL前面的"启动"按钮。**

启动成功之后，按钮会变成绿色的"停止"。

这说明你的Web环境已经跑起来了！

### 2.3.3 验证一下环境

环境搭好了，得验证一下能不能用。

**打开你的浏览器，输入：**
```
http://localhost
```
或者
```
http://127.0.0.1
```

**如果看到PHPStudy的欢迎页面，说明环境搭成功了！**

欢迎页面上会显示：
- PHP版本
- MySQL版本
- Apache版本
- 各种PHP扩展信息

如果你看到了这个页面，恭喜你，第一步成功了！

## 2.4 PHPStudy常用操作

### 2.4.1 网站根目录在哪？

很多新手会问："我把网站文件放哪啊？"

PHPStudy的网站根目录（就是放网页文件的地方）默认在：

```
D:\phpstudy_pro\WWW\
```

（如果你装在C盘就是C:\phpstudy_pro\WWW\）

你可以这样理解：
- 浏览器访问 `http://localhost/xxx.php`
- 对应的文件就是 `WWW目录下的xxx.php`

举个例子：
- 你在 `D:\phpstudy_pro\WWW\test.php` 写了一个PHP文件
- 浏览器访问 `http://localhost/test.php` 就能看到这个文件的内容

### 2.4.2 怎么改PHP版本？

不同的靶场对PHP版本要求不一样。
有的老靶场需要PHP 5.x，新一点的用PHP 7.x。

**怎么切换PHP版本？**

1. 打开PHPStudy
2. 点击左侧"网站"菜单
3. 找到你的网站，点"管理"
4. 找到"PHP版本"，下拉选择你想要的版本
5. 点确认
6. 重启Apache服务

如果没有你想要的版本，可以去"软件管理"里下载安装。

### 2.4.3 怎么创建数据库？

打靶场经常需要创建数据库。

**怎么创建？**

1. 打开PHPStudy
2. 点击左侧"数据库"菜单
3. 点击"创建数据库"
4. 填写数据库名、用户名、密码
5. 点确认

或者直接用数据库管理工具：
- PHPStudy自带了phpMyAdmin
- 浏览器访问 `http://localhost/phpmyadmin/`
- 用户名root，密码root（默认）
- 可以图形化操作数据库

### 2.4.4 怎么查看PHP配置？

有时候需要改PHP的配置，比如打开某个扩展、改上传文件大小限制。

**怎么改？**

1. 打开PHPStudy
2. 点击左侧"设置"菜单
3. 找到"配置文件"
4. 选择php.ini
5. 改完保存，重启Apache

新手暂时不用改太多配置，默认的就行。等后面遇到问题了再改。

## 2.5 常见问题排查

装环境嘛，总会遇到各种问题。我把新手最常遇到的几个问题列出来，你对着排查。

### 2.5.1 Apache启动失败？

**可能原因1：80端口被占用了**

80端口是Web服务的默认端口。如果你的电脑上有别的程序占用了80端口（比如迅雷、QQ、IIS、其他Web服务器），Apache就起不来。

**怎么排查？**

打开cmd命令行，输入：
```cmd
netstat -ano | findstr ":80"
```

看看哪个程序占用了80端口。

**怎么解决？**

- 把占用80端口的程序关掉
- 或者把Apache的端口改成别的（比如8080）
  - PHPStudy → 设置 → 端口设置 → Apache端口 → 改成8080
  - 改完之后访问地址就变成了 `http://localhost:8080`

### 2.5.2 MySQL启动失败？

**可能原因1：3306端口被占用**

和上面一样，先查端口：
```cmd
netstat -ano | findstr ":3306"
```

如果被占用了，把占用的程序关了，或者改MySQL的端口。

**可能原因2：之前装过MySQL没卸干净**

如果之前装过MySQL，可能会冲突。把之前的MySQL卸载干净，或者改个端口。

### 2.5.3 浏览器访问localhost打不开？

**检查一下：**

1. Apache启动了吗？（按钮是不是绿色的）
2. 端口是不是默认的80？
   - 如果改了端口，要访问 `http://localhost:端口号`
3. 是不是hosts文件有问题？
   - 试试访问 `http://127.0.0.1` 能不能打开
4. 防火墙是不是拦了？
   - 把防火墙关了试试，或者加个白名单

### 2.5.4 页面显示404 Not Found？

404就是"文件找不到"的意思。

**检查：**

1. 文件放对位置了吗？
   - 是不是放在了WWW目录下？
   - 文件名有没有打错？（注意大小写，Windows不区分大小写，但Linux区分）
2. 路径对吗？
   - 比如文件在 `WWW/dvwa/index.php`
   - 那访问地址应该是 `http://localhost/dvwa/`

## 2.6 顺手装几个工具

环境搭好了，顺手再装几个常用工具，后面打靶场会用到。

### 2.6.1 浏览器插件

推荐装几个Chrome插件（Edge也能用）：

- **Wappalyzer**：识别网站用了什么技术（CMS、框架、服务器等）
- **Proxy SwitchyOmega**：切换代理用的（配合Burp Suite）
- **HackBar**：简单的渗透测试工具条（发请求、编码解码）
- **Cookie-Editor**：编辑Cookie的

这些插件都可以在Chrome应用商店或者Edge插件商店搜到。

### 2.6.2 Burp Suite

Burp Suite是做Web安全必备的抓包工具，后面会经常用到。

**下载地址：** https://portswigger.net/burp

**版本选择：** Community版（免费版）就行，新手够用了。

Burp的具体用法我们后面讲，先装上就行。

### 2.6.3 蚁剑/哥斯拉/冰蝎

这三个都是WebShell管理工具，后面打文件上传漏洞的时候会用到。

- **蚁剑（AntSword）**：开源免费，界面友好，新手推荐
- **哥斯拉（Godzilla）**：功能强，免杀好
- **冰蝎（Behinder）**：经典工具，动态加密

新手建议先用蚁剑，简单好用。
