## 📐 第二部分：防守方案设计（实战题）

### 📋 模拟场景

```
你是XX金融科技公司的安全负责人，公司即将参加护网演习。
请根据以下信息，在2小时内设计一份完整的防守方案。

【公司概况】
- 员工：300人
- 核心业务：在线支付系统（日交易额2亿元）
- 办公地点：北京总部 + 上海分公司

【网络资产】
- Web服务器：10台（Nginx + Java Spring Boot）
- API网关：2台
- 数据库：MySQL主从集群（核心交易数据）
- 办公终端：300台Windows/Mac
- 网络设备：华为防火墙、Cisco交换机

【互联网暴露面】
- www.公司.com（在线支付门户）
- api.公司.com（API网关）
- mail.公司.com（企业邮箱）
- vpn.公司.com（远程接入）

【已有安全设备】
- 防火墙 × 2
- WAF × 1（只部署在www）
- 杀毒软件（仅服务器有）
- 没有SIEM、没有EDR、没有蜜罐

【你的任务】
设计包含以下内容的完整防守方案：
1. 漏洞和暴露面整改计划
2. 安全设备补充和部署方案
3. 纵深防御架构图
4. 人员排班和后勤方案
5. 应急预案（至少3类事件）
```
