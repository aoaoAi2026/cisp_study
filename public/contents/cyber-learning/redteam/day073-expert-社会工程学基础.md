## 💡 核心原理：为什么"搞定人"比"搞定系统"更容易？

> 在安全领域有一句笑话：**系统可以打补丁，人不能打补丁。**

### 一个比喻贯穿全章：攻不破的城堡 vs 说得动门卫

想象你面前有一座固若金汤的城堡：

| 攻击方式 | 难度 | 做法 | 成功率 |
|---------|------|------|--------|
| **技术攻击** | ⭐⭐⭐⭐⭐ | 绕过护城河、翻过城墙、破解门锁 | 取决于技术水平 |
| **社工攻击** | ⭐⭐ | 穿上快递制服、拿着一个包裹，跟门卫说"您有快递" | 往往很高 |

**为什么社工容易成功？** 因为人性有几个"出厂设置"永远改不了：

```
人的出厂设置（Bugs）：

1. 好奇心 → "这封邮件说什么，点开看看"
2. 信任心 → "他都说了是IT部的，应该可信吧"
3. 恐惧心 → "说我的银行账户被盗了，赶紧处理"
4. 贪婪心 → "恭喜你中了100万，点击领取"
5. 懒惰心 → "用同一个密码太方便了"
6. 面子心 → "别人都看了，我不看显得不合群"
```

**社工的核心不是技术，是心理学。** 你不需要懂汇编，你需要懂人心。

## 🤔 什么是社会工程学？

### 1.1 定义

**社会工程学（Social Engineering）**
是一种通过人际互动
利用人的心理弱点
获取信息或诱导操作的攻击方式。

说人话：
**不黑系统，黑人心。**

**核心思想：**
- 技术有漏洞可以利用
- 人也有弱点可以利用
- 有时候搞定人比搞定系统容易

**举个例子：**
你想进一个公司的大门，
有两种方法：
1. **技术方法**：研究门禁系统，找漏洞，破解
2. **社工方法**：假装成快递员，让前台帮你开门

哪个更容易？
很多时候是第2个。

### 1.2 为什么社工有效？

社工为什么这么有效？
原因有三：

**1. 人性的弱点是天生的**
- 人都有好奇心
- 人都有同情心
- 人都有恐惧心理
- 人都容易相信权威

这些弱点是刻在基因里的，
不是靠培训就能完全消除的。

**2. 安全意识不足**
- 很多人没有安全意识
- 不知道什么信息不能泄露
- 不知道什么样的人不能信

**3. 制度执行不到位**
- 公司有制度，但执行不严
- 图方便，走捷径
- 多一事不如少一事

### 1.3 社工 vs 技术攻击

| 维度 | 技术攻击 | 社工攻击 |
|------|---------|---------|
| **攻击对象** | 系统、程序、设备 | 人 |
| **利用的弱点** | 技术漏洞 | 心理弱点 |
| **技术门槛** | 高 | 低 |
| **成功率** | 取决于漏洞 | 通常更高 |
| **被发现难度** | 容易留下日志 | 不容易被发现 |
| **法律风险** | 有 | 有（甚至更大） |
| **防御难度** | 打补丁、加固 | 培训、意识提升 |

**红队中的定位：**
- 社工是重要的突破口
- 特别是在外部打点阶段
- 钓鱼邮件是最常用的社工手段

### 1.4 社工的分类

社工可以按不同方式分类：

**按渠道分：**
- **网络社工**：通过QQ、微信、邮件等
- **电话社工**：通过打电话
- **现场社工**：物理接触，假装成各种身份

**按目标分：**
- **获取信息**：套取密码、内部信息等
- **诱导操作**：让对方点链接、运行程序、开门等
- **建立信任**：长期潜伏，获取更多信息

**按手法分：**
- 钓鱼（Phishing）
- 鱼叉钓鱼（Spear Phishing）
- 伪装（Pretexting）
- 尾随（Tailgating）
- ...（后面详细讲）

## 🧠 社会工程学的心理学原理

### 2.1 影响力六大原则

著名社会心理学家
罗伯特·西奥迪尼（Robert Cialdini）
在《影响力》一书中
提出了六大影响力原则。

这六大原则
是社会工程学的心理学基础。

我们一个一个讲。

---

### 2.2 原则一：互惠（Reciprocity）

> **类比：先给糖再要钱**——超市试吃柜台的阿姨给你免费尝一块饼干，你尝完觉得"不买不好意思"，于是买了。先施小恩惠，对方下意识觉得欠你人情。

**原理：**
人都有"知恩图报"的心理。
你给了别人一点好处，
别人就会觉得欠你人情，
想要回报你。

**生活中的例子：**
- 超市里的免费试吃，吃了就不好意思不买
- 别人帮了你一个小忙，你就不好意思拒绝他的请求
- 收了别人的礼物，就更容易答应对方的要求

**社工中的应用：**
- 先给目标一点小好处（比如发个有用的资料）
- 然后再提出自己的请求
- 目标因为"欠人情"，更容易答应

**举例：**
```
攻击者："你好，我是IT部的，
我这里有一份最新的系统操作指南，
发给你参考一下。"
（发了一份真实的指南）

攻击者："对了，顺便问一下，
咱们公司的VPN地址是多少来着？
我这边新电脑配置一下。"

目标："哦，VPN地址是xxx..."
（因为刚收了好处，不好意思拒绝）
```

---

### 2.3 原则二：承诺与一致（Commitment and Consistency）

> **类比：登门槛效应**——推销员先问"可以进来喝杯水吗？"你答应了。然后"我还需要借住一晚。"你已经让他进来了，不好意思再赶他走。人一旦做了小承诺，就会倾向于保持一致，接受更大的请求。

**原理：**
人都希望自己的言行一致。
一旦做出了一个承诺，
就会倾向于遵守这个承诺，
哪怕这个承诺很小。

**生活中的例子：**
- 先让你签个请愿书（小承诺），然后让你捐款（大请求）
- 销售先让你说"对，我想买个好手机"，然后推荐贵的
- 登门槛效应：先提小要求，再提大要求

**社工中的应用：**
- 先让目标答应一个小请求
- 然后逐步提出更大的请求
- 目标为了保持一致，会一步步答应

**举例：**
```
攻击者："你好，我是新来的实习生，
请问一下咱们公司的OA系统网址是什么？"
（小请求，目标答应了）

攻击者："太好了，谢谢！
对了，OA系统的初始密码是什么呀？
我还没收到入职邮件。"
（大一点的请求）

目标："初始密码是xxx..."
（为了保持"乐于助人"的一致形象，继续回答）
```

---

### 2.4 原则三：社会认同（Social Proof）

> **类比：排队的奶茶店**——两家奶茶店门口，一家排长队，一家没人。你会选哪家？大概率是排队的。因为你下意识觉得"这么多人排队，一定好喝"。社工攻击也利用这一点："你们公司的其他人都已经更新密码了，你还没更新吗？"

**原理：**
人在不确定的时候，
会看别人怎么做，
然后跟着做。
也就是"从众心理"。

**生活中的例子：**
- 网红店排队，人越多越想去
- 淘宝买东西看销量和评价
- 电梯里大家都面朝里，你也会面朝里

**社工中的应用：**
- 制造"大家都这么做"的假象
- 让目标觉得"别人都没问题，我也没问题"

**举例：**
```
钓鱼邮件：
"各位同事：
大家好，
昨天IT部发了系统升级通知，
目前已经有80%的同事完成了升级。
还没升级的同事请尽快点击下面的链接升级，
以免影响明天的工作。"

（目标一看：这么多人都升级了，应该没问题，我也点吧）
```

---

### 2.5 原则四：喜好（Liking）

> **类比：卖保险的朋友**——多年不见的老同学突然联系你，聊得很热络，夸你变帅了/变漂亮了，回忆起当年的美好时光。聊到最后说："对了，我最近在做保险，你要不要买一份？"——熟了、有好感了，你更难拒绝。

**原理：**
人更容易答应
自己喜欢的人
提出的要求。

**什么会让我们喜欢一个人？**
- 长得好看（外表吸引力）
- 和我们相似（共同点）
- 会恭维我们（赞美）
- 熟悉的人（接触多了）
- 合作关系（一起做事）

**生活中的例子：**
- 销售会先找共同点："哎呀，你也是XX地方的人啊，老乡！"
- 网红带货，你喜欢这个网红，就容易买他推荐的东西
- 朋友推荐的东西，更容易相信

**社工中的应用：**
- 先建立好感，再提要求
- 找共同点，拉近距离
- 适度赞美，让对方喜欢你

**举例：**
```
攻击者："你好，我是市场部新来的小王，
之前在XX公司做过，
听说你也是XX大学毕业的？
咱们还是校友呢！"
（找共同点，拉近距离）

目标："是吗？太巧了！
你是哪一届的？"

（聊了一会儿校友的话题，关系拉近了）

攻击者："对了校友，
问你个事，
咱们公司的内部资料共享盘地址是什么呀？
我这边还没权限，想先看看资料。"

目标："地址是xxx...
你让你领导开个权限申请就行。"
（因为有好感，就告诉了）
```

---

### 2.6 原则五：权威（Authority）

> **类比：白大褂效应**——一个人在街上喊"吃这个药对心脏好"，没人理他。但同一个药，一个穿白大褂的"医生"推荐给你，很多人就会信。社工中常见的"我是IT部的""我是主管"、"这是公安机关的通知"都是在利用权威。

**原理：**
人天生会服从权威。
对权威的话，
我们更容易相信和服从。

**什么是权威？**
- 职位高的人（领导、专家）
- 穿制服的人（警察、医生）
- 有头衔的人（教授、博士）
- 某个领域的专家

**生活中的例子：**
- 电视广告里穿白大褂的人推荐产品
- 领导说的话，下属更容易服从
- 专家说的话，大家更容易相信

**社工中的应用：**
- 伪装成权威人士（领导、IT专家、警察）
- 利用对方对权威的服从心理
- 让对方不敢质疑，乖乖听话

**举例：**
```
攻击者打电话：
"你好，我是公司IT部的李工，
我是张总让我来找你的。
最近公司系统出了点问题，
张总让我紧急排查一下。
你的账号是xxx对吧？
我需要验证一下你的密码是否被篡改了，
你把密码告诉我，我帮你检查一下。"

（目标一听：IT部的，还是张总安排的，权威啊）
目标："哦好的，密码是xxx..."
```

这是最经典的社工手法之一——
**冒充权威。**

---

### 2.7 原则六：稀缺（Scarcity）

> **类比：最后3件！**——网店经常显示"库存仅剩3件"，刺激你赶紧下单。虽然可能实际库存有3000件。社工邮件经常用"只在今天有效""名额仅剩5个""前10名回复者免手续费"制造紧迫感。

**原理：**
东西越稀少，
人越想要。
机会越少，
人越怕错过。

**生活中的例子：**
- "限时特惠，最后3小时！"
- "限量版，只剩最后10件！"
- "名额有限，先到先得！"

**社工中的应用：**
- 制造紧迫感和稀缺感
- 让目标来不及思考就行动
- "错过就没了"的心理

**举例：**
```
钓鱼邮件：
"紧急通知：
各位同事，
公司年度体检报名今天截止！
还没报名的同事请尽快点击下面的链接报名，
逾期不再受理！
（今天下午6点截止）"

（目标一看：今天截止！赶紧点，别错过了）
（着急忙慌的，就不仔细分辨真假了）
```

---

### 2.8 六大原则总结

| 原则 | 核心 | 社工应用 |
|------|------|---------|
| **互惠** | 知恩图报 | 先给好处，再提要求 |
| **承诺一致** | 言行一致 | 先小请求，再大请求 |
| **社会认同** | 从众心理 | 大家都做了，你也做 |
| **喜好** | 喜欢的人更容易答应 | 拉近距离，建立好感 |
| **权威** | 服从权威 | 冒充领导、专家 |
| **稀缺** | 物以稀为贵 | 制造紧迫感，让人来不及想 |

**记住：**
这六大原则
是社会工程学的心理学基础。
后面讲的各种社工手法，
本质上都是
这些原则的组合运用。

## 🔍 OSINT信息收集与人物画像

### 3.1 什么是OSINT？

**OSINT**（Open Source Intelligence）
就是**开源情报**，
从公开的渠道收集信息。

说人话：
**不用黑客技术，
就从公开的网站、社交媒体、
搜索引擎上收集目标的信息。**

**为什么OSINT重要？**
因为：
- 社工攻击前，需要先了解目标
- 知道的信息越多，社工成功率越高
- 信息就是武器

**OSINT能收集到什么信息？**
- 姓名、职位、联系方式
- 工作经历、教育经历
- 兴趣爱好、社交关系
- 作息规律、生活习惯
- ...

### 3.2 OSINT信息来源

常见的OSINT信息来源：

**搜索引擎：**
- Google、百度、必应
- 各种高级搜索语法（site:、filetype:等）

**社交媒体：**
- 微博、微信、抖音
- LinkedIn（领英）
- Facebook、Twitter
- 各种论坛、社区

**招聘网站：**
- 智联招聘、前程无忧
- Boss直聘、拉勾
- LinkedIn招聘

**企业信息：**
- 企查查、天眼查
- 国家企业信用信息公示系统
- 公司官网、年报

**技术信息：**
- GitHub、GitLab
- 各种技术博客
- Stack Overflow等技术社区

**其他：**
- 新闻报道
- 论文、专利
- 各种公开数据库

### 3.3 人物画像（Profiling）

**人物画像**
就是根据收集到的信息，
勾勒出一个人的
基本特征、性格特点、
行为习惯等。

**画像包括哪些维度？**

| 维度 | 内容 | 用途 |
|------|------|------|
| **基本信息** | 姓名、年龄、性别、职位 | 基本身份识别 |
| **教育背景** | 学校、专业、学历 | 找共同话题 |
| **工作经历** | 历任公司、职位 | 了解能力和权限 |
| **兴趣爱好** | 喜欢什么、关注什么 | 拉近距离的话题 |
| **社交关系** | 朋友、同事、家人 | 了解人脉圈 |
| **性格特点** | 内向/外向、谨慎/大意 | 选择社工手法 |
| **作息规律** | 上下班时间、作息 | 选择攻击时机 |
| **常用工具** | 用什么软件、什么设备 | 选择攻击方式 |

**为什么要做人物画像？**
因为**精准的社工，
一定是建立在充分了解目标的基础上。**

你对目标了解越多，
你的伪装就越真实，
你的话术就越有针对性，
成功率就越高。

### 3.4 OSINT实战流程

**第一步：确定目标**
- 目标人物是谁？
- 目标公司是哪家？

**第二步：搜索引擎初查**
- 搜目标的名字 + 公司
- 搜目标的邮箱、手机号
- 搜目标的各种ID

**第三步：社交媒体深挖**
- 微博、微信、抖音
- LinkedIn（信息最丰富）
- 各种论坛、社区

**第四步：关联信息**
- 把不同来源的信息关联起来
- 拼凑出完整的人物画像
- 找出可以利用的点

**第五步：整理分析**
- 整理收集到的所有信息
- 分析目标的弱点
- 制定社工攻击方案

### 3.5 常用OSINT工具

| 工具 | 用途 |
|------|------|
| **Google Hacking** | 高级搜索语法，精准搜索 |
| **Maltego** | 信息可视化，关系图谱 |

| **Shodan** | 搜索引擎（网络设备） |
| **Censys** | 搜索引擎（证书、主机） |
| **Wayback Machine** | 网页历史存档 |
| **Have I Been Pwned** | 检查邮箱是否泄露 |
| **社工库** | 历史泄露数据查询 |

**注意：**
使用这些工具要遵守法律法规，
不要用于非法用途。

## 🎭 常见社工手法（10种经典）

### 4.1 手法一：钓鱼（Phishing）

**什么是钓鱼？**
通过伪装成可信赖的发件人，
发送虚假信息（通常是邮件），
诱导目标点击链接、
下载附件或泄露信息。

**特点：**
- 批量发送，广撒网
- 成本低，覆盖广
- 成功率相对低，但基数大了总有中招的

**常见类型：**
- 钓鱼邮件
- 钓鱼网站
- 钓鱼短信（Smishing）
- 钓鱼电话（Vishing）

后面两章会详细讲钓鱼。

---

### 4.2 手法二：鱼叉式钓鱼（Spear Phishing）

**什么是鱼叉式钓鱼？**
针对特定目标的
精准钓鱼攻击。
不是广撒网，
而是精准打击。

**和普通钓鱼的区别：**

| 维度 | 普通钓鱼 | 鱼叉式钓鱼 |
|------|---------|-----------|
| **目标** | 随机的大量用户 | 特定的个人或组织 |
| **精准度** | 低 | 高 |
| **准备工作** | 少 | 多（需要收集目标信息） |
| **成功率** | 低 | 高 |
| **危害** | 相对小 | 更大（针对性强） |

**鱼叉式钓鱼为什么成功率高？**
因为：
- 邮件内容是针对目标定制的
- 用目标熟悉的人和事
- 目标不容易怀疑

后面会详细讲。

---

### 4.3 手法三：伪装（Pretexting）

**什么是伪装？**
编造一个虚假的身份和故事，
通过角色扮演
获取目标的信任，
从而达到目的。

说人话：
**装成另一个人，
编一个故事，
骗对方相信你。**

**常见的伪装身份：**
- 公司内部人员（IT部、人事部、财务部）
- 领导、老板
- 客户、合作伙伴
- 快递员、外卖员、维修人员
- 警察、政府工作人员

**伪装的关键：**
1. 身份要合理（不能太离谱）
2. 故事要完整（不能有漏洞）
3. 细节要真实（越细节越可信）
4. 语气要自信（不能心虚）

**举例：**
```
攻击者伪装成IT部的人，
给前台打电话：
"你好，我是IT部的小李，
今天过来给财务部修电脑，
我现在在楼下，
门禁卡忘带了，
你帮我开下门吧。"

（身份合理，故事完整，语气自信）
前台："好的，你上来吧，
我给你开门。"
```

---

### 4.4 手法四：尾随（Tailgating）

**什么是尾随？**
跟着别人进门禁、
进大楼、进机房，
不用自己刷卡或验证。

说人话：
**蹭卡进门。**

**为什么容易成功？**
因为：
- 人都有"多一事不如少一事"的心理
- 大家都是同事，不好意思拦
- 拿着东西不方便，别人会帮忙扶门
- 觉得能进大楼的应该都是内部的

**常见场景：**
- 公司大楼门口
- 电梯口
- 机房门口
- 停车场入口

**防御方法：**
- 每个人都要刷卡验证
- 保安检查工牌
- 安全意识培训

---

### 4.5 手法五：翻垃圾桶（Dumpster Diving）

**什么是翻垃圾桶？**
从目标的垃圾桶里
找有用的信息。

**能找到什么？**
- 废弃的文件、报表
- 快递单（姓名、电话、地址）
- 旧电脑、旧硬盘
- 废弃的工牌、钥匙
- 便签纸（可能写着密码）

**为什么有效？**
因为很多人保密意识不强，
敏感文件直接扔垃圾桶，
不粉碎。

**注意：**
翻垃圾桶可能涉及违法，
不要随便试。

---

### 4.6 手法六： shoulder surfing（肩窥）

**什么是肩窥？**
站在别人身后，
偷看别人输入密码、
看电脑屏幕上的内容。

说人话：
**站你背后偷看。**

**常见场景：**
- 咖啡馆、机场等公共场所
- 电梯里
- 办公室
- ATM机旁

**防御方法：**
- 输入密码时用手遮挡
- 注意周围环境
- 屏幕防窥膜

---

### 4.7 手法七：反向社工（Reverse Social Engineering）

**什么是反向社工？**
不是攻击者去找目标，
而是**让目标主动来找攻击者。**

怎么做到？
1. 攻击者制造一个问题
2. 攻击者声称自己能解决这个问题
3. 目标遇到问题，主动来找攻击者帮忙
4. 攻击者在帮忙的过程中获取信息

**举例：**
```
攻击者先在公司内部论坛发个帖子：
"大家好，我是IT部的小王，
最近很多同事反映电脑慢、蓝屏，
我整理了一个解决办法，
大家有问题可以找我，
我的QQ是xxx。"

然后攻击者在网络上搞点小动作，
让一些人的电脑真的出问题。

目标电脑出问题了，
正好看到这个帖子，
就主动加了攻击者的QQ求助。

攻击者："你这个问题我知道，
把你的账号密码给我，
我远程帮你看看。"

目标："好的，账号是xxx，密码是xxx..."
（主动把密码送上门）
```

**反向社工为什么厉害？**
因为是目标主动找的攻击者，
目标对攻击者的信任度更高，
警惕性更低。

---

### 4.8 手法八：诱饵（Baiting）

**什么是诱饵？**
放一个"诱饵"，
勾起目标的好奇心或贪婪心，
让目标主动去碰。

**常见的诱饵：**
- 放一个U盘在公司，上面写着"工资表"
- 放一张"机密"的光盘
- 网上放一个"内部资料"的下载链接

**为什么有效？**
因为：
- 人都有好奇心
- 人都有贪便宜的心理
- "机密"、"工资"这些词太诱人了

**举例：**
```
攻击者在公司停车场、电梯口
放几个U盘，
U盘上写着：
"2024年工资调整方案 - 机密"

有人捡到了，
好奇里面是什么，
插到自己电脑上看。
U盘里的木马就运行了。
```

---

### 4.9 手法九：借口（Quid Pro Quo）

**什么是Quid Pro Quo？**
拉丁语，意思是
"交换，你给我这个，我给你那个"。

和互惠原则类似，
但更直接：
我给你一个好处，
你给我想要的信息。

**举例：**
```
攻击者打电话：
"你好，我是IT部的，
我们正在做系统升级，
配合我们的同事
可以获得500元的京东卡奖励。
现在只需要你告诉我
你的工号和部门，
我帮你登记一下。"

目标一听有奖励，就说了。
```

---

### 4.10 手法十：恐惧营销（Scareware）

**什么是恐惧营销？**
制造恐惧，
让目标因为害怕
而做出不理智的行为。

**常见套路：**
- "你的电脑中病毒了！赶紧下载杀毒！"
- "你的账号被盗了！赶紧修改密码！"
- "你涉嫌违法！赶紧配合调查！"

**为什么有效？**
因为人在恐惧的时候，
理性思考能力会下降，
更容易被操控。

**举例：**
```
钓鱼邮件：
"紧急通知：
你的邮箱账号存在异常登录，
疑似被盗。
为了你的账号安全，
请立即点击下面的链接修改密码，
否则24小时内将冻结你的账号。
—— IT安全部"

目标一看：账号要被冻结了！好怕怕！
赶紧点链接改密码。
（其实是钓鱼网站，密码被偷走了）
```

---

| 手法 | 核心原理 | 难度 | 成功率 |
|------|---------|------|-------|

| 鱼叉式钓鱼 | 精准定制 | 中 | 高 |

| 尾随 | 蹭卡进门 | 低 | 高 |

| 反向社工 | 让目标主动找你 | 高 | 很高 |
| 诱饵 | 利用好奇心/贪婪 | 低 | 中 |
| 交换 | 给好处换信息 | 低 | 中 |
| 恐惧营销 | 利用恐惧心理 | 低 | 高 |

## 🎭 身份伪装与角色扮演

### 5.1 为什么要伪装身份？

社工攻击中，
你不能用真实身份，
需要伪装成另一个人。

**伪装的目的：**
- 获取目标的信任
- 让目标放下戒备
- 让目标愿意配合你

**好的伪装，
是成功的一半。**

### 5.2 常见的伪装身份

**红队中常用的伪装身份：**

| 身份 | 适用场景 | 优势 | 劣势 |
|------|---------|------|------|
| **IT部员工** | 要密码、远程协助 | 权威，技术相关 | 需要懂技术术语 |
| **人事部员工** | 要个人信息、账号 | 管人事，合理 | 权限有限 |
| **财务部员工** | 转账、财务信息 | 管钱，敏感 | 风险大，容易被识破 |
| **新员工/实习生** | 问各种基础问题 | 新人不懂很正常 | 权限低，能获取的信息少 |
| **领导/老板** | 下命令、要东西 | 权威，不敢不听 | 目标大，容易被核实 |
| **客户/合作伙伴** | 商务往来，套信息 | 外部的，信息不对称 | 需要了解业务 |
| **快递员/外卖员** | 进大门、找个人 | 常见，不被怀疑 | 进不去深层区域 |
| **维修人员** | 进机房、修设备 | 合理进入敏感区域 | 需要懂技术 |

### 5.3 如何做好伪装？

**一个好的伪装，
需要准备充分。**

**准备清单：**

**1. 身份信息**
- 姓名（要真实一点，不能太假）
- 职位（具体是什么岗位）
- 部门（哪个部门的）
- 工号（如果需要）
- 上级是谁（领导是谁）
- 入职时间（新员工还是老员工）

**2. 背景故事**
- 你为什么来找目标？
- 事情的来龙去脉是什么？
- 遇到了什么问题，需要什么帮助？
- 故事要完整，逻辑要通顺

**3. 细节准备**
- 公司的基本情况（业务、组织架构）
- 部门的情况（有哪些人，做什么）
- 相关的专业术语（不能露馅）
- 常见问题的回答（万一被问住怎么办）

**4. 道具准备**
- 工牌（如果是现场社工）
- 制服（如果需要）
- 工具包（维修人员的工具箱）
- 手机铃声、壁纸（看起来像内部人员）

### 5.4 角色扮演的技巧

**技巧1：自信**
- 说话要自信，不能吞吞吐吐
- 语气要坚定，不能犹犹豫豫
- 你越自信，对方越不怀疑

**技巧2：自然**
- 不要太刻意
- 就像你真的是那个身份一样
- 太假了反而容易被怀疑

**技巧3：适度**
- 不要说太多，言多必失
- 不要问太多，容易引起怀疑
- 达到目的就撤，不要恋战

**技巧4：灵活应变**
- 计划赶不上变化
- 遇到意外情况不要慌
- 临场发挥，圆过去

**技巧5：换位思考**
- 如果你是目标，你会怎么想？
- 如果你是目标，你会怀疑什么？
- 站在对方的角度思考

### 5.5 容易露馅的地方

**常见的破绽：**

1. **专业术语不对**
   - 冒充IT的，却不懂技术术语
   - 冒充财务的，却不懂财务术语

2. **对公司情况不了解**
   - 连公司有几个部门都不知道
   - 连基本的流程都不清楚

3. **信息前后矛盾**
   - 刚才说自己是A部门的，后来又说B部门
   - 刚才说姓张，后来又说姓李

4. **回答不上来问题**
   - 对方问一个简单的问题就答不上来
   - 支支吾吾，转移话题

5. **太着急**
   - 一上来就要这要那
   - 急急忙忙，不正常

**怎么避免？**
- 准备充分，多做功课
- 少说多听，言多必失
- 遇到不会的，转移话题
- 实在不行就撤，不要硬撑

## 🤝 信任建立的艺术

### 6.1 为什么要建立信任？

社工攻击的核心
就是**获取目标的信任。**

信任了，
你说什么对方都信；
不信任，
你说什么对方都怀疑。

**信任是怎么建立的？**
信任不是一蹴而就的，
而是一步步建立起来的。

### 6.2 信任建立的步骤

**第一步：破冰（拉近距离）**
- 打招呼，自我介绍
- 找共同点，拉近距离
- 让对方对你有个好印象

**第二步：建立好感**
- 适度赞美
- 聊对方感兴趣的话题
- 让对方喜欢你

**第三步：展现价值**
- 给对方一点小帮助
- 展示你的"专业性"
- 让对方觉得你有价值

**第四步：逐步深入**
- 先问简单的、不敏感的问题
- 再问稍微敏感一点的
- 最后问核心的、敏感的

**第五步：达到目的**
- 获取你想要的信息
- 或者让对方做你想让他做的事

### 6.3 建立信任的技巧

**技巧1：找共同点**
- 老乡、校友、共同的朋友
- 共同的兴趣爱好
- 共同的经历
- 人都喜欢和自己相似的人

**技巧2：真诚赞美**
- 夸对方的工作能力
- 夸对方的眼光
- 夸对方的穿着打扮
- 真诚的赞美谁都喜欢

**技巧3：主动提供帮助**
- "我帮你看看这个问题吧"
- "我那边有个资料，发给你参考"
- 先给好处，对方就会信任你

**技巧4：暴露一点小弱点**
- 适当暴露一点无关紧要的小缺点
- 让你看起来更真实
- 完美的人反而不可信

**技巧5：保持一致**
- 你的身份、故事、言行要一致
- 前后矛盾会摧毁信任

### 6.4 注意事项

**不要操之过急：**
- 信任需要时间
- 一上来就要这要那，肯定被怀疑
- 慢慢来，一步一步来

**不要贪多：**
- 一次不要获取太多信息
- 达到目的就撤
- 贪多容易暴露

**不要说谎太离谱：**
- 谎言要有事实基础
- 太离谱的谎言容易被戳穿
- 九真一假最不容易被发现

## 📞 社工的三种形式

### 7.1 网络社工

**什么是网络社工？**
通过网络渠道进行的社工攻击。

**常见渠道：**
- 邮件（钓鱼邮件）
- 即时通讯（QQ、微信、钉钉）
- 社交媒体（微博、LinkedIn）
- 论坛、社区

**特点：**
- 隐蔽性强，不容易被追踪
- 可以同时针对多个目标
- 成本低
- 不需要面对面

**红队中的应用：**
- 钓鱼邮件（最常用）
- 社交平台私聊
- 论坛/群里套信息

---

### 7.2 电话社工

**什么是电话社工？**
通过打电话进行的社工攻击。

**特点：**
- 实时互动，临场发挥要求高
- 声音、语气很重要
- 比文字更有说服力
- 容易暴露声音

**电话社工的技巧：**
1. **准备充分**：把要说的话写下来，背熟
2. **语气自信**：说话要坚定，不能吞吞吐吐
3. **控制节奏**：你主导对话，不要被对方带偏
4. **学会打断**：对方问太多问题时，礼貌打断
5. **制造紧迫感**：让对方来不及思考

**举例：**
```
攻击者："你好，我是IT部的李工，
工号00123。
是这样的，
我们监控到你的电脑
有异常网络流量，
可能中病毒了。
为了不影响整个公司的网络，
需要紧急处理一下。
你现在在工位上吗？"

目标："在的，怎么处理？"

攻击者："你打开运行窗口，
输入cmd，
然后输入xxx...
我帮你远程检查一下。"

（一步步引导目标操作）
```

---

### 7.3 现场社工

**什么是现场社工？**
物理接触，
面对面的社工攻击。

**特点：**
- 效果最好，成功率最高
- 风险也最大，容易被抓
- 需要很强的临场应变能力

**常见场景：**
- 公司大楼门口（尾随）

**现场社工的技巧：**

## ⚖️ 法律与伦理边界

### 8.1 社工的法律风险

**重要的事情说三遍：**
**社工攻击可能违法！
社工攻击可能违法！
社工攻击可能违法！**

**可能涉及的法律问题：**

1. **侵犯公民个人信息罪**
   - 非法获取公民个人信息
   - 买卖、提供公民个人信息

2. **诈骗罪**
   - 用虚构事实或隐瞒真相的方法
   - 骗取数额较大的公私财物

3. **非法侵入计算机信息系统罪**
   - 通过社工手段获取账号密码
   - 然后登录系统

4. **敲诈勒索罪**
   - 用威胁、要挟的方法
   - 索取财物

5. **侵犯名誉权、隐私权**
   - 未经允许公开他人信息
   - 造谣、诽谤

**具体判多少年？**
看情节轻重，
轻的拘留罚款，
重的判好几年。

### 8.2 什么情况下是合法的？

**合法的社工：**
- **授权的渗透测试**：客户授权你做社工测试
- **安全研究**：研究防御方法，不攻击别人
- **安全教育**：培训员工，提高安全意识

**红队行动中的社工：**
- 必须有客户的书面授权
- 必须在授权范围内进行
- 不能超出约定的边界
- 不能造成不必要的损害

**记住：**
**没有授权的社工攻击，
都是违法的。**

**红队的职业操守：**
1. **只做授权范围内的事**
2. **保护目标的隐私**
3. **不造成不必要的伤害**
4. **事后帮助修复漏洞**

**学习社工的正确姿势：**

### 案例1：经典社工攻击案例分析

**背景：**

**分析：**
- 利用了什么心理？**权威**（税务局）+ **恐惧**（税务问题很严重）
- 为什么成功？信息精准，语气逼真，目标不懂税务流程

### 案例3：电话社工实战模拟

**场景：**
伪装成IT部的人，
给目标打电话，
骗取目标的账号密码。

**准备工作：**
1. 收集目标信息：姓名、职位、部门
2. 准备身份：IT部的李工，工号00123
3. 准备故事：系统升级，需要验证账号
4. 准备话术：把要说的话写下来

**话术脚本：**

```
开场：
"你好，请问是张经理吗？
我是IT部的李明，
工号00123。"

说明来意：
"是这样的张经理，
咱们公司今天晚上要做系统升级，
我们现在在做账号验证，
确保升级后大家都能正常登录。
需要跟你核对一下你的账号信息。"

获取信息：
"你的账号是zhangwei对吧？
对一下部门，你是市场部的？
好的，没问题。
现在需要验证一下密码，
你把密码告诉我，
我在后台帮你验证一下，
确保升级后能用。"

如果对方犹豫：
"张经理，这个是公司统一安排的，
今天不验证的话，
明天升级完你可能登不上系统，
影响工作就不好了。
很快的，一分钟的事。"

拿到密码后：
"好的，验证通过了，
没问题。
升级是今天晚上12点到2点，
不会影响你白天工作的。
谢谢配合啊张经理。"

结束：
"好的，那就不打扰你了，
有问题随时找IT部。
再见。"
```

**要点分析：**
- 开场直接点名，确认身份
- 用"系统升级"这个合理的理由
- 先核对账号（让对方觉得你真的是IT部的）
- 制造紧迫感（不验证明天登不上）
- 拿到信息就撤，不多聊

### 案例4：身份伪装技巧

**场景：**
伪装成新入职的员工，
去公司各个部门转，
收集信息。

**伪装身份：**
- 姓名：王磊
- 职位：市场部实习生
- 入职时间：刚入职3天
- 直属领导：刘经理（市场部确实有个刘经理）
- 任务：熟悉公司环境，认识各部门的人

**准备工作：**
1. 了解公司组织架构
2. 了解市场部的基本情况
3. 准备一套像样的衣服（商务休闲）
4. 准备一个假工牌（如果能混进去就不用）
5. 准备话术

**执行过程：**

**第一步：进大楼**
- 跟着上班的人一起进（尾随）
- 自然大方，不要东张西望

**第二步：找前台**
"你好，我是市场部新来的实习生王磊，
今天第一天来报到，
请问刘经理在吗？"
（前台一般不会太怀疑实习生）

**第三步：各部门转**
- 每个部门都去转转
- "你好，我是市场部新来的实习生，
过来认识一下大家，
以后多多关照。"
- 顺便看看各个部门在做什么
- 听听大家在聊什么

**第四步：收集信息**
- 记住各个部门的位置
- 记住关键人物的座位
- 看看有没有人把密码写在便签上
- 看看有没有人离开座位不锁屏

**第五步：离开**
- 差不多了就离开
- "我去趟卫生间"
- 然后直接走掉

**要点：**
- 身份要合理（实习生，不懂很正常）
- 态度要谦虚（新人姿态，大家不会为难）
- 不要太刻意（自然一点，就像真的一样）
- 见机行事（情况不对赶紧撤）

### 案例5：信任建立的艺术

**场景：**
通过LinkedIn联系目标，
建立联系，
逐步获取信任，
最后获取信息。

**目标：**
某公司的技术主管。

**步骤：**

**第一步：建立连接**
- 在LinkedIn上加目标好友
- 验证信息："你好，我是XX公司的，
之前在XX技术大会上听过你的演讲，
受益匪浅。
想加个好友，以后多交流。"
（提共同经历，拉近距离）
- 目标通过好友申请

**第二步：初步交流**
- 过几天，给目标发消息：
"王总你好，
最近看了你写的关于微服务的文章，
写得太好了！
我们公司也在做微服务改造，
遇到了几个问题，
想请教一下你。"
（赞美 + 请教，满足对方的成就感）
- 目标回复了，解答了问题

**第三步：建立好感**
- "太感谢了王总！
你说的这个思路我怎么没想到，
茅塞顿开啊！
改天请你吃饭，
当面请教。"
（感谢 + 邀请，拉近距离）
- 目标客气了一下

**第四步：深入交流**
- 时不时请教一些技术问题
- 每次都表示感谢
- 偶尔分享一些有价值的技术资讯
- 关系越来越近

**第五步：获取信息**
- 过了一段时间，
"王总，跟你打听个事，
你们公司现在用的是什么云服务商啊？
我们公司最近要上云，
正在选型，
想参考一下你们的方案。"
- 目标没多想，就说了

**要点分析：**
- 周期长，但成功率高
- 利用了喜好、互惠、权威等原则
- 信任是慢慢建立起来的
- 不要急，慢慢来
