## 一、音频隐写专题通关

### 1.1 音频隐写知识复习

```
【音频隐写知识速查】

1. 频谱图隐写
  原理：
    - 音频转换为频谱图
    - 频谱图上显示文字或二维码
    - 听觉无法察觉

  检测方法：
    - Audacity打开音频
    - 切换到频谱图视图
    - 查看是否有文字或图案

2. LSB隐写
  原理：
    - 修改音频采样点的最低位
    - 人耳听不出差异
    - 可以隐藏大量数据

  检测方法：
    - 使用SilentEye工具
    - 使用Python脚本提取

3. 波形隐写
  原理：
    - 波形高度编码信息
    - 波形方向编码信息
    - 静音间隔编码信息

  检测方法：
    - Audacity查看波形
    - 分析波形规律

4. 多声道隐写
  原理：
    - 左右声道内容不同
    - 差异中隐藏信息

  检测方法：
    - 分离左右声道
    - 分别分析

5. 文件结构隐写
  原理：
    - 文件尾部追加数据
    - 元数据中隐藏信息

  检测方法：
    - 010 Editor查看文件结构
    - exiftool查看元数据

【Audacity操作速查】

频谱图查看：
  1. 打开音频文件
  2. 点击轨道名称旁的下拉箭头
  3. 选择"频谱图"
  4. 或按 Ctrl + Shift + E

声道分离：
  1. 打开立体声音频
  2. 点击轨道名称旁的下拉箭头
  3. 选择"分离立体声轨道"
  4. 分别分析每个声道

频谱图参数调整：
  1. 编辑 → 首选项 → 频谱图
  2. 调整FFT大小
  3. 调整增益和范围
```

```
【压缩包知识速查】

【压缩包工具】

1. fcrackzip —— ZIP密码破解
  安装：apt install fcrackzip
  用法：fcrackzip -u -D -p wordlist.txt file.zip

2. John the Ripper —— 密码破解
  安装：apt install john
  用法：
    zip2john file.zip > hash.txt
    john hash.txt

3. 010 Editor —— 文件编辑
  用途：修改加密标志位

4. binwalk —— 文件分离
  用法：binwalk -e file.zip

5. 7-Zip —— 解压工具
  支持多种格式
```

```
【ZIP伪加密详解】

ZIP文件结构：
  本地文件头：50 4B 03 04
  ┌─────────────────────────────────┐
  │ 偏移  长度  说明                 │
  ├─────────────────────────────────┤
  │ 0     4    签名(50 4B 03 04)    │
  │ 4     2    解压所需版本          │
  │ 6     2    通用标志位 ← 关键     │
  │ 8     2    压缩方法              │
  │ ...   ...  ...                   │
  └─────────────────────────────────┘

加密标志位：
  00 00 —— 无加密
  09 00 —— 有加密（可能是伪加密）

伪加密原理：
  - 修改加密标志位为09 00
  - 解压软件认为有密码
  - 但实际数据未加密
  - 修改回00 00即可解压

伪加密检测：
  1. 用010 Editor打开
  2. 搜索50 4B 03 04
  3. 查看第6-7字节
  4. 如果是09 00，可能是伪加密
  5. 修改为00 00
  6. 保存并解压
```

## 八、常见问题FAQ

### Q1: 独立完成率低于50%怎么办？

```
回答：
  不要气馁，这是正常的学习过程。

解决方法：
  1. 分析错题原因
     - 知识点不熟悉？复习相关内容
     - 工具不会用？多练习工具
     - 思路错误？看WriteUp学习思路

  2. 制定补课计划
     - 每个错题重做一遍
     - 找同类题目练习
     - 整理错题本

  3. 调整学习方法
     - 先理解原理再做题
     - 不要急于求成
     - 多看WriteUp学习思路

  4. 寻求帮助
     - 在论坛提问
     - 向朋友请教
     - 看视频教程
```

### Q2: 做题时间太长怎么办？

```
回答：
  解题速度需要通过练习提高。

建议：
  1. 控制每题时间
     - 简单题：5分钟
     - 中等题：10分钟
     - 困难题：15分钟
     - 超时看WriteUp

  2. 建立解题流程
     - 按固定步骤检测
     - 形成肌肉记忆

  3. 熟练使用工具
     - 掌握快捷键
     - 了解工具特性

  4. 积累经验
     - 多做题
     - 总结题型
```

```
  2. 分析薄弱环节
  3. 确认是否掌握核心知识点
  4. 确认是否能熟练使用工具

  - 重点补课薄弱环节
```
