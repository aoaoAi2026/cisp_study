## 二、核心概念回顾

### SOC 与事件响应

| 概念 | 要点 |
|------|------|
| SOC 分层 | L1告警分诊 → L2事件分析 → L3深度调查 |
| MTTD | 平均检测时间 → 目标 <1小时 |
| MTTR | 平均响应时间 → 目标 <4小时 |
| PICERL | 准备→识别→遏制→清除→恢复→复盘 |
| NIST | 准备→检测分析→遏制清除恢复→事后活动 |
| 链式证据 | 证据每次转移必须有记录和签名 |

### IDS vs IPS

| IDS | IPS |
|-----|-----|
| 旁路部署 | **串联/Inline** 部署 |
| 只检测告警 | 检测 + **阻断** |
| 零网络延迟 | 有处理延迟 |
| 无单点故障风险 | 单点故障风险 |

### 检测方法

| 方法 | 优点 | 缺点 |
|------|------|------|
| 签名 | 误报低 | 不知未知攻击 |
| 异常 | 可检测未知 | 误报高 |
| 协议 | 精准 | 受变种影响 |
| 信誉 | 快 | 依赖情报时效 |

### 灾备核心

```
BCP > DRP  (BCP包含DRP)

BIA → 分析影响 → 确定RTO/RPO → 制定BCP/DRP

RTO = 多久恢复服务  (对应恢复优先级)
RPO = 丢多少数据    (决定备份频率)

灾备站点：
冷站 → 空场地 → 天/周 → 便宜
温站 → 有部分设备 → 小时/天 → 适中
热站 → 完整配置 → 分钟/小时 → 贵
镜像 → 双活 → 秒/分钟 → 最贵
```

### 备份 3-2-1 规则

```
3 份副本 + 2 种介质 + 1 份异地
   进阶：+ 1 份离线/不可变 + 0 验证错误
```

## 三、关键对比速查

| 对比项 | A | B | 关键区别 |
|--------|---|---|----------|
| SOC L1 vs L2 | 告警分诊 | 事件分析 | L1浅L2深 |
| IDS vs IPS | 旁路检测 | 串联阻断 | IPS 能断IDS只能看 |
| BCP vs DRP | 业务连续 | IT恢复 | BCP包含DRP |
| RTO vs RPO | 恢复服务时间 | 恢复数据点 | RTO问"多久"RPO问"丢多少" |
| 冷站 vs 热站 | 空场地 | 全配置 | 冷→天级/热→分钟级 |
| 增量 vs 差异 | 上次任何备份以来的变化 | 上次全量以来的变化 | 增量恢复最慢/备份最快 |
