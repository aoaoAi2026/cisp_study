## 一、消防安全体系

### 1.1 消防三要素

```
火灾发生的必要条件（三要素/火三角）：

      热量 (Heat)
        /\
       /  \
      /    \
     / 火   \
    /        \
   /__________\
 燃料(Fuel)  氧气(Oxygen)

消防目标：打破任意一个要素即可灭火
```

### 1.2 火灾检测

```
火灾检测四阶段（按时间顺序）：

① 初期(incipient)：不可见的燃烧产物 → 离子式烟雾探测
② 发烟(smoldering)：可见烟雾 → 光电式烟雾探测
③ 火焰(flame)：明火出现 → 红外/紫外火焰探测
④ 高温(heat)：大量热量 → 定温/差温探测器

常见探测器：
  离子式：检测小颗粒（快速燃烧）
  光电式：检测大颗粒（缓慢燃烧/阴燃）
  吸气式(VESDA)：主动抽气检查 → 极早期发现
```

## 二、灭火系统对比

### 2.1 灭火剂分类

```
┌────────────┬───────────────┬──────────────┬──────────────┐
│   灭火剂    │    灭火原理    │   设备损害    │    适用性     │
├────────────┼───────────────┼──────────────┼──────────────┤
│ 水（喷淋）  │ 降温          │ ⚠️ 严重损坏   │ 非电子区域    │
│ 干粉        │ 窒息+降温     │ ⚠️ 腐蚀设备   │ 户外/仓库    │
│ 二氧化碳(CO₂)│ 隔绝氧气      │ ✅ 无残留     │ 电机/无人区  │
│ FM-200(HFC) │ 化学抑制      │ ✅ 无残留     │ 数据中心     │
│ Novec 1230  │ 吸热蒸发      │ ✅ 环保推荐   │ 数据中心     │
│ 惰性气体     │ 降氧浓度      │ ✅ 无残留     │ 数据中心     │
│ 水雾         │ 降温+窒息     │ ⚠️ 可能有水   │ 特定场景    │
└────────────┴───────────────┴──────────────┴──────────────┘

机房灭火选择：
  首选：Novec 1230 / FM-200 / 惰性气体 (IG-55/IG-100)
  原因：不导电、无残留、环保
  CO₂慎用！→ 浓度达34%可致人窒息死亡！
```

### 2.2 气体灭火释放流程

```
气体灭火系统释放顺序（双触发机制）：

① 第一个探测器触发 → 声光报警
② 第二个探测器触发 → 30秒倒计时开始
③ 倒计时期间：人员必须撤离
④ 倒计时结束：释放气体
⑤ 气体释放指示灯亮 → 禁止进入
⑥ 灭火后 → 排风系统启动
```

## 三、防水与防漏

### 3.1 防水策略

```
机房防水措施：

位置选择：
  ✅ 不在建筑底层/地下室（水淹风险）
  ✅ 不在顶层（漏水风险）
  ✅ 远离水管/卫生间区域

检测系统：
  · 漏水检测绳(Leak Detection Rope)
  · 地板下安装（架空地板下）
  · 空调接水盘液位传感器

隔离措施：
  · 机房抬高地面（架空地板≥30cm）
  · 防水坎/挡水墙
  · 所有水管安装截止阀和报警
```

## 四、电力保障体系

### 4.1 供电架构

```
数据中心电力链路：

市电 ─→ ATS(自动转换开关) ─→ UPS ─→ PDU ─→ 设备
  │         │
  └─── 发电机 ──┘

ATS：检测市电中断 → 启动发电机 → 自动切换
UPS：不间断电源
  在线式：电池始终在电路中 → 零切换时间（推荐）
  离线式：市电直通 → 断电时切换（有毫秒级间隙）
  在线互动式：稳压+电池切换（介于两者之间）

PDU (Power Distribution Unit)：机柜级电力分配
```

### 4.2 UPS电池类型

| 类型 | 优点 | 缺点 |
|------|------|------|
| 铅酸(VRLA) | 便宜、成熟 | 寿命短(3-5年)、重 |
| 锂离子 | 寿命长(10年+)、轻 | 成本高、需BMS管理 |
| 镍镉 | 耐高低温 | 重金属污染 |

### 4.3 双路供电

```
双路供电：来自不同变电站(或至少不同变压器)

  ┌───────┐     ┌───────┐
  │市电A路 │     │市电B路 │
  └───┬───┘     └───┬───┘
      │             │
  ┌───▼───┐     ┌───▼───┐
  │ UPS A │     │ UPS B │
  └───┬───┘     └───┬───┘
      │             │
  ┌───▼─────────────▼───┐
  │ ATS / 双路设备       │
  └─────────────────────┘

同样关键：
  发电机燃料储备 ≥ 24小时（重要数据中心 ≥ 72小时）
  定期带载测试（至少季度一次）
```

## 五、温湿度与环境控制

### 5.1 数据中心环境标准

```
ASHRAE 推荐范围：
  温度：18°C - 27°C (推荐)
  湿度：露点 5.5°C - 15°C (相对湿度约40-60%)

湿度过低 → 静电(ESD)风险
湿度过高 → 冷凝、腐蚀风险

空调配置：
  N+1 冗余（最少）
  精密空调(恒温恒湿)
  冷热通道分离
```

## 六、电磁安全(TEMPEST)

### 6.1 TEMPEST概念

```
TEMPEST：防止电磁信号泄露导致信息泄露

威胁：
  显示设备：CRT/LCD的电磁辐射可被远距离还原图像
  键盘信号：击键信号可被还原
  网线：非屏蔽双绞线(UTP)辐射数据信号

防御：
  · 电磁屏蔽(法拉第笼)
  · 红黑分离：涉密设备与非涉密设备安全距离
  · 光缆替代铜缆(光纤无电磁辐射)
  · TEMPEST认证设备
  · 信号干扰(白噪声发生器)
```

### 6.2 红黑分离

```
红/黑 (Red/Black) 概念：

  Red设备：处理涉密信息
  Black设备：处理非密信息
  Red线缆：传输涉密信号
  Black线缆：传输非密信号

  分离规则：
  · Red和Black设备需物理分隔
  · Red和Black线缆至少间隔1m（或金属管隔离）
  · Red信号的任何泄露都不能进入Black环境
```
