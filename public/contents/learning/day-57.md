## 一、物理安全在信息安全中的定位

### 1.1 物理安全 = 信息安全基础

```
▸ 如果你的服务器被偷走了 → 加密、防火墙全部失效
▸ 如果有人进入机房直接操作 → 网络边界形同虚设
▸ 如果火灾烧毁数据中心 → 所有业务中断

物理安全是信息安全防御的第一道防线！
```

### 1.2 CIA与物理安全

| CIA要素 | 物理安全贡献 |
|---------|-------------|
| 机密性 | 防止未授权人员接触设备/介质 |
| 完整性 | 防止物理篡改、硬件后门植入 |
| 可用性 | 防火、防水、电力保障 → 服务不中断 |

### 2.2 各层防护措施

| 层次 | 典型措施 | 关键设备 |
|------|---------|---------|
| 边界 | 围墙、栅栏、岗亭 | 车牌识别、红外探测器 |
| 建筑 | 门禁系统、安检 | 读卡器、金属探测门 |
| 楼层 | 楼层权限隔离 | 电梯刷卡控制 |
| 机房 | 双重认证进入 | 生物识别+IC卡 |
| 机柜 | 机柜锁、监控 | 电子锁、温湿度传感器 |
| 设备 | 防篡改、标签 | 防拆标签、资产标签 |

## 四、物理安全威胁分类

| 类别 | 威胁示例 |
|------|---------|
| 自然灾害 | 火灾、洪水、地震、台风 |

## 六、访问控制物理层面

### 6.1 人员管控

```
访问控制三要素：
  Something you HAVE → 门禁卡、钥匙、令牌
  Something you ARE → 指纹、虹膜、人脸
  Something you KNOW → PIN码、密码

多因素(至少两个不同类别) ≥ 单因素

防尾随(Anti-tailgating)：
  · 互锁门(Mantrap)：一个人通过后关闭才开下一个
  · 光学栅栏：检测多人同时通过
  · 保安值守+登记
```

## 七、物理安全评估方法

```
评估清单：

□ 边界安全：围墙高度、照明、摄像头覆盖
□ 建筑安全：门窗加固、门禁系统、保安巡逻
□ 机房安全：位置（非底层/非顶层）、防水措施
□ 访问控制：访问申请流程、访客登记、授权记录
□ 环境控制：消防、空调、UPS、双路供电
□ 监控记录：CCTV覆盖、录像保存期、回放测试
□ 应急预案：火灾、断电、入侵场景
```
